In [linear algebra](https://en.wikipedia.org/wiki/Linear_algebra "Linear algebra"), an **augmented matrix**![Image 1: {\displaystyle (A\vert B)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/116d0c4ede70511eef9e6b0f557ebf89182c7b0f) is a ![Image 2: {\displaystyle k\times (n+1)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/f0b52b9e56bfe607df9f41ebb1a707fd66e5ca78)[matrix](https://en.wikipedia.org/wiki/Matrix_(mathematics) "Matrix (mathematics)") obtained by appending a ![Image 3: {\displaystyle k}](https://wikimedia.org/api/rest_v1/media/math/render/svg/c3c9a2c7b599b37105512c5d570edc034056dd40)-dimensional column vector ![Image 4: {\displaystyle B}](https://wikimedia.org/api/rest_v1/media/math/render/svg/47136aad860d145f75f3eed3022df827cee94d7a), on the right, as a further column to a ![Image 5: {\displaystyle k\times n}](https://wikimedia.org/api/rest_v1/media/math/render/svg/8e173885ca0f29dd406df781144cd635276c68e1)-dimensional matrix ![Image 6: {\displaystyle A}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7daff47fa58cdfd29dc333def748ff5fa4c923e3). This is usually done for the purpose of performing the same [elementary row operations](https://en.wikipedia.org/wiki/Elementary_row_operations "Elementary row operations") on the augmented matrix ![Image 7: {\displaystyle (A\vert B)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/116d0c4ede70511eef9e6b0f557ebf89182c7b0f) as is done on the original one ![Image 8: {\displaystyle A}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7daff47fa58cdfd29dc333def748ff5fa4c923e3) when solving a system of linear equations by [Gaussian elimination](https://en.wikipedia.org/wiki/Gaussian_elimination "Gaussian elimination").

For example, given the matrices ![Image 9: {\displaystyle A}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7daff47fa58cdfd29dc333def748ff5fa4c923e3) and column vector ![Image 10: {\displaystyle B}](https://wikimedia.org/api/rest_v1/media/math/render/svg/47136aad860d145f75f3eed3022df827cee94d7a), where ![Image 11: {\displaystyle A={\begin{bmatrix}1&3&2\\2&0&1\\5&2&2\end{bmatrix}},\quad B={\begin{bmatrix}4\\3\\1\end{bmatrix}},}](https://wikimedia.org/api/rest_v1/media/math/render/svg/8aa03933780c8f6678a1747bb9d206a1a9b22278) the augmented matrix ![Image 12: {\displaystyle (A\vert B)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/116d0c4ede70511eef9e6b0f557ebf89182c7b0f) is ![Image 13: {\displaystyle (A|B)=\left[{\begin{array}{ccc|c}1&3&2&4\\2&0&1&3\\5&2&2&1\end{array}}\right].}](https://wikimedia.org/api/rest_v1/media/math/render/svg/9f14272674b4b17491481e7ae9bcfcbc168ccb58)

For a given number ![Image 14: {\displaystyle n}](https://wikimedia.org/api/rest_v1/media/math/render/svg/a601995d55609f2d9f5e233e36fbe9ea26011b3b) of unknowns, the number of solutions to a system of ![Image 15: {\displaystyle k}](https://wikimedia.org/api/rest_v1/media/math/render/svg/c3c9a2c7b599b37105512c5d570edc034056dd40) linear equations depends only on the [rank](https://en.wikipedia.org/wiki/Rank_(linear_algebra) "Rank (linear algebra)") of the matrix of coefficients ![Image 16: {\displaystyle A}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7daff47fa58cdfd29dc333def748ff5fa4c923e3) representing the system and the rank of the corresponding augmented matrix ![Image 17: {\displaystyle (A\vert B)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/116d0c4ede70511eef9e6b0f557ebf89182c7b0f) where the components of ![Image 18: {\displaystyle B}](https://wikimedia.org/api/rest_v1/media/math/render/svg/47136aad860d145f75f3eed3022df827cee94d7a) consist of the right hand sides of the ![Image 19: {\displaystyle k}](https://wikimedia.org/api/rest_v1/media/math/render/svg/c3c9a2c7b599b37105512c5d570edc034056dd40) successive linear equations. According to the [Rouché–Capelli theorem](https://en.wikipedia.org/wiki/Rouch%C3%A9%E2%80%93Capelli_theorem "Rouché–Capelli theorem"), any system of linear equations

![Image 20: {\displaystyle AX=B}](https://wikimedia.org/api/rest_v1/media/math/render/svg/e51b626e9793be3e577dc5157905281ec0af0d32)
where ![Image 21: {\displaystyle X=(x_{1},\dots ,x_{n})^{T}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/5d5ecede666a1753ff62c90ec7330cdfb258e06a) is the ![Image 22: {\displaystyle n}](https://wikimedia.org/api/rest_v1/media/math/render/svg/a601995d55609f2d9f5e233e36fbe9ea26011b3b)-component column vector whose entries are the unknowns of the system is [inconsistent](https://en.wikipedia.org/wiki/System_of_linear_equations#Consistency "System of linear equations") (has no solutions) if the [rank](https://en.wikipedia.org/wiki/Rank_(linear_algebra) "Rank (linear algebra)") of the augmented matrix ![Image 23: {\displaystyle (A\vert B)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/116d0c4ede70511eef9e6b0f557ebf89182c7b0f) is greater than the rank of the [coefficient matrix](https://en.wikipedia.org/wiki/Coefficient_matrix "Coefficient matrix")![Image 24: {\displaystyle A}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7daff47fa58cdfd29dc333def748ff5fa4c923e3) . If, on the other hand, the ranks of these two matrices are equal, the system must have at least one solution. The solution is unique if and only if the rank equals the number of variables ![Image 25: {\displaystyle n}](https://wikimedia.org/api/rest_v1/media/math/render/svg/a601995d55609f2d9f5e233e36fbe9ea26011b3b). Otherwise the general solution has ![Image 26: {\displaystyle j}](https://wikimedia.org/api/rest_v1/media/math/render/svg/2f461e54f5c093e92a55547b9764291390f0b5d0) free parameters where ![Image 27: {\displaystyle j}](https://wikimedia.org/api/rest_v1/media/math/render/svg/2f461e54f5c093e92a55547b9764291390f0b5d0) is the difference between the number of variables ![Image 28: {\displaystyle n}](https://wikimedia.org/api/rest_v1/media/math/render/svg/a601995d55609f2d9f5e233e36fbe9ea26011b3b) and the rank. In such a case there as an affine space of solutions of dimension equal to this difference.

The inverse of a nonsingular square matrix ![Image 29: {\displaystyle A}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7daff47fa58cdfd29dc333def748ff5fa4c923e3) of dimension ![Image 30: {\displaystyle n\times n}](https://wikimedia.org/api/rest_v1/media/math/render/svg/59d2b4cb72e304526cf5b5887147729ea259da78) may be found by appending the ![Image 31: {\displaystyle n\times n}](https://wikimedia.org/api/rest_v1/media/math/render/svg/59d2b4cb72e304526cf5b5887147729ea259da78)[identity matrix](https://en.wikipedia.org/wiki/Identity_matrix "Identity matrix")![Image 32: {\displaystyle \mathbf {I} }](https://wikimedia.org/api/rest_v1/media/math/render/svg/8a458c8aeb096ce732abf346ae8edf3e4f53a126) to the right of ![Image 33: {\displaystyle A}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7daff47fa58cdfd29dc333def748ff5fa4c923e3) to form the ![Image 34: {\displaystyle n\times 2n}](https://wikimedia.org/api/rest_v1/media/math/render/svg/000d52160506e70fa3a3e166969e51200ebc42bc) dimensional augmented matrix ![Image 35: {\displaystyle (A\vert \mathbf {I} )}](https://wikimedia.org/api/rest_v1/media/math/render/svg/bbdfbcec7164b4fc9c3ae2025da95741306d2345). Applying elementary row operations to transform the left-hand ![Image 36: {\displaystyle n\times n}](https://wikimedia.org/api/rest_v1/media/math/render/svg/59d2b4cb72e304526cf5b5887147729ea259da78) block to the [identity matrix](https://en.wikipedia.org/wiki/Identity_matrix "Identity matrix")![Image 37: {\displaystyle \mathbf {I} }](https://wikimedia.org/api/rest_v1/media/math/render/svg/8a458c8aeb096ce732abf346ae8edf3e4f53a126), the right-hand ![Image 38: {\displaystyle n\times n}](https://wikimedia.org/api/rest_v1/media/math/render/svg/59d2b4cb72e304526cf5b5887147729ea259da78) block is then the inverse matrix ![Image 39: {\displaystyle A^{-1}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/83ba3a7118652cffd5de466dc439ee9184371d50)

Example of finding the inverse of a matrix
------------------------------------------

[[edit](https://en.wikipedia.org/w/index.php?title=Augmented_matrix&action=edit&section=1 "Edit section: Example of finding the inverse of a matrix")]

Let ![Image 40: {\displaystyle A}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7daff47fa58cdfd29dc333def748ff5fa4c923e3) be the square 2×2 matrix ![Image 41: {\displaystyle A={\begin{bmatrix}1&3\\-5&0\end{bmatrix}}.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/82e4b8749e634ba7d2d95f670f90c8a39a125da4)

To find the inverse of ![Image 42: {\displaystyle A}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7daff47fa58cdfd29dc333def748ff5fa4c923e3) we form the augmented matrix ![Image 43: {\displaystyle (A\vert \mathbf {I} _{2})}](https://wikimedia.org/api/rest_v1/media/math/render/svg/349bf5a76be83cf0c1ca6fa8c92b3aa579f09a83) where ![Image 44: {\displaystyle \mathbf {I} _{2}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/b6c26a33e26a53cf4c97e01e4a871dc271abaea6) is the ![Image 45: {\displaystyle 2\times 2}](https://wikimedia.org/api/rest_v1/media/math/render/svg/f8a0e3400ffb97d67c00267ed50cddfe824cbe80)[identity matrix](https://en.wikipedia.org/wiki/Identity_matrix "Identity matrix"). We then reduce the part of ![Image 46: {\displaystyle (A\vert \mathbf {I} _{2})}](https://wikimedia.org/api/rest_v1/media/math/render/svg/349bf5a76be83cf0c1ca6fa8c92b3aa579f09a83) corresponding to ![Image 47: {\displaystyle A}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7daff47fa58cdfd29dc333def748ff5fa4c923e3) to the identity matrix using [elementary row operations](https://en.wikipedia.org/wiki/Elementary_row_operations "Elementary row operations") on ![Image 48: {\displaystyle (A\vert \mathbf {I} _{2})}](https://wikimedia.org/api/rest_v1/media/math/render/svg/349bf5a76be83cf0c1ca6fa8c92b3aa579f09a83) . ![Image 49: {\displaystyle (A\vert \mathbf {I} _{2})=\left[{\begin{array}{cc|cc}1&3&1&0\\-5&0&0&1\end{array}}\right]}](https://wikimedia.org/api/rest_v1/media/math/render/svg/18c2e09362190134833dbf61386d1d1781a45fcd)![Image 50: {\displaystyle (I|A^{-1})=\left[{\begin{array}{cc|cc}1&0&0&-{\frac {1}{5}}\\0&1&{\frac {1}{3}}&{\frac {1}{15}}\end{array}}\right],}](https://wikimedia.org/api/rest_v1/media/math/render/svg/f715df704af30e8145062d80a27b921bf9da9d80) the right part of which is the inverse ![Image 51: {\displaystyle A^{-1}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/83ba3a7118652cffd5de466dc439ee9184371d50).

Existence and number of solutions
---------------------------------

[[edit](https://en.wikipedia.org/w/index.php?title=Augmented_matrix&action=edit&section=2 "Edit section: Existence and number of solutions")]

Consider the system of equations ![Image 52: {\displaystyle {\begin{aligned}x+y+2z&=2\\x+y+z&=3\\2x+2y+2z&=6.\end{aligned}}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/e416fc84dd870472c62fc797be52a7d2fea68d60)

The coefficient matrix is ![Image 53: {\displaystyle A={\begin{bmatrix}1&1&2\\1&1&1\\2&2&2\\\end{bmatrix}},}](https://wikimedia.org/api/rest_v1/media/math/render/svg/f6a44433047b7223b5e8cae39961b2370fa92df1) and the augmented matrix is ![Image 54: {\displaystyle (A|B)=\left[{\begin{array}{ccc|c}1&1&2&2\\1&1&1&3\\2&2&2&6\end{array}}\right].}](https://wikimedia.org/api/rest_v1/media/math/render/svg/ddd62594a9d0a3e473d280b74cb6d643c4544272)

Since both of these have the same rank, namely 2, there exists at least one solution; and since their rank is less than the number of unknowns, the latter being 3, there are an infinite number of solutions.

In contrast, consider the system ![Image 55: {\displaystyle {\begin{aligned}x+y+2z&=3\\x+y+z&=1\\2x+2y+2z&=5.\end{aligned}}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/b57e8b7c3e61fa29f8e60b38a2fa911ec248a794)

The coefficient matrix is ![Image 56: {\displaystyle A={\begin{bmatrix}1&1&2\\1&1&1\\2&2&2\\\end{bmatrix}},}](https://wikimedia.org/api/rest_v1/media/math/render/svg/f6a44433047b7223b5e8cae39961b2370fa92df1) and the augmented matrix is ![Image 57: {\displaystyle (A|B)=\left[{\begin{array}{ccc|c}1&1&2&3\\1&1&1&1\\2&2&2&5\end{array}}\right].}](https://wikimedia.org/api/rest_v1/media/math/render/svg/d8d486765032f65d6f336123d4e956685ad6257d)

In this example the coefficient matrix has rank 2 while the augmented matrix has rank 3; so this system of equations has no solution. Indeed, an increase in the number of linearly independent rows has made the system of equations **inconsistent**.

Solution of a linear system
---------------------------

[[edit](https://en.wikipedia.org/w/index.php?title=Augmented_matrix&action=edit&section=3 "Edit section: Solution of a linear system")]

As used in linear algebra, an augmented matrix is used to represent the [coefficients](https://en.wikipedia.org/wiki/Coefficients "Coefficients") and the solution vector of each equation set. For the set of equations ![Image 58: {\displaystyle {\begin{aligned}x+2y+3z&=0\\3x+4y+7z&=2\\6x+5y+9z&=11\end{aligned}}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/a1fe8a9fc70039035c0020cd369a7e64b44f000d) the coefficients and constant terms give the matrices ![Image 59: {\displaystyle A={\begin{bmatrix}1&2&3\\3&4&7\\6&5&9\end{bmatrix}},\quad B={\begin{bmatrix}0\\2\\11\end{bmatrix}},}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7f2fc739d8d4c4229dfcb13b636fe33ff4f8e4b9) and hence give the augmented matrix ![Image 60: {\displaystyle (A|B)=\left[{\begin{array}{ccc|c}1&2&3&0\\3&4&7&2\\6&5&9&11\end{array}}\right].}](https://wikimedia.org/api/rest_v1/media/math/render/svg/9eb7c23bbf2f8af0089b7a231c325be643bf8fe3)

Note that the rank of the coefficient matrix, which is 3, equals the rank of the augmented matrix, so at least one solution exists; and since this rank equals the number of unknowns, there is exactly one solution.

To obtain the solution, row operations can be performed on the augmented matrix to obtain the identity matrix on the left side, yielding ![Image 61: {\displaystyle \left[{\begin{array}{ccc|r}1&0&0&4\\0&1&0&1\\0&0&1&-2\\\end{array}}\right],}](https://wikimedia.org/api/rest_v1/media/math/render/svg/e3dcb9c270edb7a4fdeb0d06b5717409e47c5df2) so the solution of the system is (_x_, _y_, _z_) = (4, 1, −2).
