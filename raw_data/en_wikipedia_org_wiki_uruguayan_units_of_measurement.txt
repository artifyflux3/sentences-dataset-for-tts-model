Metric system was optional in Uruguay since 1866, and has been compulsory since 1894.

[ 1 ] [ 2 ] System before metric system [ edit ] The older system was the Spanish (Castilian) system with some modification.

[ 1 ] [ 3 ] Mass [ edit ] One libra was equal to 460 g (1.014286 lb).

[ 3 ] Capacity [ edit ] One fanega was equal to 274 L (7.776 bushels).
