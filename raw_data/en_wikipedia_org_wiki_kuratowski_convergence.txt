First introduced by Paul Painlevé in lectures on mathematical analysis in 1902, [ 1 ] the concept was popularized in texts by Felix Hausdorff [ 2 ] and Kazimierz Kuratowski .

[ 3 ] Intuitively, the Kuratowski limit of a sequence of sets is where the sets " accumulate ".

Definitions [ edit ] For a given sequence { x n } n = 1 ∞ ∞ {\displaystyle \{x_{n}\}_{n=1}^{\infty }} of points in a space X {\displaystyle X} , a limit point of the sequence can be understood as any point x ∈ ∈ X {\displaystyle x\in X} where the sequence eventually becomes arbitrarily close to x {\displaystyle x} . On the other hand, a cluster point of the sequence can be thought of as a point x ∈ ∈ X {\displaystyle x\in X} where the sequence frequently becomes arbitrarily close to x {\displaystyle x} . The Kuratowski limits inferior and superior generalize this intuition of limit and cluster points to subsets of the given space X {\displaystyle X} .

Metric Spaces [ edit ] Let ( X , d ) {\displaystyle (X,d)} be a metric space , where X {\displaystyle X} is a given set. For any point x {\displaystyle x} and any non-empty subset A ⊂ ⊂ X {\displaystyle A\subset X} , define the distance between the point and the subset: d ( x , A ) := inf y ∈ ∈ A d ( x , y ) , x ∈ ∈ X .

{\displaystyle d(x,A):=\inf _{y\in A}d(x,y),\qquad x\in X.} For any sequence of subsets { A n } n = 1 ∞ ∞ {\displaystyle \{A_{n}\}_{n=1}^{\infty }} of X {\displaystyle X} , the Kuratowski limit inferior (or lower closed limit ) of A n {\displaystyle A_{n}} as n → → ∞ ∞ {\displaystyle n\to \infty } ; is L i ⁡ ⁡ A n := { x ∈ ∈ X : for all open neighbourhoods U of x , U ∩ ∩ A n ≠ ≠ ∅ ∅ for large enough n } = { x ∈ ∈ X : lim sup n → → ∞ ∞ d ( x , A n ) = 0 } ; {\displaystyle {\begin{aligned}\mathop {\mathrm {Li} } A_{n}:=&\left\{x\in X:{\begin{matrix}{\mbox{for all open neighbourhoods }}U{\mbox{ of }}x,U\cap A_{n}\neq \varnothing {\mbox{ for large enough }}n\end{matrix}}\right\}\\=&\left\{x\in X:\limsup _{n\to \infty }d(x,A_{n})=0\right\};\end{aligned}}} the Kuratowski limit superior (or upper closed limit ) of A n {\displaystyle A_{n}} as n → → ∞ ∞ {\displaystyle n\to \infty } ; is L s ⁡ ⁡ A n := { x ∈ ∈ X : for all open neighbourhoods U of x , U ∩ ∩ A n ≠ ≠ ∅ ∅ for infinitely many n } = { x ∈ ∈ X : lim inf n → → ∞ ∞ d ( x , A n ) = 0 } ; {\displaystyle {\begin{aligned}\mathop {\mathrm {Ls} } A_{n}:=&\left\{x\in X:{\begin{matrix}{\mbox{for all open neighbourhoods }}U{\mbox{ of }}x,U\cap A_{n}\neq \varnothing {\mbox{ for infinitely many }}n\end{matrix}}\right\}\\=&\left\{x\in X:\liminf _{n\to \infty }d(x,A_{n})=0\right\};\end{aligned}}} If the Kuratowski limits inferior and superior agree, then the common set is called the Kuratowski limit of A n {\displaystyle A_{n}} and is denoted L i m n → → ∞ ∞ ⁡ ⁡ A n {\displaystyle \mathop {\mathrm {Lim} } _{n\to \infty }A_{n}} .

Topological Spaces [ edit ] If ( X , τ τ ) {\textstyle (X,\tau )} is a topological space , and { A i } i ∈ ∈ I {\textstyle \{A_{i}\}_{i\in I}} are a net of subsets of X {\textstyle X} , the limits inferior and superior follow a similar construction. For a given point x ∈ ∈ X {\textstyle x\in X} denote N ( x ) {\textstyle {\mathcal {N}}(x)} the collection of open neighborhoods of x {\textstyle x} . The Kuratowski limit inferior of { A i } i ∈ ∈ I {\textstyle \{A_{i}\}_{i\in I}} is the set L i ⁡ ⁡ A i := { x ∈ ∈ X : ∀ ∀ U ∈ ∈ N ( x ) ∃ ∃ i 0 ∈ ∈ I : : ∀ ∀ i ≥ ≥ i 0 ⇒ ⇒ U ∩ ∩ A i ≠ ≠ ∅ ∅ } , {\displaystyle \mathop {\mathrm {Li} } A_{i}:=\left\{x\in X:\forall U\in {\mathcal {N}}(x)\,\exists i_{0}\in I\colon \forall i\geq i_{0}\Rightarrow U\cap A_{i}\neq \varnothing \right\},} and the Kuratowski limit superior is the set L s ⁡ ⁡ A i := { x ∈ ∈ X : ∀ ∀ U ∈ ∈ N ( x ) ∀ ∀ i 0 ∈ ∈ I ∃ ∃ i ≥ ≥ i 0 : : U ∩ ∩ A i ≠ ≠ ∅ ∅ } .

{\displaystyle \mathop {\mathrm {Ls} } A_{i}:=\left\{x\in X:\forall U\in {\mathcal {N}}(x)\,\forall i_{0}\in I\,\exists i\geq i_{0}\colon U\cap A_{i}\neq \varnothing \right\}.} Elements of L i ⁡ ⁡ A i {\textstyle \mathop {\mathrm {Li} } A_{i}} are called limit points of { A i } i ∈ ∈ I {\textstyle \{A_{i}\}_{i\in I}} and elements of L s ⁡ ⁡ A i {\textstyle \mathop {\mathrm {Ls} } A_{i}} are called cluster points of { A i } i ∈ ∈ I {\textstyle \{A_{i}\}_{i\in I}} . In other words, x {\displaystyle x} is a limit point of { A i } i ∈ ∈ I {\textstyle \{A_{i}\}_{i\in I}} if each of its neighborhoods intersects A i {\displaystyle A_{i}} for all i {\displaystyle i} in a "residual" subset of I {\displaystyle I} , while x {\displaystyle x} is a cluster point of { A i } i ∈ ∈ I {\textstyle \{A_{i}\}_{i\in I}} if each of its neighborhoods intersects A i {\displaystyle A_{i}} for all i {\displaystyle i} in a cofinal subset of I {\displaystyle I} .

When these sets agree, the common set is the Kuratowski limit of { A i } i ∈ ∈ I {\textstyle \{A_{i}\}_{i\in I}} , denoted L i m ⁡ ⁡ A i {\displaystyle \mathop {\mathrm {Lim} } A_{i}} .

Examples [ edit ] Suppose ( X , d ) {\displaystyle (X,d)} is separable where X {\displaystyle X} is a perfect set, and let D = { d 1 , d 2 , … … } {\displaystyle D=\{d_{1},d_{2},\dots \}} be an enumeration of a countable dense subset of X {\displaystyle X} . Then the sequence { A n } n = 1 ∞ ∞ {\displaystyle \{A_{n}\}_{n=1}^{\infty }} defined by A n := { d 1 , d 2 , … … , d n } {\displaystyle A_{n}:=\{d_{1},d_{2},\dots ,d_{n}\}} has L i m ⁡ ⁡ A n = X {\displaystyle \mathop {\mathrm {Lim} } A_{n}=X} .

Given two closed subsets B , C ⊂ ⊂ X {\displaystyle B,C\subset X} , defining A 2 n − − 1 := B {\displaystyle A_{2n-1}:=B} and A 2 n := C {\displaystyle A_{2n}:=C} for each n = 1 , 2 , … … {\displaystyle n=1,2,\dots } yields L i ⁡ ⁡ A n = B ∩ ∩ C {\displaystyle \mathop {\mathrm {Li} } A_{n}=B\cap C} and L s ⁡ ⁡ A n = B ∪ ∪ C {\displaystyle \mathop {\mathrm {Ls} } A_{n}=B\cup C} .

The sequence of closed balls A n := { y ∈ ∈ X : d ( x n , y ) ≤ ≤ r n } {\displaystyle A_{n}:=\{y\in X:d(x_{n},y)\leq r_{n}\}} converges in the sense of Kuratowski when x n → → x {\displaystyle x_{n}\to x} in X {\displaystyle X} and r n → → r {\displaystyle r_{n}\to r} in [ 0 , + ∞ ∞ ) {\displaystyle [0,+\infty )} , and in particular, L i m ⁡ ⁡ ( A n ) = { y ∈ ∈ X : d ( x , y ) ≤ ≤ r } {\displaystyle \mathop {\mathrm {Lim} } (A_{n})=\{y\in X:d(x,y)\leq r\}} . If r n → → + ∞ ∞ {\displaystyle r_{n}\to +\infty } , then L i m ⁡ ⁡ A n = X {\displaystyle \mathop {\mathrm {Lim} } A_{n}=X} while L i m ⁡ ⁡ ( X ∖ ∖ A n ) = ∅ ∅ {\displaystyle \mathop {\mathrm {Lim} } (X\setminus A_{n})=\emptyset } .

Let A n := { x ∈ ∈ R : sin ⁡ ⁡ ( n x ) = 0 } {\textstyle A_{n}:=\{x\in \mathbb {R} :\sin(nx)=0\}} . Then A n {\displaystyle A_{n}} converges in the Kuratowski sense to the entire line.

In a topological vector space , if { A n } n = 1 ∞ ∞ {\displaystyle \{A_{n}\}_{n=1}^{\infty }} is a sequence of cones , then so are the Kuratowski limits superior and inferior. For example, the sets A n := { ( x , y ) ∈ ∈ R 2 : y ≥ ≥ n | x | } {\displaystyle A_{n}:=\{(x,y)\in \mathbb {R} ^{2}:y\geq n|x|\}} converge to { ( 0 , y ) ∈ ∈ R 2 : y ≥ ≥ 0 } {\displaystyle \{(0,y)\in \mathbb {R} ^{2}:y\geq 0\}} .

Properties [ edit ] The following properties hold for the limits inferior and superior in both the metric and topological contexts, but are stated in the metric formulation for ease of reading.

[ 4 ] Both L i ⁡ ⁡ A n {\displaystyle \mathop {\mathrm {Li} } A_{n}} and L s ⁡ ⁡ A n {\displaystyle \mathop {\mathrm {Ls} } A_{n}} are closed subsets of X {\displaystyle X} , and L i ⁡ ⁡ A n ⊂ ⊂ L s ⁡ ⁡ A n {\displaystyle \mathop {\mathrm {Li} } A_{n}\subset \mathop {\mathrm {Ls} } A_{n}} always holds.

The upper and lower limits do not distinguish between sets and their closures : L i ⁡ ⁡ A n = L i ⁡ ⁡ c l ⁡ ⁡ ( A n ) {\displaystyle \mathop {\mathrm {Li} } A_{n}=\mathop {\mathrm {Li} } \mathop {\mathrm {cl} } (A_{n})} and L s ⁡ ⁡ A n = L s ⁡ ⁡ c l ⁡ ⁡ ( A n ) {\displaystyle \mathop {\mathrm {Ls} } A_{n}=\mathop {\mathrm {Ls} } \mathop {\mathrm {cl} } (A_{n})} .

If A n := A {\displaystyle A_{n}:=A} is a constant sequence, then L i m ⁡ ⁡ A n = c l ⁡ ⁡ A {\displaystyle \mathop {\mathrm {Lim} } A_{n}=\mathop {\mathrm {cl} } A} .

If A n := { x n } {\displaystyle A_{n}:=\{x_{n}\}} is a sequence of singletons, then L i ⁡ ⁡ A n {\displaystyle \mathop {\mathrm {Li} } A_{n}} and L s ⁡ ⁡ A n {\displaystyle \mathop {\mathrm {Ls} } A_{n}} consist of the limit points and cluster points, respectively, of the sequence { x n } n = 1 ∞ ∞ ⊂ ⊂ X {\displaystyle \{x_{n}\}_{n=1}^{\infty }\subset X} .

If A n ⊂ ⊂ B n ⊂ ⊂ C n {\displaystyle A_{n}\subset B_{n}\subset C_{n}} and B := L i m ⁡ ⁡ A n = L i m ⁡ ⁡ C n {\displaystyle B:=\mathop {\mathrm {Lim} } A_{n}=\mathop {\mathrm {Lim} } C_{n}} , then L i m ⁡ ⁡ B n = B {\displaystyle \mathop {\mathrm {Lim} } B_{n}=B} .

( Hit and miss criteria ) For a closed subset A ⊂ ⊂ X {\displaystyle A\subset X} , one has A ⊂ ⊂ L i ⁡ ⁡ A n {\displaystyle A\subset \mathop {\mathrm {Li} } A_{n}} , if and only if for every open set U ⊂ ⊂ X {\displaystyle U\subset X} with A ∩ ∩ U ≠ ≠ ∅ ∅ {\displaystyle A\cap U\neq \emptyset } there exists n 0 {\displaystyle n_{0}} such that A n ∩ ∩ U ≠ ≠ ∅ ∅ {\displaystyle A_{n}\cap U\neq \emptyset } for all n 0 ≤ ≤ n {\displaystyle n_{0}\leq n} , L s ⁡ ⁡ A n ⊂ ⊂ A {\displaystyle \mathop {\mathrm {Ls} } A_{n}\subset A} , if and only if for every compact set K ⊂ ⊂ X {\displaystyle K\subset X} with A ∩ ∩ K = ∅ ∅ {\displaystyle A\cap K=\emptyset } there exists n 0 {\displaystyle n_{0}} such that A n ∩ ∩ K = ∅ ∅ {\displaystyle A_{n}\cap K=\emptyset } for all n 0 ≤ ≤ n {\displaystyle n_{0}\leq n} .

If A 1 ⊂ ⊂ A 2 ⊂ ⊂ A 3 ⊂ ⊂ ⋯ ⋯ {\displaystyle A_{1}\subset A_{2}\subset A_{3}\subset \cdots } then the Kuratowski limit exists, and L i m ⁡ ⁡ A n = c l ⁡ ⁡ ( ⋃ ⋃ n = 1 ∞ ∞ A n ) {\textstyle \mathop {\mathrm {Lim} } A_{n}=\mathop {\mathrm {cl} } \left(\bigcup _{n=1}^{\infty }A_{n}\right)} . Conversely, if A 1 ⊃ ⊃ A 2 ⊃ ⊃ A 3 ⊃ ⊃ ⋯ ⋯ {\displaystyle A_{1}\supset A_{2}\supset A_{3}\supset \cdots } then the Kuratowski limit exists, and L i m ⁡ ⁡ A n = ⋂ ⋂ n = 1 ∞ ∞ c l ⁡ ⁡ ( A n ) {\textstyle \mathop {\mathrm {Lim} } A_{n}=\bigcap _{n=1}^{\infty }\mathop {\mathrm {cl} } (A_{n})} .

If d H {\displaystyle d_{H}} denotes Hausdorff metric , then d H ( A n , A ) → → 0 {\displaystyle d_{H}(A_{n},A)\to 0} implies c l ⁡ ⁡ A = L i m ⁡ ⁡ A n {\displaystyle \mathop {\mathrm {cl} } A=\mathop {\mathrm {Lim} } A_{n}} . However, noncompact closed sets may converge in the sense of Kuratowski while d H ( A n , L i m ⁡ ⁡ A n ) = + ∞ ∞ {\displaystyle d_{H}(A_{n},\mathop {\mathrm {Lim} } A_{n})=+\infty } for each n = 1 , 2 , … … {\displaystyle n=1,2,\dots } [ 5 ] Convergence in the sense of Kuratowski is weaker than convergence in the sense of Vietoris but equivalent to convergence in the sense of Fell. If X {\displaystyle X} is compact, then these are all equivalent and agree with convergence in Hausdorff metric.

Kuratowski Continuity of Set-Valued Functions [ edit ] Let S : X ⇉ ⇉ Y {\displaystyle S:X\rightrightarrows Y} be a set-valued function between the spaces X {\displaystyle X} and Y {\displaystyle Y} ; namely, S ( x ) ⊂ ⊂ Y {\displaystyle S(x)\subset Y} for all x ∈ ∈ X {\displaystyle x\in X} . Denote S − − 1 ( y ) = { x ∈ ∈ X : y ∈ ∈ S ( x ) } {\displaystyle S^{-1}(y)=\{x\in X:y\in S(x)\}} . We can define the operators L i x ′ → → x ⁡ ⁡ S ( x ′ ) := ⋂ ⋂ x ′ → → x L i ⁡ ⁡ S ( x ′ ) , x ∈ ∈ X L s x ′ → → x ⁡ ⁡ S ( x ′ ) := ⋃ ⋃ x ′ → → x L s ⁡ ⁡ S ( x ′ ) , x ∈ ∈ X {\displaystyle {\begin{aligned}\mathop {\mathrm {Li} } _{x'\to x}S(x'):=&\bigcap _{x'\to x}\mathop {\mathrm {Li} } S(x'),\qquad x\in X\\\mathop {\mathrm {Ls} } _{x'\to x}S(x'):=&\bigcup _{x'\to x}\mathop {\mathrm {Ls} } S(x'),\qquad x\in X\\\end{aligned}}} where x ′ → → x {\displaystyle x'\to x} means convergence in sequences when X {\displaystyle X} is metrizable and convergence in nets otherwise. Then, S {\displaystyle S} is inner semi-continuous at x ∈ ∈ X {\displaystyle x\in X} if S ( x ) ⊂ ⊂ L i x ′ → → x ⁡ ⁡ S ( x ′ ) {\textstyle S(x)\subset \mathop {\mathrm {Li} } _{x'\to x}S(x')} ; S {\displaystyle S} is outer semi-continuous at x ∈ ∈ X {\displaystyle x\in X} if L s x ′ → → x ⁡ ⁡ S ( x ′ ) ⊂ ⊂ S ( x ) {\textstyle \mathop {\mathrm {Ls} } _{x'\to x}S(x')\subset S(x)} .

When S {\displaystyle S} is both inner and outer semi-continuous at x ∈ ∈ X {\displaystyle x\in X} , we say that S {\displaystyle S} is continuous (or continuous in the sense of Kuratowski ).

Continuity of set-valued functions is commonly defined in terms of lower- and upper-hemicontinuity popularized by Berge .

[ 6 ] In this sense, a set-valued function is continuous if and only if the function f S : X → → 2 Y {\displaystyle f_{S}:X\to 2^{Y}} defined by f ( x ) = S ( x ) {\displaystyle f(x)=S(x)} is continuous with respect to the Vietoris hyperspace topology of 2 Y {\displaystyle 2^{Y}} . For set-valued functions with closed values, continuity in the sense of Vietoris-Berge is stronger than continuity in the sense of Kuratowski.

Examples [ edit ] The set-valued function B ( x , r ) = { y ∈ ∈ X : d ( x , y ) ≤ ≤ r } {\displaystyle B(x,r)=\{y\in X:d(x,y)\leq r\}} is continuous X × × [ 0 , + ∞ ∞ ) ⇉ ⇉ X {\displaystyle X\times [0,+\infty )\rightrightarrows X} .

Given a function f : X → → [ − − ∞ ∞ , + ∞ ∞ ] {\displaystyle f:X\to [-\infty ,+\infty ]} , the superlevel set mapping S f ( x ) := { λ λ ∈ ∈ R : f ( x ) ≤ ≤ λ λ } {\displaystyle S_{f}(x):=\{\lambda \in \mathbb {R} :f(x)\leq \lambda \}} is outer semi-continuous at x {\displaystyle x} , if and only if f {\displaystyle f} is lower semi-continuous at x {\displaystyle x} . Similarly, S f {\displaystyle S_{f}} is inner semi-continuous at x {\displaystyle x} , if and only if f {\displaystyle f} is upper semi-continuous at x {\displaystyle x} .

Properties [ edit ] If S {\displaystyle S} is continuous at x {\displaystyle x} , then S ( x ) {\displaystyle S(x)} is closed.

S {\displaystyle S} is outer semi-continuous at x {\displaystyle x} , if and only if for every y ∉ ∉ S ( x ) {\displaystyle y\notin S(x)} there are neighborhoods V ∈ ∈ N ( y ) {\displaystyle V\in {\mathcal {N}}(y)} and U ∈ ∈ N ( x ) {\displaystyle U\in {\mathcal {N}}(x)} such that U ∩ ∩ S − − 1 ( V ) = ∅ ∅ {\displaystyle U\cap S^{-1}(V)=\emptyset } .

S {\displaystyle S} is inner semi-continuous at x {\displaystyle x} , if and only if for every y ∈ ∈ S ( x ) {\displaystyle y\in S(x)} and neighborhood V ∈ ∈ N ( y ) {\displaystyle V\in {\mathcal {N}}(y)} there is a neighborhood U ∈ ∈ N ( x ) {\displaystyle U\in {\mathcal {N}}(x)} such that V ∩ ∩ S ( x ′ ) ≠ ≠ ∅ ∅ {\displaystyle V\cap S(x')\neq \emptyset } for all x ′ ∈ ∈ U {\displaystyle x'\in U} .

S {\displaystyle S} is (globally) outer semi-continuous, if and only if its graph { ( x , y ) ∈ ∈ X × × Y : y ∈ ∈ S ( x ) } {\displaystyle \{(x,y)\in X\times Y:y\in S(x)\}} is closed.

( Relations to Vietoris-Berge continuity ). Suppose S ( x ) {\displaystyle S(x)} is closed.

S {\displaystyle S} is inner semi-continuous at x {\displaystyle x} , if and only if S {\displaystyle S} is lower hemi-continuous at x {\displaystyle x} in the sense of Vietoris-Berge.

If S {\displaystyle S} is upper hemi-continuous at x {\displaystyle x} , then S {\displaystyle S} is outer semi-continuous at x {\displaystyle x} . The converse is false in general, but holds when Y {\displaystyle Y} is a compact space.

If S : R n → → R m {\displaystyle S:\mathbb {R} ^{n}\to \mathbb {R} ^{m}} has a convex graph, then S {\displaystyle S} is inner semi-continuous at each point of the interior of the domain of S {\displaystyle S} . Conversely, given any inner semi-continuous set-valued function S {\displaystyle S} , the convex hull mapping T ( x ) := c o n v ⁡ ⁡ S ( x ) {\displaystyle T(x):=\mathop {\mathrm {conv} } S(x)} is also inner semi-continuous.

Epi-convergence and Γ-convergence [ edit ] Main articles: Epi-convergence and Γ-convergence For the metric space ( X , d ) {\displaystyle (X,d)} a sequence of functions f n : X → → [ − − ∞ ∞ , + ∞ ∞ ] {\displaystyle f_{n}:X\to [-\infty ,+\infty ]} , the epi-limit inferior (or lower epi-limit ) is the function e lim inf ⁡ ⁡ f n {\displaystyle \mathop {\mathrm {e} \liminf } f_{n}} defined by the epigraph equation e p i ⁡ ⁡ ( e lim inf ⁡ ⁡ f n ) := L s ⁡ ⁡ ( e p i ⁡ ⁡ f n ) , {\displaystyle \mathop {\mathrm {epi} } \left(\mathop {\mathrm {e} \liminf } f_{n}\right):=\mathop {\mathrm {Ls} } \left(\mathop {\mathrm {epi} } f_{n}\right),} and similarly the epi-limit superior (or upper epi-limit ) is the function e lim sup ⁡ ⁡ f n {\displaystyle \mathop {\mathrm {e} \limsup } f_{n}} defined by the epigraph equation e p i ⁡ ⁡ ( e lim sup ⁡ ⁡ f n ) := L i ⁡ ⁡ ( e p i ⁡ ⁡ f n ) .

{\displaystyle \mathop {\mathrm {epi} } \left(\mathop {\mathrm {e} \limsup } f_{n}\right):=\mathop {\mathrm {Li} } \left(\mathop {\mathrm {epi} } f_{n}\right).} Since Kuratowski upper and lower limits are closed sets, it follows that both e lim inf ⁡ ⁡ f n {\displaystyle \mathop {\mathrm {e} \liminf } f_{n}} and e lim sup ⁡ ⁡ f n {\displaystyle \mathop {\mathrm {e} \limsup } f_{n}} are lower semi-continuous functions. Similarly, since L i ⁡ ⁡ e p i ⁡ ⁡ f n ⊂ ⊂ L s ⁡ ⁡ e p i ⁡ ⁡ f n {\displaystyle \mathop {\mathrm {Li} } \mathop {\mathrm {epi} } f_{n}\subset \mathop {\mathrm {Ls} } \mathop {\mathrm {epi} } f_{n}} , it follows that e lim inf ⁡ ⁡ f n ≤ ≤ e lim inf ⁡ ⁡ f n {\displaystyle \mathop {\mathrm {e} \liminf } f_{n}\leq \mathop {\mathrm {e} \liminf } f_{n}} uniformly. These functions agree, if and only if L i m ⁡ ⁡ e p i ⁡ ⁡ f n {\displaystyle \mathop {\mathrm {Lim} } \mathop {\mathrm {epi} } f_{n}} exists, and the associated function is called the epi-limit of { f n } n = 1 ∞ ∞ {\displaystyle \{f_{n}\}_{n=1}^{\infty }} .

When ( X , τ τ ) {\displaystyle (X,\tau )} is a topological space, epi-convergence of the sequence { f n } n = 1 ∞ ∞ {\displaystyle \{f_{n}\}_{n=1}^{\infty }} is called Γ-convergence. From the perspective of Kuratowski convergence there is no distinction between epi-limits and Γ-limits. The concepts are usually studied separately, because epi-convergence admits special characterizations that rely on the metric space structure of X {\displaystyle X} , which does not hold in topological spaces generally.
