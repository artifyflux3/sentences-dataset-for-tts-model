( July 2010 ) ( Learn how and when to remove this message ) In communication , a code word is an element of a standardized code or protocol . Each code word is assembled in accordance with the specific rules of the code and assigned a unique meaning. Code words are typically used for reasons of reliability, clarity, brevity, or secrecy.
