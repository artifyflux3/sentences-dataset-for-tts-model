In [mathematics](https://en.wikipedia.org/wiki/Mathematics "Mathematics"), a **function space** is a [set](https://en.wikipedia.org/wiki/Set_(mathematics) "Set (mathematics)") of [functions](https://en.wikipedia.org/wiki/Function_(mathematics) "Function (mathematics)") between two fixed sets. Often, the [domain](https://en.wikipedia.org/wiki/Domain_of_a_function "Domain of a function") and/or [codomain](https://en.wikipedia.org/wiki/Codomain "Codomain") will have additional [structure](https://en.wikipedia.org/wiki/Mathematical_structure "Mathematical structure") which is inherited by the function space. For example, the set of functions from any set X into a [vector space](https://en.wikipedia.org/wiki/Vector_space "Vector space") has a [natural](https://en.wikipedia.org/wiki/List_of_mathematical_jargon#natural "List of mathematical jargon") vector space structure given by [pointwise](https://en.wikipedia.org/wiki/Pointwise "Pointwise") addition and scalar multiplication. In other scenarios, the function space might inherit a [topological](https://en.wikipedia.org/wiki/Topological_space "Topological space") or [metric](https://en.wikipedia.org/wiki/Metric_space "Metric space") structure, hence the name function _space_.

In linear algebra
-----------------

[[edit](https://en.wikipedia.org/w/index.php?title=Function_space&action=edit&section=1 "Edit section: In linear algebra")]

Let F be a [field](https://en.wikipedia.org/wiki/Field_(mathematics) "Field (mathematics)") and let X be any set. The functions X ‚Üí F can be given the structure of a vector space over F where the operations are defined pointwise, that is, for any f, g: X ‚Üí F, any x in X, and any c in F, define (f+g)(x)=f(x)+g(x)(c‚ãÖf)(x)=c‚ãÖf(x){\displaystyle {\begin{aligned}(f+g)(x)&=f(x)+g(x)\\(c\cdot f)(x)&=c\cdot f(x)\end{aligned}}}![Image 14: {\displaystyle {\begin{aligned}(f+g)(x)&=f(x)+g(x)\\(c\cdot f)(x)&=c\cdot f(x)\end{aligned}}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/9bd4f7381dcac15e530dcd1775869991536e70c4) When the domain X has additional structure, one might consider instead the [subset](https://en.wikipedia.org/wiki/Subset "Subset") (or [subspace](https://en.wikipedia.org/wiki/Linear_subspace "Linear subspace")) of all such functions which respect that structure. For example, if V and also X itself are vector spaces over F, the set of [linear maps](https://en.wikipedia.org/wiki/Linear_map "Linear map")X ‚Üí V form a vector space over F with pointwise operations (often denoted [Hom](https://en.wikipedia.org/wiki/Hom_set "Hom set")(X,V)). One such space is the [dual space](https://en.wikipedia.org/wiki/Dual_space "Dual space") of X: the set of [linear functionals](https://en.wikipedia.org/wiki/Linear_form "Linear form")X ‚Üí F with addition and scalar multiplication defined pointwise.

The cardinal [dimension](https://en.wikipedia.org/wiki/Dimension "Dimension") of a function space with no extra structure can be found by the [Erd≈ës‚ÄìKaplansky theorem](https://en.wikipedia.org/wiki/Erd%C5%91s%E2%80%93Kaplansky_theorem "Erd≈ës‚ÄìKaplansky theorem").

Examples
--------

[[edit](https://en.wikipedia.org/w/index.php?title=Function_space&action=edit&section=2 "Edit section: Examples")]

Function spaces appear in various areas of mathematics:

*   In [set theory](https://en.wikipedia.org/wiki/Set_theory "Set theory"), the set of functions from _X_ to _Y_ may be denoted {_X_ ‚Üí _Y_} or _Y_ _X_. 
    *   As a special case, the [power set](https://en.wikipedia.org/wiki/Power_set "Power set") of a set _X_ may be identified with the set of all functions from _X_ to {0, 1}, denoted 2 _X_.

*   The set of [bijections](https://en.wikipedia.org/wiki/Bijection "Bijection") from _X_ to _Y_ is denoted X‚ÜîY{\displaystyle X\leftrightarrow Y}![Image 15: {\displaystyle X\leftrightarrow Y}](https://wikimedia.org/api/rest_v1/media/math/render/svg/96b69b60af3edb830806b7a11982101e7facaa3c). The factorial notation _X_! may be used for permutations of a single set _X_.
*   In [functional analysis](https://en.wikipedia.org/wiki/Functional_analysis "Functional analysis"), the same is seen for [continuous](https://en.wikipedia.org/wiki/Continuous_function "Continuous function") linear transformations, including [topologies on the vector spaces](https://en.wikipedia.org/wiki/Topological_vector_space "Topological vector space") in the above, and many of the major examples are function spaces carrying a [topology](https://en.wikipedia.org/wiki/Topology "Topology"); the best known examples include [Hilbert spaces](https://en.wikipedia.org/wiki/Hilbert_space "Hilbert space") and [Banach spaces](https://en.wikipedia.org/wiki/Banach_space "Banach space").
*   In [functional analysis](https://en.wikipedia.org/wiki/Functional_analysis "Functional analysis"), the set of all functions from the [natural numbers](https://en.wikipedia.org/wiki/Natural_number "Natural number") to some set _X_ is called a _[sequence space](https://en.wikipedia.org/wiki/Sequence\_space "Sequence space")_. It consists of the set of all possible [sequences](https://en.wikipedia.org/wiki/Sequences "Sequences") of elements of _X_.
*   In [topology](https://en.wikipedia.org/wiki/Topology "Topology"), one may attempt to put a topology on the space of continuous functions from a [topological space](https://en.wikipedia.org/wiki/Topological_space "Topological space")_X_ to another one _Y_, with utility depending on the nature of the spaces. A commonly used example is the [compact-open topology](https://en.wikipedia.org/wiki/Compact-open_topology "Compact-open topology"), e.g. [loop space](https://en.wikipedia.org/wiki/Loop_space "Loop space"). Also available is the [product topology](https://en.wikipedia.org/wiki/Product_topology "Product topology") on the space of set theoretic functions (i.e. not necessarily continuous functions) _Y_ _X_. In this context, this topology is also referred to as the [topology of pointwise convergence](https://en.wikipedia.org/wiki/Topology_of_pointwise_convergence "Topology of pointwise convergence").
*   In [algebraic topology](https://en.wikipedia.org/wiki/Algebraic_topology "Algebraic topology"), the study of [homotopy theory](https://en.wikipedia.org/wiki/Homotopy_theory "Homotopy theory") is essentially that of discrete invariants of function spaces;
*   In the theory of [stochastic processes](https://en.wikipedia.org/wiki/Stochastic_process "Stochastic process"), the basic technical problem is how to construct a [probability measure](https://en.wikipedia.org/wiki/Probability_measure "Probability measure") on a function space of _paths of the process_ (functions of time);
*   In [category theory](https://en.wikipedia.org/wiki/Category_theory "Category theory"), the function space is called an [exponential object](https://en.wikipedia.org/wiki/Exponential_object "Exponential object") or [map object](https://en.wikipedia.org/wiki/Exponential_object "Exponential object"). It appears in one way as the representation [canonical bifunctor](https://en.wikipedia.org/wiki/Canonical_bifunctor "Canonical bifunctor"); but as (single) functor, of type [X,‚àí]{\displaystyle [X,-]}![Image 16: {\displaystyle [X,-]}](https://wikimedia.org/api/rest_v1/media/math/render/svg/b68c05513b50af3f5d38201b86c984b6061149c0), it appears as an [adjoint functor](https://en.wikipedia.org/wiki/Adjoint_functor "Adjoint functor") to a functor of type ‚àí√óX{\displaystyle -\times X}![Image 17: {\displaystyle -\times X}](https://wikimedia.org/api/rest_v1/media/math/render/svg/9217169749da5e8d527876e4765da9723ba7014c) on objects;
*   In [functional programming](https://en.wikipedia.org/wiki/Functional_programming "Functional programming") and [lambda calculus](https://en.wikipedia.org/wiki/Lambda_calculus "Lambda calculus"), [function types](https://en.wikipedia.org/wiki/Function_type "Function type") are used to express the idea of [higher-order functions](https://en.wikipedia.org/wiki/Higher-order_function "Higher-order function")
*   In programming more generally, many [higher-order function](https://en.wikipedia.org/wiki/Higher-order_function "Higher-order function") concepts occur with or without explicit typing, such as [closures](https://en.wikipedia.org/wiki/Closure_(computer_programming) "Closure (computer programming)").
*   In [domain theory](https://en.wikipedia.org/wiki/Domain_theory "Domain theory"), the basic idea is to find constructions from [partial orders](https://en.wikipedia.org/wiki/Partial_order "Partial order") that can model lambda calculus, by creating a well-behaved [Cartesian closed category](https://en.wikipedia.org/wiki/Cartesian_closed_category "Cartesian closed category").
*   In the [representation theory of finite groups](https://en.wikipedia.org/wiki/Representation_theory_of_finite_groups "Representation theory of finite groups"), given two finite-dimensional representations V and W of a group G, one can form a representation of G over the vector space of linear maps Hom(V,W) called the [Hom representation](https://en.wikipedia.org/wiki/Hom_representation "Hom representation").[[1]](https://en.wikipedia.org/wiki/Function_space#cite_note-1)

Functional analysis
-------------------

[[edit](https://en.wikipedia.org/w/index.php?title=Function_space&action=edit&section=3 "Edit section: Functional analysis")]

Uniform norm
------------

[[edit](https://en.wikipedia.org/w/index.php?title=Function_space&action=edit&section=4 "Edit section: Uniform norm")]

If _y_ is an element of the function space C(a,b){\displaystyle {\mathcal {C}}(a,b)}![Image 38: {\displaystyle {\mathcal {C}}(a,b)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/d89418b6b41c85096bfadf1f877d9113642270f0) of all [continuous functions](https://en.wikipedia.org/wiki/Continuous_function "Continuous function") that are defined on a [closed interval](https://en.wikipedia.org/wiki/Closed_interval "Closed interval")[_a_, _b_], the **[norm](https://en.wikipedia.org/wiki/Norm_(mathematics) "Norm (mathematics)")‚Äñy‚Äñ‚àû{\displaystyle \|y\|_{\infty }}![Image 39: {\displaystyle \|y\|_{\infty }}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7b93729fac75d797d1c804f37d0535f7d59e293f)** defined on C(a,b){\displaystyle {\mathcal {C}}(a,b)}![Image 40: {\displaystyle {\mathcal {C}}(a,b)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/d89418b6b41c85096bfadf1f877d9113642270f0) is the maximum [absolute value](https://en.wikipedia.org/wiki/Absolute_value "Absolute value") of _y_ (_x_) for _a_ ‚â§ _x_ ‚â§ _b_,[[2]](https://en.wikipedia.org/wiki/Function_space#cite_note-GelfandFominP6-2)‚Äñy‚Äñ‚àû‚â°max a‚â§x‚â§b|y(x)|where y‚ààC(a,b){\displaystyle \|y\|_{\infty }\equiv \max _{a\leq x\leq b}|y(x)|\qquad {\text{where}}\ \ y\in {\mathcal {C}}(a,b)}![Image 41: {\displaystyle \|y\|_{\infty }\equiv \max _{a\leq x\leq b}|y(x)|\qquad {\text{where}}\ \ y\in {\mathcal {C}}(a,b)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/60d7587acbb263f7f13ee81af34c93577ad55b6c)

is called the _[uniform norm](https://en.wikipedia.org/wiki/Uniform\_norm "Uniform norm")_ or _supremum norm_ ('sup norm').
