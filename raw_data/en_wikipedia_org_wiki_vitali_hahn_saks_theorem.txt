Statement of the theorem [ edit ] If ( S , B , m ) {\displaystyle (S,{\mathcal {B}},m)} is a measure space with m ( S ) < ∞ ∞ , {\displaystyle m(S)<\infty ,} and a sequence λ λ n {\displaystyle \lambda _{n}} of complex measures . Assuming that each λ λ n {\displaystyle \lambda _{n}} is absolutely continuous with respect to m , {\displaystyle m,} and that for all B ∈ ∈ B {\displaystyle B\in {\mathcal {B}}} the finite limits exist lim n → → ∞ ∞ λ λ n ( B ) = λ λ ( B ) {\displaystyle \lim _{n\to \infty }\lambda _{n}(B)=\lambda (B)} . Then the absolute continuity of the λ λ n {\displaystyle \lambda _{n}} with respect to m {\displaystyle m} is uniform in n {\displaystyle n} , that is, lim B m ( B ) = 0 {\displaystyle \lim _{B}m(B)=0} implies that lim B λ λ n ( B ) = 0 {\displaystyle \lim _{B}\lambda _{n}(B)=0} uniformly in n {\displaystyle n} . Also λ λ {\displaystyle \lambda } is countably additive on B {\displaystyle {\mathcal {B}}} .

Proof of Vitali-Hahn-Saks theorem [ edit ] Each λ λ n {\displaystyle \lambda _{n}} defines a function λ λ ¯ ¯ n ( B ¯ ¯ ) {\displaystyle {\overline {\lambda }}_{n}({\overline {B}})} on B ~ ~ {\displaystyle {\tilde {\mathcal {B}}}} by taking λ λ ¯ ¯ n ( B ¯ ¯ ) = λ λ n ( B ) {\displaystyle {\overline {\lambda }}_{n}({\overline {B}})=\lambda _{n}(B)} . This function is well defined, this is it is independent on the representative B {\displaystyle B} of the class B ¯ ¯ {\displaystyle {\overline {B}}} due to the absolute continuity of λ λ n {\displaystyle \lambda _{n}} with respect to m {\displaystyle m} . Moreover λ λ ¯ ¯ n {\displaystyle {\overline {\lambda }}_{n}} is continuous.

{\displaystyle d(B,B_{0})<\delta \Rightarrow \sup _{n\geq 1}|{\overline {\lambda }}_{k_{0}}({\overline {B}})-{\overline {\lambda }}_{k_{0}+n}({\overline {B}})|\leq \epsilon .} On the other hand, any B ∈ ∈ B {\displaystyle B\in {\mathcal {B}}} with m ( B ) ≤ ≤ δ δ {\displaystyle m(B)\leq \delta } can be represented as B = B 1 ∖ ∖ B 2 {\displaystyle B=B_{1}\setminus B_{2}} with d ( B 1 , B 0 ) ≤ ≤ δ δ {\displaystyle d(B_{1},B_{0})\leq \delta } and d ( B 2 , B 0 ) ≤ ≤ δ δ {\displaystyle d(B_{2},B_{0})\leq \delta } .
