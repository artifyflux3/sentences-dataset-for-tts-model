A web browser ([Safari](https://en.wikipedia.org/wiki/Safari_(web_browser) "Safari (web browser)")) displaying a [web page](https://en.wikipedia.org/wiki/Web_page "Web page")

A **web browser**, often shortened to **browser**, is an [application](https://en.wikipedia.org/wiki/Application_software "Application software") for accessing [websites](https://en.wikipedia.org/wiki/Website "Website"). When a [user](https://en.wikipedia.org/wiki/User_(computing) "User (computing)") requests a [web page](https://en.wikipedia.org/wiki/Web_page "Web page") from a particular website, the browser retrieves its [files](https://en.wikipedia.org/wiki/Computer_file "Computer file") from a [web server](https://en.wikipedia.org/wiki/Web_server "Web server") and then displays the page on the user's screen. Browsers can also display content stored locally on the user's device.

Browsers are used on a range of devices, including [desktops](https://en.wikipedia.org/wiki/Desktop_computer "Desktop computer"), [laptops](https://en.wikipedia.org/wiki/Laptop "Laptop"), [tablets](https://en.wikipedia.org/wiki/Tablet_computer "Tablet computer"), [smartphones](https://en.wikipedia.org/wiki/Smartphone "Smartphone"), [smartwatches](https://en.wikipedia.org/wiki/Smartwatch "Smartwatch") and [consoles](https://en.wikipedia.org/wiki/Video_game_console "Video game console"). As of 2024, the most used browsers worldwide are [Google Chrome](https://en.wikipedia.org/wiki/Google_Chrome "Google Chrome") (~66% [market share](https://en.wikipedia.org/wiki/Market_share "Market share")), [Safari](https://en.wikipedia.org/wiki/Safari_(web_browser) "Safari (web browser)") (~16%), [Edge](https://en.wikipedia.org/wiki/Microsoft_Edge "Microsoft Edge") (~6%), [Firefox](https://en.wikipedia.org/wiki/Firefox "Firefox") (~3%), [Samsung Internet](https://en.wikipedia.org/wiki/Samsung_Internet "Samsung Internet") (~2%), and [Opera](https://en.wikipedia.org/wiki/Opera_(web_browser) "Opera (web browser)") (~2%).[[1]](https://en.wikipedia.org/wiki/Web_browser#cite_note-statcounter-1)[[2]](https://en.wikipedia.org/wiki/Web_browser#cite_note-cloudflare-2) As of 2023, an estimated 5.4 billion people had used a browser.[[3]](https://en.wikipedia.org/wiki/Web_browser#cite_note-3)

Function
--------

Navigating to [English Wikipedia](https://en.wikipedia.org/wiki/English_Wikipedia "English Wikipedia") using a web browser ([Firefox](https://en.wikipedia.org/wiki/Firefox "Firefox"))

The purpose of a web browser is to fetch content and display it on the user's device.[[4]](https://en.wikipedia.org/wiki/Web_browser#cite_note-4) This process begins when the user inputs a [Uniform Resource Locator](https://en.wikipedia.org/wiki/Uniform_Resource_Locator "Uniform Resource Locator") (URL), such as _`https://en.wikipedia.org/`_, into the browser's [address bar](https://en.wikipedia.org/wiki/Address_bar "Address bar"). Virtually all URLs on [the Web](https://en.wikipedia.org/wiki/World_Wide_Web "World Wide Web") start with either _`http:`_ or _`https:`_ which means they are retrieved with the [Hypertext Transfer Protocol](https://en.wikipedia.org/wiki/HTTP "HTTP") (HTTP). For [secure mode](https://en.wikipedia.org/wiki/HTTPS "HTTPS") (HTTPS), the connection between the browser and [web server](https://en.wikipedia.org/wiki/Web_server "Web server") is [encrypted](https://en.wikipedia.org/wiki/Encryption "Encryption"), providing a [secure](https://en.wikipedia.org/wiki/Communications_security "Communications security") and [private](https://en.wikipedia.org/wiki/Information_privacy "Information privacy") data transfer.[[5]](https://en.wikipedia.org/wiki/Web_browser#cite_note-5) For this reason, a web browser is often referred to as an HTTP client[[6]](https://en.wikipedia.org/wiki/Web_browser#cite_note-6)[[7]](https://en.wikipedia.org/wiki/Web_browser#cite_note-7) or a [user agent](https://en.wikipedia.org/wiki/User_agent "User agent"). Requisite materials, including text, [style sheets](https://en.wikipedia.org/wiki/Style_sheet_(web_development) "Style sheet (web development)"), [images](https://en.wikipedia.org/wiki/Images "Images"), and other types of [multimedia](https://en.wikipedia.org/wiki/Website#Multimedia_and_interactive_content "Website"), are [downloaded](https://en.wikipedia.org/wiki/Downloaded "Downloaded") from the server. Once the materials have been downloaded, the web browser's [engine](https://en.wikipedia.org/wiki/Browser_engine "Browser engine") (also known as a layout engine or rendering engine) is responsible for converting those resources into an interactive visual representation of the page on the user's device.[[8]](https://en.wikipedia.org/wiki/Web_browser#cite_note-behindscene-8) Modern web browsers also contain separate [JavaScript engines](https://en.wikipedia.org/wiki/JavaScript_engine "JavaScript engine") which enable more complex interactive applications inside the browser.[[9]](https://en.wikipedia.org/wiki/Web_browser#cite_note-howBlinkWorks-9) A web browser that does not render a [graphical user interface](https://en.wikipedia.org/wiki/Graphical_user_interface "Graphical user interface") is known as a [headless browser](https://en.wikipedia.org/wiki/Headless_browser "Headless browser").

Web pages usually contain [hyperlinks](https://en.wikipedia.org/wiki/Hyperlink "Hyperlink") to other pages and resources. Each link contains a URL, and when it is [clicked](https://en.wikipedia.org/wiki/Point_and_click "Point and click") or [tapped](https://en.wikipedia.org/wiki/Touchscreen "Touchscreen"), the browser navigates to the new resource. Most browsers use an internal [cache](https://en.wikipedia.org/wiki/Web_cache "Web cache") of web page resources to improve loading times for subsequent visits to the same page. The cache can store many items, such as large images, so they do not need to be downloaded from the server again. Cached items are usually only stored for as long as the web server stipulates in its HTTP response messages.[[10]](https://en.wikipedia.org/wiki/Web_browser#cite_note-10)[[11]](https://en.wikipedia.org/wiki/Web_browser#cite_note-11)

A web browser is not the same thing as a [search engine](https://en.wikipedia.org/wiki/Search_engine "Search engine"), though the two are often confused.[[12]](https://en.wikipedia.org/wiki/Web_browser#cite_note-12)[[13]](https://en.wikipedia.org/wiki/Web_browser#cite_note-13) A search engine is a website that provides [links](https://en.wikipedia.org/wiki/Hyperlink "Hyperlink") to other websites and allows users to search for specific resources using a textual [query](https://en.wikipedia.org/wiki/Web_query "Web query"). However, web browsers are often used to access search engines, and most modern browsers allow users to access a default search engine directly by typing a query into the address bar.[[14]](https://en.wikipedia.org/wiki/Web_browser#cite_note-14)

History
-------

The first web browser, called [WorldWideWeb](https://en.wikipedia.org/wiki/WorldWideWeb "WorldWideWeb"), was created in 1990 by Sir [Tim Berners-Lee](https://en.wikipedia.org/wiki/Tim_Berners-Lee "Tim Berners-Lee").[[15]](https://en.wikipedia.org/wiki/Web_browser#cite_note-15)[[16]](https://en.wikipedia.org/wiki/Web_browser#cite_note-16) He then recruited [Nicola Pellow](https://en.wikipedia.org/wiki/Nicola_Pellow "Nicola Pellow") to write the [Line Mode Browser](https://en.wikipedia.org/wiki/Line_Mode_Browser "Line Mode Browser"), which displayed web pages on [dumb terminals](https://en.wikipedia.org/wiki/Dumb_terminals "Dumb terminals").[[17]](https://en.wikipedia.org/wiki/Web_browser#cite_note-:5-17) The [Mosaic](https://en.wikipedia.org/wiki/Mosaic_(web_browser) "Mosaic (web browser)") web browser was released in April 1993, and was later credited as the first web browser to find mainstream popularity.[[18]](https://en.wikipedia.org/wiki/Web_browser#cite_note-18)[[19]](https://en.wikipedia.org/wiki/Web_browser#cite_note-bloomberg-19) Its innovative [graphical user interface](https://en.wikipedia.org/wiki/Graphical_user_interface "Graphical user interface") made the [World Wide Web](https://en.wikipedia.org/wiki/World_Wide_Web "World Wide Web") easy to navigate and thus more accessible to the average person. This, in turn, sparked the Internet boom of the 1990s, when the Web grew at a very rapid rate.[[19]](https://en.wikipedia.org/wiki/Web_browser#cite_note-bloomberg-19) The lead developers of Mosaic then founded the [Netscape](https://en.wikipedia.org/wiki/Netscape "Netscape") corporation, which released the Mosaic-influenced [Netscape Navigator](https://en.wikipedia.org/wiki/Netscape_Navigator "Netscape Navigator") in 1994. Navigator quickly became the [most popular](https://en.wikipedia.org/wiki/Usage_share_of_web_browsers#Older_reports_(pre-2000) "Usage share of web browsers") browser.[[20]](https://en.wikipedia.org/wiki/Web_browser#cite_note-20)

[Microsoft](https://en.wikipedia.org/wiki/Microsoft "Microsoft") debuted [Internet Explorer](https://en.wikipedia.org/wiki/Internet_Explorer "Internet Explorer") in 1995, leading to a [browser war](https://en.wikipedia.org/wiki/Browser_war "Browser war") with Netscape. Within a few years, Microsoft gained a dominant position in the browser market for two reasons: it bundled Internet Explorer with its popular [Windows](https://en.wikipedia.org/wiki/Microsoft_Windows "Microsoft Windows")[operating system](https://en.wikipedia.org/wiki/Operating_system "Operating system") and did so as [freeware](https://en.wikipedia.org/wiki/Freeware "Freeware") with no restrictions on usage. The market share of Internet Explorer peaked at over 95% in the early 2000s.[[21]](https://en.wikipedia.org/wiki/Web_browser#cite_note-searchenginejournal.com-21) In 1998, Netscape launched what would become the [Mozilla Foundation](https://en.wikipedia.org/wiki/Mozilla_Foundation "Mozilla Foundation") to create a new browser using the [open-source software](https://en.wikipedia.org/wiki/Open-source_software "Open-source software") model. This work evolved into the [Firefox](https://en.wikipedia.org/wiki/Firefox "Firefox") browser, first released by Mozilla in 2004. Firefox's market share peaked at 32% in 2010.[[22]](https://en.wikipedia.org/wiki/Web_browser#cite_note-22)[Apple](https://en.wikipedia.org/wiki/Apple_Inc. "Apple Inc.") released its [Safari](https://en.wikipedia.org/wiki/Safari_(web_browser) "Safari (web browser)") browser in 2003; it remains the dominant browser on Apple devices, though it did not become popular elsewhere.[[23]](https://en.wikipedia.org/wiki/Web_browser#cite_note-browsershare-23)

[Google](https://en.wikipedia.org/wiki/Google "Google") debuted its [Chrome](https://en.wikipedia.org/wiki/Google_Chrome "Google Chrome") browser in 2008, which steadily took market share from Internet Explorer and became the most popular browser in 2012.[[24]](https://en.wikipedia.org/wiki/Web_browser#cite_note-24)[[25]](https://en.wikipedia.org/wiki/Web_browser#cite_note-25) Chrome has [remained dominant](https://en.wikipedia.org/wiki/Usage_share_of_web_browsers "Usage share of web browsers") ever since.[[1]](https://en.wikipedia.org/wiki/Web_browser#cite_note-statcounter-1) In 2015, Microsoft replaced Internet Explorer with [Edge [Legacy]](https://en.wikipedia.org/wiki/Microsoft_Edge_Legacy "Microsoft Edge Legacy") for the [Windows 10](https://en.wikipedia.org/wiki/Windows_10 "Windows 10") release.[[26]](https://en.wikipedia.org/wiki/Web_browser#cite_note-26) In 2020, this legacy version was replaced by a new [Chromium](https://en.wikipedia.org/wiki/Chromium_(web_browser) "Chromium (web browser)")-based version of [Edge](https://en.wikipedia.org/wiki/Microsoft_Edge "Microsoft Edge").

Since the early 2000s, browsers have greatly expanded their [HTML](https://en.wikipedia.org/wiki/HTML "HTML"), [CSS](https://en.wikipedia.org/wiki/CSS "CSS"), [JavaScript](https://en.wikipedia.org/wiki/JavaScript "JavaScript"), and [multimedia](https://en.wikipedia.org/wiki/Multimedia "Multimedia") capabilities. One reason has been to enable more sophisticated websites, such as [web apps](https://en.wikipedia.org/wiki/Web_application "Web application"). Another factor is the significant increase of [broadband](https://en.wikipedia.org/wiki/Broadband "Broadband") connectivity in [many parts](https://en.wikipedia.org/wiki/List_of_sovereign_states_by_number_of_broadband_Internet_subscriptions "List of sovereign states by number of broadband Internet subscriptions") of the world, enabling people to access data-intensive content, such as [streaming](https://en.wikipedia.org/wiki/Streaming_media "Streaming media")[HD video](https://en.wikipedia.org/wiki/High-definition_video "High-definition video") on [YouTube](https://en.wikipedia.org/wiki/YouTube "YouTube"), that was not possible during the era of [dial-up modems](https://en.wikipedia.org/wiki/Dial-up_Internet_access "Dial-up Internet access").[[27]](https://en.wikipedia.org/wiki/Web_browser#cite_note-27)

Starting in the mid-2020s, browsers with integrated [artificial intelligence](https://en.wikipedia.org/wiki/Artificial_intelligence "Artificial intelligence") (AI) capabilities, known as [AI browsers](https://en.wikipedia.org/wiki/AI_browser "AI browser"), have become increasingly common. This includes both new entrants to the browser market and established browsers that added AI features, such as Chrome with the [Gemini](https://en.wikipedia.org/wiki/Gemini_(chatbot) "Gemini (chatbot)")[chatbot](https://en.wikipedia.org/wiki/Chatbot "Chatbot") and Edge with the [Copilot](https://en.wikipedia.org/wiki/Microsoft_Copilot "Microsoft Copilot") chatbot.[[28]](https://en.wikipedia.org/wiki/Web_browser#cite_note-28)[[29]](https://en.wikipedia.org/wiki/Web_browser#cite_note-29)

Features
--------

The most popular browsers share many [features](https://en.wikipedia.org/wiki/Software_feature "Software feature") in common. They automatically log users' [browsing history](https://en.wikipedia.org/wiki/Web_browsing_history "Web browsing history"), unless the users turn off their browsing history or use the non-logging [private mode](https://en.wikipedia.org/wiki/Private_browsing "Private browsing"). They also allow users to set [bookmarks](https://en.wikipedia.org/wiki/Bookmark_(digital) "Bookmark (digital)"), customize the browser with [extensions](https://en.wikipedia.org/wiki/Browser_extension "Browser extension"), and [manage their downloads](https://en.wikipedia.org/wiki/Download_manager "Download manager")[[30]](https://en.wikipedia.org/wiki/Web_browser#cite_note-30) and [passwords](https://en.wikipedia.org/wiki/Password "Password").[[31]](https://en.wikipedia.org/wiki/Web_browser#cite_note-31) Some provide a sync service[[32]](https://en.wikipedia.org/wiki/Web_browser#cite_note-32) and [web accessibility](https://en.wikipedia.org/wiki/Web_accessibility "Web accessibility") features.[[33]](https://en.wikipedia.org/wiki/Web_browser#cite_note-33)

[![Image 3](https://upload.wikimedia.org/wikipedia/commons/thumb/1/13/Chromium_%28web_browser%29.png/250px-Chromium_%28web_browser%29.png)](https://en.wikipedia.org/wiki/File:Chromium_(web_browser).png)

Traditional browser arrangement has [user interface](https://en.wikipedia.org/wiki/User_interface "User interface") features above page content.

Common [user interface](https://en.wikipedia.org/wiki/User_interface "User interface") (UI) features:

*   Allowing the user to have multiple [pages](https://en.wikipedia.org/wiki/Web_page "Web page") open at the same time, either in different browser windows or in different [tabs](https://en.wikipedia.org/wiki/Tab_(interface) "Tab (interface)") of the same window.
*   _Back_ and _forward_ buttons to go back to the previous page visited or forward to the next one.
*   A _refresh_ or _reload_ and a _stop_ button to reload and cancel loading the current page. (In most browsers, the stop button is merged with the reload button.)
*   A _home_ button to return to the [start page](https://en.wikipedia.org/wiki/Home_page "Home page").
*   An address bar to input the [URL](https://en.wikipedia.org/wiki/URL "URL") of a page and display it, and a search bar to input [queries](https://en.wikipedia.org/wiki/Web_query "Web query") into a [search engine](https://en.wikipedia.org/wiki/Search_engine "Search engine"). (In most browsers, the search bar is merged with the address bar.)

While [mobile browsers](https://en.wikipedia.org/wiki/Mobile_browser "Mobile browser") have similar UI features as [desktop](https://en.wikipedia.org/wiki/Desktop_computer "Desktop computer") versions, the limitations of the often-smaller [touch screens](https://en.wikipedia.org/wiki/Touchscreen "Touchscreen") require mobile UIs to be simpler.[[34]](https://en.wikipedia.org/wiki/Web_browser#cite_note-34) The difference is significant for users accustomed to [keyboard shortcuts](https://en.wikipedia.org/wiki/Keyboard_shortcut "Keyboard shortcut").[[35]](https://en.wikipedia.org/wiki/Web_browser#cite_note-35)[Responsive web design](https://en.wikipedia.org/wiki/Responsive_web_design "Responsive web design") is used to create websites that offer a consistent experience across the desktop and mobile versions of the website and across varying screen sizes. The most popular desktop browsers also have sophisticated [web development tools](https://en.wikipedia.org/wiki/Web_development_tools "Web development tools").[[36]](https://en.wikipedia.org/wiki/Web_browser#cite_note-36)

Access to some web content — particularly [streaming services](https://en.wikipedia.org/wiki/Streaming_media "Streaming media") like Netflix, Disney+, and Spotify — is restricted by [Digital Rights Management](https://en.wikipedia.org/wiki/Digital_Rights_Management "Digital Rights Management") (DRM) software. A web browser is able to access DRM-restricted content through the use of a [Content Decryption Module](https://en.wikipedia.org/wiki/Content_Decryption_Module "Content Decryption Module") (CDM) such as [Widevine](https://en.wikipedia.org/wiki/Widevine "Widevine"). As of 2020, the CDMs used by dominant web browsers require browser providers to pay costly license fees, making it unfeasible for most independent open-source browsers to offer access to DRM-restricted content.[[37]](https://en.wikipedia.org/wiki/Web_browser#cite_note-37)

--------------

Security
--------

Web browsers are popular targets for [hackers](https://en.wikipedia.org/wiki/Hacker "Hacker"), who exploit [security holes](https://en.wikipedia.org/wiki/Vulnerability_(computing) "Vulnerability (computing)") to steal information, destroy [files](https://en.wikipedia.org/wiki/Computer_file "Computer file"), and partake in other [malicious](https://en.wikipedia.org/wiki/Malware "Malware") activities. Browser vendors regularly patch these security holes, so users are strongly encouraged to keep their browser software updated. Other protection measures are [antivirus software](https://en.wikipedia.org/wiki/Antivirus_software "Antivirus software") and being aware of [scams](https://en.wikipedia.org/wiki/Internet_fraud "Internet fraud").[[41]](https://en.wikipedia.org/wiki/Web_browser#cite_note-41)

### Privacy

During the course of browsing, [cookies](https://en.wikipedia.org/wiki/HTTP_cookie "HTTP cookie") received from various [websites](https://en.wikipedia.org/wiki/Website "Website") are stored by the browser. Some of them contain login credentials or site preferences.[[42]](https://en.wikipedia.org/wiki/Web_browser#cite_note-tom's_guide-42) However, others are used for tracking user behavior over long periods of time, so browsers typically provide a section in the menu for deleting cookies.[[42]](https://en.wikipedia.org/wiki/Web_browser#cite_note-tom's_guide-42) Some browsers have more proactive protection against cookies and trackers that limit their functionality and ability to track user behaviour.[[43]](https://en.wikipedia.org/wiki/Web_browser#cite_note-43) Finer-grained management of cookies usually requires a [browser extension](https://en.wikipedia.org/wiki/Browser_extension "Browser extension").[[44]](https://en.wikipedia.org/wiki/Web_browser#cite_note-44)[Telemetry](https://en.wikipedia.org/wiki/Telemetry_(software) "Telemetry (software)") data is [collected](https://en.wikipedia.org/wiki/Data_collection "Data collection") by most popular web browsers, which can usually be opted out of by the user.[[45]](https://en.wikipedia.org/wiki/Web_browser#cite_note-45)
