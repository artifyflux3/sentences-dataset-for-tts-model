(Redirected from [Column matrix](https://en.wikipedia.org/w/index.php?title=Column_matrix&redirect=no "Column matrix"))

In [linear algebra](https://en.wikipedia.org/wiki/Linear_algebra "Linear algebra"), a **column vector** with ⁠![Image 1: {\displaystyle m}](https://wikimedia.org/api/rest_v1/media/math/render/svg/0a07d98bb302f3856cbabc47b2b9016692e3f7bc)⁠ elements is an ![Image 2: {\displaystyle m\times 1}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7f7ea91d81567531f5ef6d3b669be211ff953e6a)[matrix](https://en.wikipedia.org/wiki/Matrix_(mathematics) "Matrix (mathematics)")[[1]](https://en.wikipedia.org/wiki/Column_matrix#cite_note-Artin-1) consisting of a single column of ⁠![Image 3: {\displaystyle m}](https://wikimedia.org/api/rest_v1/media/math/render/svg/0a07d98bb302f3856cbabc47b2b9016692e3f7bc)⁠ entries. Similarly, a **row vector** is a ![Image 4: {\displaystyle 1\times n}](https://wikimedia.org/api/rest_v1/media/math/render/svg/0bce5f6a6d0d32834484048c16f3b39f9c23d076) matrix, consisting of a single row of ⁠![Image 5: {\displaystyle n}](https://wikimedia.org/api/rest_v1/media/math/render/svg/a601995d55609f2d9f5e233e36fbe9ea26011b3b)⁠ entries. For example, ⁠![Image 6: {\displaystyle {\boldsymbol {x}}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/606b7680d510560a505937143775ea80fa958051)⁠ is a column vector and ⁠![Image 7: {\displaystyle {\boldsymbol {a}}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/df0c47bb21acb94ccb65db3da51e075abc68a898)⁠ is a row matrix:

![Image 8: {\displaystyle {\boldsymbol {x}}={\begin{bmatrix}x_{1}\\x_{2}\\\vdots \\x_{m}\end{bmatrix}},\quad {\boldsymbol {a}}={\begin{bmatrix}a_{1}&a_{2}&\dots &a_{n}\end{bmatrix}}.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/6f9d2e23feac1b08393b6daaa57abbaf597b8e7e) (Throughout this article, boldface is used for both row and column vectors.)

The [transpose](https://en.wikipedia.org/wiki/Transpose "Transpose") (indicated by T) of any row vector is a column vector, and the transpose of any column vector is a row vector: ![Image 9: {\displaystyle {\begin{bmatrix}x_{1}\;x_{2}\;\dots \;x_{m}\end{bmatrix}}^{\rm {T}}={\begin{bmatrix}x_{1}\\x_{2}\\\vdots \\x_{m}\end{bmatrix}},\quad {\begin{bmatrix}x_{1}\\x_{2}\\\vdots \\x_{m}\end{bmatrix}}^{\rm {T}}={\begin{bmatrix}x_{1}\;x_{2}\;\dots \;x_{m}\end{bmatrix}}.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7315b303d3843ec49455c3196f09aac958c1a6c6) Taking the transpose twice returns the original (row or column) vector: ⁠![Image 10: {\displaystyle \textstyle {\bigl (}{\boldsymbol {x}}^{\rm {T}}{\bigr )}{\vphantom {)}}^{\rm {T}}={\boldsymbol {x}}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/d3ab9d148c56bdd45f73ba1b71a0478f0d8c2433)⁠.

The set of all row vectors with n entries in a given [field](https://en.wikipedia.org/wiki/Field_(mathematics) "Field (mathematics)") (such as the [real numbers](https://en.wikipedia.org/wiki/Real_numbers "Real numbers")) forms an n-dimensional [vector space](https://en.wikipedia.org/wiki/Vector_space "Vector space"); similarly, the set of all column vectors with m entries forms an m-dimensional vector space.

The space of row vectors with n entries can be regarded as the [dual space](https://en.wikipedia.org/wiki/Dual_space "Dual space") of the space of column vectors with n entries, since any linear functional on the space of column vectors can be represented as the left-multiplication of a unique row vector.

To simplify writing column vectors in-line with other text, sometimes they are written as row vectors with the transpose operation applied to them.

![Image 11: {\displaystyle {\boldsymbol {x}}={\begin{bmatrix}x_{1}\;x_{2}\;\dots \;x_{m}\end{bmatrix}}^{\rm {T}}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/ebb1f2cf41828a7d6a73918b4696bfa11b569c8b)

or

![Image 12: {\displaystyle {\boldsymbol {x}}={\begin{bmatrix}x_{1},x_{2},\dots ,x_{m}\end{bmatrix}}^{\rm {T}}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/b7fb2a1f3a523f75e432b8ab6dd8fba6fea7eca2)

Some authors also use the convention of writing both column vectors and row vectors as rows, but separating row vector elements with [commas](https://en.wikipedia.org/wiki/Comma "Comma") and column vector elements with [semicolons](https://en.wikipedia.org/wiki/Semicolon "Semicolon") (see alternative notation 2 in the table below).[_[citation needed](https://en.wikipedia.org/wiki/Wikipedia:Citation\_needed "Wikipedia:Citation needed")_]

| **Alternative notation 1** (commas, transpose signs) | ![Image 15: {\displaystyle {\begin{bmatrix}x_{1},x_{2},\dots ,x_{m}\end{bmatrix}}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/bb84a3f9002a580531fc893fd99025d166839e72) | ![Image 16: {\displaystyle {\begin{bmatrix}x_{1},x_{2},\dots ,x_{m}\end{bmatrix}}^{\rm {T}}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/4de211fc97fc662f55f10e4f32bc54d60344ae6e) |
| **Alternative notation 2** (commas and semicolons, no transpose signs) | ![Image 17: {\displaystyle {\begin{bmatrix}x_{1},x_{2},\dots ,x_{m}\end{bmatrix}}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/bb84a3f9002a580531fc893fd99025d166839e72) | ![Image 18: {\displaystyle {\begin{bmatrix}x_{1};x_{2};\dots ;x_{m}\end{bmatrix}}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/76cb3947659e67ac10298232bfb47a63adc5c39f) |

[Matrix multiplication](https://en.wikipedia.org/wiki/Matrix_multiplication "Matrix multiplication") involves the action of multiplying each row vector of one matrix by each column vector of another matrix.

The [dot product](https://en.wikipedia.org/wiki/Dot_product "Dot product") of two column vectors **a**, **b**, considered as elements of a coordinate space, is equal to the matrix product of the transpose of **a** with **b**,

![Image 19: {\displaystyle \mathbf {a} \cdot \mathbf {b} =\mathbf {a} ^{\rm {T}}\mathbf {b} ={\begin{bmatrix}a_{1}&\cdots &a_{n}\end{bmatrix}}{\begin{bmatrix}b_{1}\\\vdots \\b_{n}\end{bmatrix}}=a_{1}b_{1}+\cdots +a_{n}b_{n}\,,}](https://wikimedia.org/api/rest_v1/media/math/render/svg/ce09db40400275e12a556ea43bf3dc7636026508)

By the symmetry of the dot product, the [dot product](https://en.wikipedia.org/wiki/Dot_product "Dot product") of two column vectors **a**, **b** is also equal to the matrix product of the transpose of **b** with **a**,

![Image 20: {\displaystyle \mathbf {b} \cdot \mathbf {a} =\mathbf {b} ^{\rm {T}}\mathbf {a} ={\begin{bmatrix}b_{1}&\cdots &b_{n}\end{bmatrix}}{\begin{bmatrix}a_{1}\\\vdots \\a_{n}\end{bmatrix}}=a_{1}b_{1}+\cdots +a_{n}b_{n}\,.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/941b1485bc84bd36b6dbcfbc43d0b6a5a08ac534)

The matrix product of a column and a row vector gives the [outer product](https://en.wikipedia.org/wiki/Outer_product "Outer product") of two vectors **a**, **b**, an example of the more general [tensor product](https://en.wikipedia.org/wiki/Tensor_product "Tensor product"). The matrix product of the column vector representation of **a** and the row vector representation of **b** gives the components of their dyadic product,

![Image 21: {\displaystyle \mathbf {a} \otimes \mathbf {b} =\mathbf {a} \mathbf {b} ^{\rm {T}}={\begin{bmatrix}a_{1}\\a_{2}\\a_{3}\end{bmatrix}}{\begin{bmatrix}b_{1}&b_{2}&b_{3}\end{bmatrix}}={\begin{bmatrix}a_{1}b_{1}&a_{1}b_{2}&a_{1}b_{3}\\a_{2}b_{1}&a_{2}b_{2}&a_{2}b_{3}\\a_{3}b_{1}&a_{3}b_{2}&a_{3}b_{3}\\\end{bmatrix}}\,,}](https://wikimedia.org/api/rest_v1/media/math/render/svg/f9f4cea35eb4de004a31e108345d6263c115786d)

which is the [transpose](https://en.wikipedia.org/wiki/Transpose "Transpose") of the matrix product of the column vector representation of **b** and the row vector representation of **a**,

![Image 22: {\displaystyle \mathbf {b} \otimes \mathbf {a} =\mathbf {b} \mathbf {a} ^{\rm {T}}={\begin{bmatrix}b_{1}\\b_{2}\\b_{3}\end{bmatrix}}{\begin{bmatrix}a_{1}&a_{2}&a_{3}\end{bmatrix}}={\begin{bmatrix}b_{1}a_{1}&b_{1}a_{2}&b_{1}a_{3}\\b_{2}a_{1}&b_{2}a_{2}&b_{2}a_{3}\\b_{3}a_{1}&b_{3}a_{2}&b_{3}a_{3}\\\end{bmatrix}}\,.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/e6368212c8b0b4cc2a094347155d2ac3d3c8003f)

Matrix transformations
----------------------

[[edit](https://en.wikipedia.org/w/index.php?title=Row_and_column_vectors&action=edit&section=3 "Edit section: Matrix transformations")]

An _n_ × _n_ matrix M can represent a [linear map](https://en.wikipedia.org/wiki/Linear_map "Linear map") and act on row and column vectors as the linear map's [transformation matrix](https://en.wikipedia.org/wiki/Transformation_matrix "Transformation matrix"). For a row vector **v**, the product **v**_M_ is another row vector **p**:

![Image 23: {\displaystyle \mathbf {v} M=\mathbf {p} \,.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/22cda9e6efd23e4eb3c2cb10f58cda2a312883d2)

Another _n_ × _n_ matrix Q can act on **p**,

![Image 24: {\displaystyle \mathbf {p} Q=\mathbf {t} \,.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/e26d94f8ba3a07d1e4857505b14d65f71c7552f9)

Then one can write **t** = **p**_Q_ = **v**_MQ_, so the [matrix product](https://en.wikipedia.org/wiki/Matrix_product "Matrix product") transformation MQ maps **v** directly to **t**. Continuing with row vectors, matrix transformations further reconfiguring n-space can be applied to the right of previous outputs.

When a column vector is transformed to another column vector under an _n_ × _n_ matrix action, the operation occurs to the left,

![Image 25: {\displaystyle \mathbf {p} ^{\mathrm {T} }=M\mathbf {v} ^{\mathrm {T} }\,,\quad \mathbf {t} ^{\mathrm {T} }=Q\mathbf {p} ^{\mathrm {T} },}](https://wikimedia.org/api/rest_v1/media/math/render/svg/5b1df36d3127cc6ab392464d91b4efff633ecc7a)

leading to the algebraic expression _QM_**v**T for the composed output from **v**T input. The matrix transformations mount up to the left in this use of a column vector for input to matrix transformation.
