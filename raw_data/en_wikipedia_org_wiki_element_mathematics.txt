From Wikipedia, the free encyclopedia

Any one of the distinct objects that make up a set in set theory

For elements in category theory, see [Element (category theory)](https://en.wikipedia.org/wiki/Element_(category_theory) "Element (category theory)").

For the components of the Periodic Table, see [Chemical element](https://en.wikipedia.org/wiki/Chemical_element "Chemical element").

In [mathematics](https://en.wikipedia.org/wiki/Mathematics "Mathematics"), an **element** (or **member**) of a [set](https://en.wikipedia.org/wiki/Set_(mathematics) "Set (mathematics)") is any one of the [distinct](https://en.wikipedia.org/wiki/Equality_(mathematics) "Equality (mathematics)")[objects](https://en.wikipedia.org/wiki/Mathematical_object "Mathematical object") that belong to that set. For example, given a set called A containing the first four positive [integers](https://en.wikipedia.org/wiki/Integer "Integer") (A={1,2,3,4}{\displaystyle A=\{1,2,3,4\}}![Image 17: {\displaystyle A=\{1,2,3,4\}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/803f93dc0e1132de0c52ef8ef17b6c7224b05917)), one could say that "3 is an element of A", expressed notationally as 3∈A{\displaystyle 3\in A}![Image 18: {\displaystyle 3\in A}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7a6eac52d6bea5535685396ff8a429be280ff2a2).

Sets
----

[[edit](https://en.wikipedia.org/w/index.php?title=Element_(mathematics)&action=edit&section=1 "Edit section: Sets")]

Writing A={1,2,3,4}{\displaystyle A=\{1,2,3,4\}}![Image 19: {\displaystyle A=\{1,2,3,4\}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/803f93dc0e1132de0c52ef8ef17b6c7224b05917) means that the elements of the set A are the numbers 1, 2, 3 and 4. Sets of elements of A, for example {1,2}{\displaystyle \{1,2\}}![Image 20: {\displaystyle \{1,2\}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/111aef9df2c1a71ed7820a5426ef3e8c7b781564), are [subsets](https://en.wikipedia.org/wiki/Subset "Subset") of A.

Sets can themselves be elements. For example, consider the set B={1,2,{3,4}}{\displaystyle B=\{1,2,\{3,4\}\}}![Image 21: {\displaystyle B=\{1,2,\{3,4\}\}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/5490bab678438b0481941f2f7808155e6c2a0383). The elements of B are _not_ 1, 2, 3, and 4. Rather, there are only three elements of B, namely the numbers 1 and 2, and the set {3,4}{\displaystyle \{3,4\}}![Image 22: {\displaystyle \{3,4\}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/4ce803a141c2412f297a65b77a43d94ce3d17b27).

The elements of a set can be anything. For example the elements of the set C={r e d,12,B}{\displaystyle C=\{\mathrm {\color {Red}red} ,\mathrm {12} ,B\}}![Image 23: {\displaystyle C=\{\mathrm {\color {Red}red} ,\mathrm {12} ,B\}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/6f1916509ee51fb3e2e91d629b7aa72bcff35754) are the color red, the number 12, and the set B.

In logic, a set can be defined in terms of the membership of its elements as (x∈y)↔∀x[P x=y]:x∈D y{\displaystyle (x\in y)\leftrightarrow \forall x[P_{x}=y]:x\in {\mathfrak {D}}y}![Image 24: {\displaystyle (x\in y)\leftrightarrow \forall x[P_{x}=y]:x\in {\mathfrak {D}}y}](https://wikimedia.org/api/rest_v1/media/math/render/svg/1876b2dc5aac64f7b8d67e5da8f789acc75d00b8). This basically means that there is a general [predication](https://en.wikipedia.org/wiki/Predicate_(logic) "Predicate (logic)") of x called membership that is equivalent to the statement ‘x is a member of y if and only if, for all objects x, the general predication of x is identical to y, where x is a member of the [domain](https://en.wikipedia.org/wiki/Domain_of_a_function "Domain of a function") of y.’ The expression x ∈ 𝔇y makes this definition well-defined by ensuring that x is a bound variable in its predication of membership in y.

In this case, the domain of Px, which is the set containing all dependent logical values x that satisfy the stated conditions for membership in y, is called the Universe (U) of y. The range of Px, which is the set of all possible dependent set variables y resulting from satisfaction of the conditions of membership for x, is the [power set](https://en.wikipedia.org/wiki/Power_set "Power set") of U such that the [binary relation](https://en.wikipedia.org/wiki/Binary_relation "Binary relation") of the membership of x in y is any subset of the [cartesian product](https://en.wikipedia.org/wiki/Cartesian_product "Cartesian product") U × 𝒫(U) (the Cartesian Product of set U with the Power Set of U).

Notation and terminology
------------------------

[[edit](https://en.wikipedia.org/w/index.php?title=Element_(mathematics)&action=edit&section=2 "Edit section: Notation and terminology")]

The [binary relation](https://en.wikipedia.org/wiki/Binary_relation "Binary relation") "is an element of", also called **set membership**, is denoted by the symbol"∈". Writing

x∈A{\displaystyle x\in A}![Image 25: {\displaystyle x\in A}](https://wikimedia.org/api/rest_v1/media/math/render/svg/27bcc9b2afb295d4234bc294860cd0c63bcad2ca)
means that "_x_ is an element of _A_".[[1]](https://en.wikipedia.org/wiki/Element_(mathematics)#cite_note-1) Equivalent expressions are "_x_ is a member of _A_", "_x_ belongs to _A_", "_x_ is in _A_" and "_x_ lies in _A_". The expressions "_A_ includes _x_" and "_A_ contains _x_" are also used to mean set membership, although some authors use them to mean instead "_x_ is a [subset](https://en.wikipedia.org/wiki/Subset "Subset") of _A_".[[2]](https://en.wikipedia.org/wiki/Element_(mathematics)#cite_note-schech-2) Logician [George Boolos](https://en.wikipedia.org/wiki/George_Boolos "George Boolos") strongly urged that "contains" be used for membership only, and "includes" for the subset relation only.[[3]](https://en.wikipedia.org/wiki/Element_(mathematics)#cite_note-boolos-3)

For the relation ∈ , the [converse relation](https://en.wikipedia.org/wiki/Converse_relation "Converse relation") ∈T may be written

A∋x{\displaystyle A\ni x}![Image 26: {\displaystyle A\ni x}](https://wikimedia.org/api/rest_v1/media/math/render/svg/cff688698c6af2962bbfc3959fe6526ec3e76d20)
meaning "_A_ contains or includes _x_".

The [negation](https://en.wikipedia.org/wiki/Negation "Negation") of set membership is denoted by the symbol"∉". Writing

x∉A{\displaystyle x\notin A}![Image 27: {\displaystyle x\notin A}](https://wikimedia.org/api/rest_v1/media/math/render/svg/8152431575305d6a6145adf9b279891a65923eba)
means that "_x_ is not an element of _A_".

The symbol ∈ was first used by [Giuseppe Peano](https://en.wikipedia.org/wiki/Giuseppe_Peano "Giuseppe Peano"), in his 1889 work _[Arithmetices principia, nova methodo exposita](https://en.wikipedia.org/wiki/Arithmetices\_principia,\_nova\_methodo\_exposita "Arithmetices principia, nova methodo exposita")_.[[4]](https://en.wikipedia.org/wiki/Element_(mathematics)#cite_note-ken-4) Here he wrote on page X:

> _Signum ∈ significat est. Ita a ∈ b legitur a est quoddam b; …_

which means

> The symbol ∈ means _is_. So _a_ ∈ _b_ is read as a _is a certain_ b; …

The symbol itself is a stylized lowercase Greek letter [epsilon](https://en.wikipedia.org/wiki/Epsilon "Epsilon") ("ϵ"), the first letter of the word [ἐστί](https://en.wiktionary.org/wiki/%E1%BC%90%CF%83%CF%84%CE%AF#Ancient_Greek "wikt:ἐστί"), which means "is".[[4]](https://en.wikipedia.org/wiki/Element_(mathematics)#cite_note-ken-4)

Examples
--------

[[edit](https://en.wikipedia.org/w/index.php?title=Element_(mathematics)&action=edit&section=3 "Edit section: Examples")]

Using the sets defined above, namely _A_ = {1, 2, 3, 4}, _B_ = {1, 2, {3, 4}} and _C_ = {red, green, blue}, the following statements are true:

*   2 ∈ _A_
*   5 ∉ _A_

*   {3, 4} ∈ _B_
*   3 ∉ _B_
*   4 ∉ _B_
*   yellow ∉ _C_

Cardinality of sets
-------------------

[[edit](https://en.wikipedia.org/w/index.php?title=Element_(mathematics)&action=edit&section=4 "Edit section: Cardinality of sets")]

Main article: [Cardinality](https://en.wikipedia.org/wiki/Cardinality "Cardinality")

The number of elements in a particular set is a property known as [cardinality](https://en.wikipedia.org/wiki/Cardinality "Cardinality"); informally, this is the size of a set.[[5]](https://en.wikipedia.org/wiki/Element_(mathematics)#cite_note-5) In the above examples, the cardinality of the set _A_ is 4, while the cardinality of set _B_ and set _C_ are both 3. An infinite set is a set with an infinite number of elements, while a [finite set](https://en.wikipedia.org/wiki/Finite_set "Finite set") is a set with a finite number of elements. The above examples are examples of finite sets. An example of an infinite set is the set of positive integers {1, 2, 3, 4, ...}.

Formal relation
---------------

[[edit](https://en.wikipedia.org/w/index.php?title=Element_(mathematics)&action=edit&section=5 "Edit section: Formal relation")]

As a [relation](https://en.wikipedia.org/wiki/Relation_(mathematics) "Relation (mathematics)"), set membership must have a domain and a range. Conventionally the domain is called the [universe](https://en.wikipedia.org/wiki/Universe_(mathematics) "Universe (mathematics)") denoted _U_. The range is the set of [subsets](https://en.wikipedia.org/wiki/Subset "Subset") of _U_ called the [power set](https://en.wikipedia.org/wiki/Power_set "Power set") of _U_ and denoted P(_U_). Thus the relation ∈{\displaystyle \in }![Image 28: {\displaystyle \in }](https://wikimedia.org/api/rest_v1/media/math/render/svg/6fe4d5b0a594c1da89b5e78e7dfbeed90bdcc32f) is a subset of _U_ × P(_U_). The converse relation ∋{\displaystyle \ni }![Image 29: {\displaystyle \ni }](https://wikimedia.org/api/rest_v1/media/math/render/svg/91c3deaee7d8f5b0a1dc4c3913acdf15044b8eb3) is a subset of P(_U_) × _U_.
