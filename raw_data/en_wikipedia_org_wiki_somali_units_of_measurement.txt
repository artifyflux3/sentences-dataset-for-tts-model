In 1950, Somalia adopted the metric system , which has been compulsory there since 1972.

[ 1 ] System before metric system [ edit ] A number of units were used.  These units were local and varied with materials and provinces.

Length [ edit ] Several units were used to measure length.  One top was equal to 3.92 m.

[ 1 ] [ 2 ] One cubito was equal to 1/7 top.

[ 1 ] [ 2 ] Mass [ edit ] A number of units were used to measure mass.  One rottolo (pound) was equal to 0.448 kg.

[ 1 ] [ 2 ] Some other units are given below: 1 okia (ounce) = 1 ⁄ 16 rottolo 1 frasla = 36 rottolo 1 gisla = 360 rottolo.

[ 1 ] [ 2 ] Area [ edit ] One darat was equal to 8000 m 2 .

[ 2 ] Capacity [ edit ] Two main systems, dry and liquid, were used.

Dry [ edit ] Several units were used to measure dry capacity.  One chela was equal to 1.359 L .

[ 1 ] [ 2 ] Some other units were given below: 1 tabla = 15 chela 1 gisla = 120 chela.

[ 1 ] [ 2 ] Liquid [ edit ] One caba was equal to 0.453 L .
