From Wikipedia, the free encyclopedia

Linear map or polynomial function of degree one

For the use of the term in calculus, see [Linear function (calculus)](https://en.wikipedia.org/wiki/Linear_function_(calculus) "Linear function (calculus)").

In [mathematics](https://en.wikipedia.org/wiki/Mathematics "Mathematics"), the term **linear function** refers to two distinct but related notions:[[1]](https://en.wikipedia.org/wiki/Linear_function#cite_note-1)

*   In [calculus](https://en.wikipedia.org/wiki/Calculus "Calculus") and related areas, a linear function is a [function](https://en.wikipedia.org/wiki/Function_(mathematics) "Function (mathematics)") whose [graph](https://en.wikipedia.org/wiki/Graph_of_a_function "Graph of a function") is a [straight line](https://en.wikipedia.org/wiki/Straight_line "Straight line"), that is, a [polynomial function](https://en.wikipedia.org/wiki/Polynomial_function "Polynomial function") of [degree](https://en.wikipedia.org/wiki/Polynomial_degree "Polynomial degree") zero or one.[[2]](https://en.wikipedia.org/wiki/Linear_function#cite_note-2) For distinguishing such a linear function from the other concept, the term _[affine function](https://en.wikipedia.org/wiki/Affine\_function "Affine function")_ is often used.[[3]](https://en.wikipedia.org/wiki/Linear_function#cite_note-3)
*   In [linear algebra](https://en.wikipedia.org/wiki/Linear_algebra "Linear algebra"), [mathematical analysis](https://en.wikipedia.org/wiki/Mathematical_analysis "Mathematical analysis"),[[4]](https://en.wikipedia.org/wiki/Linear_function#cite_note-4) and [functional analysis](https://en.wikipedia.org/wiki/Functional_analysis "Functional analysis"), a linear function is a [linear map](https://en.wikipedia.org/wiki/Linear_map "Linear map").[[5]](https://en.wikipedia.org/wiki/Linear_function#cite_note-5)

As a polynomial function
------------------------

[[edit](https://en.wikipedia.org/w/index.php?title=Linear_function&action=edit&section=1 "Edit section: As a polynomial function")]

Main article: [Linear function (calculus)](https://en.wikipedia.org/wiki/Linear_function_(calculus) "Linear function (calculus)")

[![Image 11](https://upload.wikimedia.org/wikipedia/commons/thumb/0/0e/Linear_Function_Graph.svg/250px-Linear_Function_Graph.svg.png)](https://en.wikipedia.org/wiki/File:Linear_Function_Graph.svg)

Graphs of two linear functions.

In calculus, [analytic geometry](https://en.wikipedia.org/wiki/Analytic_geometry "Analytic geometry") and related areas, a linear function is a polynomial of degree one or less, including the [zero polynomial](https://en.wikipedia.org/wiki/Zero_polynomial "Zero polynomial") (the latter not being considered to have degree zero).

When the function is of only one [variable](https://en.wikipedia.org/wiki/Variable_(mathematics) "Variable (mathematics)"), it is of the form

f(x)=a x+b,{\displaystyle f(x)=ax+b,}![Image 12: {\displaystyle f(x)=ax+b,}](https://wikimedia.org/api/rest_v1/media/math/render/svg/4bb78c557e6ea8b777bb975d3ef12ab17bb411cc)
where _a_ and _b_ are [constants](https://en.wikipedia.org/wiki/Constant_(mathematics) "Constant (mathematics)"), often [real numbers](https://en.wikipedia.org/wiki/Real_number "Real number"). The [graph](https://en.wikipedia.org/wiki/Graph_of_a_function "Graph of a function") of such a function of one variable is a nonvertical line. _a_ is frequently referred to as the slope of the line, and _b_ as the intercept.

If _a > 0_ then the [gradient](https://en.wikipedia.org/wiki/Slope "Slope") is positive and the graph slopes upwards.

If _a < 0_ then the [gradient](https://en.wikipedia.org/wiki/Slope "Slope") is negative and the graph slopes downwards.

For a function f(x 1,…,x k){\displaystyle f(x_{1},\ldots ,x_{k})}![Image 13: {\displaystyle f(x_{1},\ldots ,x_{k})}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7afedaaa7dfef56c4f5825bc313635b1c6755af3) of any finite number of variables, the general formula is

f(x 1,…,x k)=b+a 1 x 1+⋯+a k x k,{\displaystyle f(x_{1},\ldots ,x_{k})=b+a_{1}x_{1}+\cdots +a_{k}x_{k},}![Image 14: {\displaystyle f(x_{1},\ldots ,x_{k})=b+a_{1}x_{1}+\cdots +a_{k}x_{k},}](https://wikimedia.org/api/rest_v1/media/math/render/svg/eb5a25377a0e452b240c75d17695d347a33e0e28)
and the graph is a [hyperplane](https://en.wikipedia.org/wiki/Hyperplane "Hyperplane") of dimension _k_.

A [constant function](https://en.wikipedia.org/wiki/Constant_function "Constant function") is also considered linear in this context, as it is a polynomial of degree zero or is the zero polynomial. Its graph, when there is only one variable, is a horizontal line.

In this context, a function that is also a linear map (the other meaning) may be referred to as a [homogeneous](https://en.wikipedia.org/wiki/Homogeneous_function "Homogeneous function") linear function or a [linear form](https://en.wikipedia.org/wiki/Linear_form "Linear form"). In the context of linear algebra, the polynomial functions of degree 0 or 1 are the scalar-valued [affine maps](https://en.wikipedia.org/wiki/Affine_map "Affine map").

As a linear map
---------------

[[edit](https://en.wikipedia.org/w/index.php?title=Linear_function&action=edit&section=2 "Edit section: As a linear map")]

Main article: [Linear map](https://en.wikipedia.org/wiki/Linear_map "Linear map")

[![Image 15](https://upload.wikimedia.org/wikipedia/commons/thumb/f/f2/Integral_as_region_under_curve.svg/250px-Integral_as_region_under_curve.svg.png)](https://en.wikipedia.org/wiki/File:Integral_as_region_under_curve.svg)

The [integral](https://en.wikipedia.org/wiki/Integral "Integral") of a function is a linear map from the vector space of integrable functions to the real numbers.

In linear algebra, a linear function is a map _f_ between two [vector spaces](https://en.wikipedia.org/wiki/Vector_space "Vector space") such that

f(x+y)=f(x)+f(y){\displaystyle f(\mathbf {x} +\mathbf {y} )=f(\mathbf {x} )+f(\mathbf {y} )}![Image 16: {\displaystyle f(\mathbf {x} +\mathbf {y} )=f(\mathbf {x} )+f(\mathbf {y} )}](https://wikimedia.org/api/rest_v1/media/math/render/svg/20c1db57416b4644aacdfc90beb5481f08e39462)f(a x)=a f(x).{\displaystyle f(a\mathbf {x} )=af(\mathbf {x} ).}![Image 17: {\displaystyle f(a\mathbf {x} )=af(\mathbf {x} ).}](https://wikimedia.org/api/rest_v1/media/math/render/svg/79be6bd64f9093829d58fc9e67ee5529eb0a848d)
Here _a_ denotes a constant belonging to some [field](https://en.wikipedia.org/wiki/Field_(mathematics) "Field (mathematics)")_K_ of [scalars](https://en.wikipedia.org/wiki/Scalar_(mathematics) "Scalar (mathematics)") (for example, the [real numbers](https://en.wikipedia.org/wiki/Real_number "Real number")) and **x** and **y** are elements of a [vector space](https://en.wikipedia.org/wiki/Vector_space "Vector space"), which might be _K_ itself.

In other terms the linear function preserves [vector addition](https://en.wikipedia.org/wiki/Vector_addition "Vector addition") and [scalar multiplication](https://en.wikipedia.org/wiki/Scalar_multiplication "Scalar multiplication").

Some authors use "linear function" only for linear maps that take values in the scalar field;[[6]](https://en.wikipedia.org/wiki/Linear_function#cite_note-6) these are more commonly called [linear forms](https://en.wikipedia.org/wiki/Linear_form "Linear form").

The "linear functions" of calculus qualify as "linear maps" when (and only when) _f_(0, ..., 0) = 0, or, equivalently, when the constant b equals zero in the one-degree polynomial above. Geometrically, the graph of the function must pass through the origin.
