In [algebra](https://en.wikipedia.org/wiki/Algebra "Algebra"), an **absolute value**[[a]](https://en.wikipedia.org/wiki/Absolute_value_(algebra)#cite_note-1) is a [function](https://en.wikipedia.org/wiki/Function_(mathematics) "Function (mathematics)") that generalizes the usual [absolute value](https://en.wikipedia.org/wiki/Absolute_value "Absolute value").[[1]](https://en.wikipedia.org/wiki/Absolute_value_(algebra)#cite_note-2) More precisely, if _D_ is a [field](https://en.wikipedia.org/wiki/Field_(mathematics) "Field (mathematics)") or (more generally) an [integral domain](https://en.wikipedia.org/wiki/Integral_domain "Integral domain"), an _absolute value_ on _D_ is a function, commonly denoted ![Image 1: {\displaystyle |x|,}](https://wikimedia.org/api/rest_v1/media/math/render/svg/a005758d43adc00c141f077e4f0fb44738505506) from _D_ to the [real numbers](https://en.wikipedia.org/wiki/Real_number "Real number") satisfying:

•![Image 2: {\displaystyle \left|x\right|\geq 0}](https://wikimedia.org/api/rest_v1/media/math/render/svg/30932eacd49bc0ab2f421498e08366d1c302efaf)(non-negativity)
•![Image 3: {\displaystyle \left|x\right|=0}](https://wikimedia.org/api/rest_v1/media/math/render/svg/5e08dc46e0de2119631c39994a6edf454a72be63) if and only if ![Image 4: {\displaystyle x=0}](https://wikimedia.org/api/rest_v1/media/math/render/svg/953917eaf52f2e1baad54c8c9e3d6f9bb3710cdc)([positive definiteness](https://en.wikipedia.org/wiki/Positive-definite_function "Positive-definite function"))
•![Image 5: {\displaystyle \left|xy\right|=\left|x\right|\left|y\right|}](https://wikimedia.org/api/rest_v1/media/math/render/svg/df27721b9b788dafea519cca6742c171158af2b6)(multiplicativity)
•![Image 6: {\displaystyle \left|x+y\right|\leq \left|x\right|+\left|y\right|}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7ce392e252c2226e1f53deebd0f4d1d83b558dca)([triangle inequality](https://en.wikipedia.org/wiki/Triangle_inequality "Triangle inequality"))

It follows from the axioms that ![Image 7: {\displaystyle |1|=1,}](https://wikimedia.org/api/rest_v1/media/math/render/svg/5bf6d51b81430124e016abc59688c978f688646c)![Image 8: {\displaystyle |-1|=1,}](https://wikimedia.org/api/rest_v1/media/math/render/svg/a26880d613194dff1d0d64f31ef454695f40b996) and ![Image 9: {\displaystyle |-x|=|x|}](https://wikimedia.org/api/rest_v1/media/math/render/svg/ad1ae817ef6992006b7852312910e6759b33081f) for every ⁠![Image 10: {\displaystyle x}](https://wikimedia.org/api/rest_v1/media/math/render/svg/87f9e315fd7e2ba406057a97300593c4802b53e4)⁠. Furthermore, for every positive [integer](https://en.wikipedia.org/wiki/Integer "Integer")_n_, ![Image 11: {\displaystyle |n|\leq n,}](https://wikimedia.org/api/rest_v1/media/math/render/svg/fd2bde1b13633e87996d0be16a5c1b1b23243ff6) where the leftmost n denotes the sum of n summands equal to the [identity element](https://en.wikipedia.org/wiki/Identity_element "Identity element") of D.

The classical [absolute value](https://en.wikipedia.org/wiki/Absolute_value "Absolute value") and its [square root](https://en.wikipedia.org/wiki/Square_root "Square root") are examples of absolute values, but the square of the classical absolute value is not, as it does not fulfill the triangular inequality.

An absolute value induces a [metric](https://en.wikipedia.org/wiki/Metric_(mathematics) "Metric (mathematics)") (and thus a [topology](https://en.wikipedia.org/wiki/Topological_space#Definitions "Topological space")) on _D_ by settting ![Image 12: {\displaystyle d(x,y)=|x-y|.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/407bd64e1184ff03e4394a2d6276cc0d8e9a8f15)

Types of absolute value
-----------------------

[[edit](https://en.wikipedia.org/w/index.php?title=Absolute_value_(algebra)&action=edit&section=2 "Edit section: Types of absolute value")]

The **trivial** absolute value is the absolute value with |_x_|=0 when _x_=0 and |_x_|=1 otherwise.[[2]](https://en.wikipedia.org/wiki/Absolute_value_(algebra)#cite_note-3) Every integral domain can carry at least the trivial absolute value. The trivial value is the only possible absolute value on a [finite field](https://en.wikipedia.org/wiki/Finite_field "Finite field") because any non-zero element can be raised to some power to yield 1.

If an absolute value satisfies the stronger property |_x_+_y_|≤max(|_x_|,|_y_|) for all _x_ and _y_, then |_x_| is called an [ultrametric](https://en.wikipedia.org/wiki/Ultrametric "Ultrametric") or **non-Archimedean absolute value**, and otherwise an **Archimedean absolute value**.

If |_x_|1 and |_x_|2 are two absolute values on the same integral domain _D_, then the two absolute values are _equivalent_ if |_x_|1< 1 [if and only if](https://en.wikipedia.org/wiki/If_and_only_if "If and only if") |_x_|2< 1 for all _x_. If two nontrivial absolute values are equivalent, then for some exponent _e_ we have |_x_|1 _e_ = |_x_|2 for all _x_. Raising an absolute value to a power less than 1 results in another absolute value, but raising to a power greater than 1 does not necessarily result in an absolute value. (For instance, squaring the usual absolute value on the real numbers yields a function which is not an absolute value because it violates the rule |_x_+_y_|≤|_x_|+|_y_|.) Absolute values up to equivalence, or in other words, an [equivalence class](https://en.wikipedia.org/wiki/Equivalence_class "Equivalence class") of absolute values, is called a **[place](https://en.wikipedia.org/wiki/Algebraic_number_theory#Places "Algebraic number theory")**.

[Ostrowski's theorem](https://en.wikipedia.org/wiki/Ostrowski%27s_theorem "Ostrowski's theorem") states that the nontrivial places of the [rational numbers](https://en.wikipedia.org/wiki/Rational_numbers "Rational numbers")**Q** are the ordinary [absolute value](https://en.wikipedia.org/wiki/Absolute_value "Absolute value") and the [_p_-adic absolute value](https://en.wikipedia.org/wiki/P-adic_number "P-adic number") for each prime _p_[[3]](https://en.wikipedia.org/wiki/Absolute_value_(algebra)#cite_note-4) (see the third example above.)

Since the ordinary absolute value and the _p_-adic absolute values are absolute values according to the definition above, these define places.

If for some ultrametric absolute value and any base _b_>1, we define _ν_(_x_)=−log _b_|_x_| for _x_≠0 and _ν_(0)=∞, where ∞ is ordered to be greater than all real numbers, then we obtain a function from _D_ to **R**∪{∞}, with the following properties:

*   _ν_(_x_) = ∞ ⇒ _x_ = 0,
*   _ν_(_xy_) = _ν_(_x_)+_ν_(_y_),
*   _ν_(_x_ + _y_) ≥ min(ν(_x_), _ν_(_y_)).

Such a function is known as a _[valuation](https://en.wikipedia.org/wiki/Valuation\_(algebra) "Valuation (algebra)")_ in the terminology of [Bourbaki](https://en.wikipedia.org/wiki/Nicolas_Bourbaki "Nicolas Bourbaki"), but other authors use the term _valuation_ for _absolute value_ and then say _exponential valuation_ instead of _valuation_.

Given an integral domain _D_ with an absolute value, we can define the [Cauchy sequences](https://en.wikipedia.org/wiki/Cauchy_sequence "Cauchy sequence") of elements of _D_ with respect to the absolute value by requiring that for every ε>0 there is a positive integer _N_ such that for all integers _m_, _n_>_N_ one has |_x_ _m_ − _x_ _n_| < ε. Cauchy sequences form a [ring](https://en.wikipedia.org/wiki/Ring_(mathematics) "Ring (mathematics)") under pointwise addition and multiplication. One can also define null sequences as sequences (_a_ _n_) of elements of _D_ such that |_a_ _n_| converges to zero. Null sequences are a [prime ideal](https://en.wikipedia.org/wiki/Prime_ideal "Prime ideal") in the ring of Cauchy sequences, and the [quotient ring](https://en.wikipedia.org/wiki/Quotient_ring "Quotient ring") is therefore an integral domain. The domain _D_ is [embedded](https://en.wikipedia.org/wiki/Embedding "Embedding") in this quotient ring, called the [completion](https://en.wikipedia.org/wiki/Complete_metric_space "Complete metric space") of _D_ with respect to the absolute value |_x_|.

Since fields are integral domains, this is also a construction for the completion of a field with respect to an absolute value. To show that the result is a field, and not just an integral domain, we can either show that null sequences form a [maximal ideal](https://en.wikipedia.org/wiki/Maximal_ideal "Maximal ideal"), or else construct the inverse directly. The latter can be easily done by taking, for all nonzero elements of the quotient ring, a sequence starting from a point beyond the last zero element of the sequence. Any nonzero element of the quotient ring will differ by a null sequence from such a sequence, and by taking pointwise inversion we can find a representative inverse element.

Another theorem of [Alexander Ostrowski](https://en.wikipedia.org/wiki/Alexander_Ostrowski "Alexander Ostrowski") has it that any field complete with respect to an [Archimedean](https://en.wikipedia.org/wiki/Archimedes "Archimedes") absolute value is [isomorphic](https://en.wikipedia.org/wiki/Isomorphic "Isomorphic") to either the real or the complex numbers, and the valuation is equivalent to the usual one.[[4]](https://en.wikipedia.org/wiki/Absolute_value_(algebra)#cite_note-5) The **Gelfand-Tornheim theorem** states that any field with an Archimedean valuation is isomorphic to a [subfield](https://en.wikipedia.org/wiki/Field_extension "Field extension") of **C**, the valuation being equivalent to the usual absolute value on **C**.[[5]](https://en.wikipedia.org/wiki/Absolute_value_(algebra)#cite_note-6)

Fields and integral domains
---------------------------

[[edit](https://en.wikipedia.org/w/index.php?title=Absolute_value_(algebra)&action=edit&section=6 "Edit section: Fields and integral domains")]

If _D_ is an integral domain with absolute value |_x_|, then we may extend the definition of the absolute value to the [field of fractions](https://en.wikipedia.org/wiki/Field_of_fractions "Field of fractions") of _D_ by setting

![Image 13: {\displaystyle |x/y|=|x|/|y|.\,}](https://wikimedia.org/api/rest_v1/media/math/render/svg/10525179a3567a13a0ecd97ff3b04a38b60d1c19)
On the other hand, if _F_ is a field with ultrametric absolute value |_x_|, then the set of elements of _F_ such that |_x_| ≤ 1 defines a [valuation ring](https://en.wikipedia.org/wiki/Valuation_ring "Valuation ring"), which is a [subring](https://en.wikipedia.org/wiki/Subring "Subring")_D_ of _F_ such that for every nonzero element _x_ of _F_, at least one of _x_ or _x_−1 belongs to _D_. Since _F_ is a field, _D_ has no [zero divisors](https://en.wikipedia.org/wiki/Zero_divisor "Zero divisor") and is an integral domain. It has a unique [maximal ideal](https://en.wikipedia.org/wiki/Maximal_ideal "Maximal ideal") consisting of all _x_ such that |_x_| < 1, and is therefore a [local ring](https://en.wikipedia.org/wiki/Local_ring "Local ring").
