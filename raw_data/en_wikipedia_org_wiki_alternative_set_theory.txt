Find sources: "List of alternative set theories" – news · newspapers · books · scholar · JSTOR ( September 2023 ) See also: Vopěnka's alternative set theory In mathematical logic , an alternative set theory is any of the alternative mathematical approaches to the concept of set and any alternative to the de facto standard set theory described in axiomatic set theory by the axioms of Zermelo–Fraenkel set theory .
