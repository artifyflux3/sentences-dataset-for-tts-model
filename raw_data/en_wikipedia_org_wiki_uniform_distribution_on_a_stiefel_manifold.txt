There one often encounters integrals over the orthogonal group or over the Stiefel manifold with respect to an invariant measure. For example, this distribution arises in the study of the functional determinant under transformations involving orthogonal or semi-orthogonal matrices. The uniform distribution on the Stiefel manifold corresponds to the normalized Haar measure on the Stiefel manifold.

A random matrix uniformly distributed on the Stiefel manifold is invariant under the two-sided group action of the product O ( p ) × × O ( n ) {\displaystyle O(p)\times O(n)} of orthogonal groups, i.e.

X ∼ ∼ V 1 X V 2 {\displaystyle X\sim V_{1}XV_{2}} for all V 1 ∈ ∈ O ( p ) {\displaystyle V_{1}\in O(p)} and V 2 ∈ ∈ O ( n ) {\displaystyle V_{2}\in O(n)} .

Uniform Distribution on a Stiefel Manifold [ edit ] Introduction [ edit ] Let V p , n := V n ( R p ) {\displaystyle V_{p,n}:=V_{n}(\mathbb {R} ^{p})} be the Stiefel manifold, i.e., the set of all orthonormal n {\displaystyle n} -frames in R p {\displaystyle \mathbb {R} ^{p}} for n ≤ ≤ p {\displaystyle n\leq p} . This manifold can also be represented as the matrix set V p , n = { X ∈ ∈ R p × × n : : X ′ X = I n } {\displaystyle V_{p,n}=\{X\in \mathbb {R} ^{p\times n}\colon X'X=I_{n}\}} .

The Stiefel manifold is homeomorphic to the quotient space of the orthogonal groups V p , n ≅ ≅ O ( p ) / O ( p − − n ) .

{\displaystyle V_{p,n}\cong O(p)/O(p-n).} These two can be identified, and in the case p = n {\displaystyle p=n} we obtain the full orthogonal group. The Stiefel manifold inherits the left group action X ↦ ↦ V X , V ∈ ∈ O ( p ) .

{\displaystyle X\mapsto VX,\quad V\in O(p).} Here, O ( p − − n ) {\displaystyle O(p-n)} is a compact, closed Lie subgroup of O ( p ) {\displaystyle O(p)} . By Haar's theorem there exists a Haar measure on O ( p ) {\displaystyle O(p)} which induces an invariant measure on the quotient space O ( p ) / O ( p − − n ) {\displaystyle O(p)/O(p-n)} .

Derivation of the Haar Measure on the Stiefel Manifold [ edit ] Let X ∈ ∈ O ( p ) {\displaystyle X\in O(p)} . Differentiating X ′ X = I p {\displaystyle X'X=I_{p}} yields: d X ′ X + X ′ d X = 0.

{\displaystyle dX'X+X'dX=0.} Let x 1 , … … , x p {\displaystyle x_{1},\dots ,x_{p}} be the columns of X = ( x 1 , … … , x p ) {\displaystyle X=(x_{1},\dots ,x_{p})} . The exterior product of the superdiagonal elements defines a differential form ( X ′ d X ) := ⋀ ⋀ 1 ≤ ≤ i < j ≤ ≤ p x i ′ d x j = ⋀ ⋀ 1 ≤ ≤ i < j ≤ ≤ p ( x 1 i d x 1 j + ⋯ ⋯ + x p i d x p j ) .

{\displaystyle (X'dX):=\bigwedge _{1\leq i<j\leq p}x_{i}'dx_{j}=\bigwedge _{1\leq i<j\leq p}(x_{1i}dx_{1j}+\cdots +x_{pi}dx_{pj}).} of degree 1 2 p ( p − − 1 ) {\displaystyle {\tfrac {1}{2}}p(p-1)} . This form is invariant under both left and right group actions of the orthogonal group. Integration of this form gives the Haar measure on O ( p ) {\displaystyle O(p)} .

Let X ∈ ∈ V p , n {\displaystyle X\in V_{p,n}} be an element of the Stiefel manifold with the form X = ( x 1 , … … , x n ) {\displaystyle X=(x_{1},\dots ,x_{n})} . We extend this to an orthogonal matrix [ X : X ⊥ ⊥ ] = ( x 1 , … … , x p ) ∈ ∈ O ( p ) {\displaystyle [X:X^{\perp }]=(x_{1},\dots ,x_{p})\in O(p)} by choosing X ⊥ ⊥ = ( x n + 1 , … … , x p ) {\displaystyle X^{\perp }=(x_{n+1},\dots ,x_{p})} . The induced differential form on the Stiefel manifold is ( X ′ d X ) := ⋀ ⋀ j = 1 p − − n ⋀ ⋀ i = 1 n x n + j ′ d x i ⋀ ⋀ 1 ≤ ≤ i < j ≤ ≤ n x j ′ d x i {\displaystyle (X'dX):=\bigwedge _{j=1}^{p-n}\bigwedge _{i=1}^{n}x_{n+j}'dx_{i}\bigwedge _{1\leq i<j\leq n}x_{j}'dx_{i}} and of maximal degree 1 2 n ( 2 p − − n − − 1 ) {\displaystyle {\tfrac {1}{2}}n(2p-n-1)} .

This differential form is independent of the specific choice of X ⊥ ⊥ {\displaystyle X^{\perp }} and remains invariant under the left and right actions of the orthogonal group.

This leads to the formula v ( p , n ) := ∫ ∫ V p , n ( X ′ d X ) = 2 n π π p n / 2 Γ Γ n ( 1 2 p ) , {\displaystyle v(p,n):=\int _{V_{p,n}}(X'dX)={\frac {2^{n}\pi ^{pn/2}}{\Gamma _{n}({\tfrac {1}{2}}p)}},} where Γ Γ n {\displaystyle \Gamma _{n}} is the multivariate gamma function .
