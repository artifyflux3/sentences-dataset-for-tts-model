In [systems theory](https://en.wikipedia.org/wiki/Systems_theory "Systems theory"), a **linear system** is a [mathematical model](https://en.wikipedia.org/wiki/Mathematical_model "Mathematical model") of a [system](https://en.wikipedia.org/wiki/System "System") based on the use of a [linear operator](https://en.wikipedia.org/wiki/Linear_operator "Linear operator"). Linear systems typically exhibit features and properties that are much simpler than the [nonlinear](https://en.wikipedia.org/wiki/Nonlinear "Nonlinear") case. As a mathematical abstraction or idealization, linear systems find important applications in [automatic control](https://en.wikipedia.org/wiki/Automatic_control "Automatic control") theory, [signal processing](https://en.wikipedia.org/wiki/Signal_processing "Signal processing"), and [telecommunications](https://en.wikipedia.org/wiki/Telecommunications "Telecommunications"). For example, the propagation medium for wireless communication systems can often be modeled by linear systems.

A general [deterministic system](https://en.wikipedia.org/wiki/Deterministic_system_(mathematics) "Deterministic system (mathematics)") can be described by an operator, _H_, that maps an input, _x_(_t_), as a function of t to an output, _y_(_t_), a type of [black box](https://en.wikipedia.org/wiki/Black_box_(systems) "Black box (systems)") description.

A system is linear if and only if it satisfies the [superposition principle](https://en.wikipedia.org/wiki/Superposition_principle "Superposition principle"), or equivalently both the additivity and homogeneity properties, without restrictions (that is, for all inputs, all scaling constants and all time.)[[1]](https://en.wikipedia.org/wiki/Linear_system#cite_note-Phillips_2008-1)[[2]](https://en.wikipedia.org/wiki/Linear_system#cite_note-Bessai_2005-2)[[3]](https://en.wikipedia.org/wiki/Linear_system#cite_note-Alkin_2014-3)[[4]](https://en.wikipedia.org/wiki/Linear_system#cite_note-Nahvi_2014-4)

The superposition principle means that a linear combination of inputs to the system produces a linear combination of the individual zero-state outputs (that is, outputs setting the initial conditions to zero) corresponding to the individual inputs.[[5]](https://en.wikipedia.org/wiki/Linear_system#cite_note-Sundararajan_2008-5)[[6]](https://en.wikipedia.org/wiki/Linear_system#cite_note-Roberts_2018-6)

In a system that satisfies the homogeneity property, scaling the input always results in scaling the zero-state response by the same factor.[[6]](https://en.wikipedia.org/wiki/Linear_system#cite_note-Roberts_2018-6) In a system that satisfies the additivity property, adding two inputs always results in adding the corresponding two zero-state responses due to the individual inputs.[[6]](https://en.wikipedia.org/wiki/Linear_system#cite_note-Roberts_2018-6)

Mathematically, for a continuous-time system, given two arbitrary inputs ![Image 18: {\displaystyle {\begin{aligned}x_{1}(t)\\x_{2}(t)\end{aligned}}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/c9aa9555b44f46c7fa8e0bd4f7c8d014877a3968) as well as their respective zero-state outputs ![Image 19: {\displaystyle {\begin{aligned}y_{1}(t)&=H\left\{x_{1}(t)\right\}\\y_{2}(t)&=H\left\{x_{2}(t)\right\}\end{aligned}}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/0262a2cdd4bd9c98d748a2c5d224ff55cb0eb3f5) then a linear system must satisfy ![Image 20: {\displaystyle \alpha y_{1}(t)+\beta y_{2}(t)=H\left\{\alpha x_{1}(t)+\beta x_{2}(t)\right\}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/368bfe3e1af4d0f981caa29c60decebb5ddef1f9) for any [scalar](https://en.wikipedia.org/wiki/Scalar_(mathematics) "Scalar (mathematics)") values α and β, for any input signals _x_ 1(_t_) and _x_ 2(_t_), and for all time t.

The system is then defined by the equation _H_(_x_(_t_)) = _y_(_t_), where _y_(_t_) is some arbitrary function of time, and _x_(_t_) is the system state. Given _y_(_t_) and _H_, the system can be solved for _x_(_t_).

The behavior of the resulting system subjected to a complex input can be described as a sum of responses to simpler inputs. In nonlinear systems, there is no such relation. This mathematical property makes the solution of modelling equations simpler than many nonlinear systems. For [time-invariant](https://en.wikipedia.org/wiki/Time-invariant_system "Time-invariant system") systems this is the basis of the [impulse response](https://en.wikipedia.org/wiki/Impulse_response "Impulse response") or the [frequency response](https://en.wikipedia.org/wiki/Frequency_response "Frequency response") methods (see [LTI system theory](https://en.wikipedia.org/wiki/LTI_system_theory "LTI system theory")), which describe a general input function _x_(_t_) in terms of [unit impulses](https://en.wikipedia.org/wiki/Unit_impulse "Unit impulse") or [frequency components](https://en.wikipedia.org/wiki/Frequency_component "Frequency component").

Typical [differential equations](https://en.wikipedia.org/wiki/Differential_equation "Differential equation") of linear [time-invariant](https://en.wikipedia.org/wiki/Time-invariant_system "Time-invariant system") systems are well adapted to analysis using the [Laplace transform](https://en.wikipedia.org/wiki/Laplace_transform "Laplace transform") in the [continuous](https://en.wikipedia.org/wiki/Continuous_function "Continuous function") case, and the [Z-transform](https://en.wikipedia.org/wiki/Z-transform "Z-transform") in the [discrete](https://en.wikipedia.org/wiki/Discrete_mathematics "Discrete mathematics") case (especially in computer implementations).

Another perspective is that solutions to linear systems comprise a system of [functions](https://en.wikipedia.org/wiki/Function_(mathematics) "Function (mathematics)") which act like [vectors](https://en.wikipedia.org/wiki/Vector_(geometric) "Vector (geometric)") in the geometric sense.

A common use of linear models is to describe a nonlinear system by [linearization](https://en.wikipedia.org/wiki/Linearization "Linearization"). This is usually done for mathematical convenience.

The previous definition of a linear system is applicable to SISO (single-input single-output) systems. For MIMO (multiple-input multiple-output) systems, input and output signal vectors (![Image 21: {\displaystyle {\mathbf {x} }_{1}(t)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7ee4877feb209effd7b4f556775dfaeacc88701a), ![Image 22: {\displaystyle {\mathbf {x} }_{2}(t)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/4781d3f458ba06d3ea38c4f332d26cb1fdfd6330), ![Image 23: {\displaystyle {\mathbf {y} }_{1}(t)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/88aa88e66f2a4773ca918f0ffd72a3a4df2f2294), ![Image 24: {\displaystyle {\mathbf {y} }_{2}(t)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/361f177e88f0a3eed066a094575e7ebad17d713b)) are considered instead of input and output signals (![Image 25: {\displaystyle x_{1}(t)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/edc245640b5794e3f754bee7b713761282b60c21), ![Image 26: {\displaystyle x_{2}(t)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/8821f9acaecc995a1b0bdfea31bb01605db7142c), ![Image 27: {\displaystyle y_{1}(t)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7b17efec23e0d4f93bac22db3e978eae43f9e9a0), ![Image 28: {\displaystyle y_{2}(t)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/69d36d96452e41973c07684c23e8baf8308ad663).)[[2]](https://en.wikipedia.org/wiki/Linear_system#cite_note-Bessai_2005-2)[[4]](https://en.wikipedia.org/wiki/Linear_system#cite_note-Nahvi_2014-4)

This definition of a linear system is analogous to the definition of a [linear differential equation](https://en.wikipedia.org/wiki/Linear_differential_equation "Linear differential equation") in [calculus](https://en.wikipedia.org/wiki/Calculus "Calculus"), and a [linear transformation](https://en.wikipedia.org/wiki/Linear_map "Linear map") in [linear algebra](https://en.wikipedia.org/wiki/Linear_algebra "Linear algebra").

A [simple harmonic oscillator](https://en.wikipedia.org/wiki/Simple_harmonic_oscillator "Simple harmonic oscillator") obeys the differential equation: ![Image 29: {\displaystyle m{\frac {d^{2}(x)}{dt^{2}}}=-kx.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7fc54beac698a0d7d233d1d431df7a9e5a1cc8bc)

If ![Image 30: {\displaystyle H(x(t))=m{\frac {d^{2}(x(t))}{dt^{2}}}+kx(t),}](https://wikimedia.org/api/rest_v1/media/math/render/svg/131a287a279c878d9602c1905dad2562ada2614c) then _H_ is a linear operator. Letting _y_(_t_) = 0, we can rewrite the differential equation as _H_(_x_(_t_)) = _y_(_t_), which shows that a simple harmonic oscillator is a linear system.

The output versus input graph of a linear system need not be a straight line through the origin. For example, consider a system described by ![Image 41: {\displaystyle y(t)=k\,{\frac {\mathrm {d} x(t)}{\mathrm {d} t}}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/f02c10807337be80f18c8f96906cf7c0d045c317) (such as a constant-capacitance [capacitor](https://en.wikipedia.org/wiki/Capacitor "Capacitor") or a constant-inductance [inductor](https://en.wikipedia.org/wiki/Inductor "Inductor")). It is linear because it satisfies the superposition principle. However, when the input is a sinusoid, the output is also a sinusoid, and so its output-input plot is an ellipse centered at the origin rather than a straight line passing through the origin.

Also, the output of a linear system can contain [harmonics](https://en.wikipedia.org/wiki/Harmonic_analysis "Harmonic analysis") (and have a smaller fundamental frequency than the input) even when the input is a sinusoid. For example, consider a system described by ![Image 42: {\displaystyle y(t)=(1.5+\cos {(t)})\,x(t)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/c8d62ed5244e071eb902713543511a89ccf08f7a). It is linear because it satisfies the superposition principle. However, when the input is a sinusoid of the form ![Image 43: {\displaystyle x(t)=\cos {(3t)}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/16faf1229e2c267a509a901ebaf4af9fb0e3e1e2), using [product-to-sum trigonometric identities](https://en.wikipedia.org/wiki/List_of_trigonometric_identities#Product-to-sum_and_sum-to-product_identities "List of trigonometric identities") it can be easily shown that the output is ![Image 44: {\displaystyle y(t)=1.5\cos {(3t)}+0.5\cos {(2t)}+0.5\cos {(4t)}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/68948e507dfa27fadb8bbcecd0db6279215424a6), that is, the output doesn't consist only of sinusoids of same frequency as the input (3 rad/s), but instead also of sinusoids of frequencies 2 rad/s and 4 rad/s; furthermore, taking the [least common multiple](https://en.wikipedia.org/wiki/Least_common_multiple "Least common multiple") of the fundamental period of the sinusoids of the output, it can be shown the fundamental angular frequency of the output is 1 rad/s, which is different than that of the input.

Time-varying impulse response
-----------------------------

[[edit](https://en.wikipedia.org/w/index.php?title=Linear_system&action=edit&section=3 "Edit section: Time-varying impulse response")]

The **time-varying impulse response**_h_(_t_ 2, _t_ 1) of a linear system is defined as the response of the system at time _t_ = _t_ 2 to a single [impulse](https://en.wikipedia.org/wiki/Impulse_function "Impulse function") applied at time _t_ = _t_ 1. In other words, if the input _x_(_t_) to a linear system is ![Image 45: {\displaystyle x(t)=\delta (t-t_{1})}](https://wikimedia.org/api/rest_v1/media/math/render/svg/fd4aecca59766e6651844aaed1050199b1ecc1a5) where δ(_t_) represents the [Dirac delta function](https://en.wikipedia.org/wiki/Dirac_delta_function "Dirac delta function"), and the corresponding response _y_(_t_) of the system is ![Image 46: {\displaystyle y(t=t_{2})=h(t_{2},t_{1})}](https://wikimedia.org/api/rest_v1/media/math/render/svg/4a1b87b9e2a50f228b6b8d225a3bcad28ab8cf57) then the function _h_(_t_ 2, _t_ 1) is the time-varying impulse response of the system. Since the system cannot respond before the input is applied the following **causality condition** must be satisfied: ![Image 47: {\displaystyle h(t_{2},t_{1})=0,t_{2}<t_{1}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/77a3b9cd0dd5d43b483907ed75c5e541d233e404)

The convolution integral
------------------------

[[edit](https://en.wikipedia.org/w/index.php?title=Linear_system&action=edit&section=4 "Edit section: The convolution integral")]

The output of any general continuous-time linear system is related to the input by an integral which may be written over a doubly infinite range because of the causality condition: ![Image 48: {\displaystyle y(t)=\int _{-\infty }^{t}h(t,t')x(t')dt'=\int _{-\infty }^{\infty }h(t,t')x(t')dt'}](https://wikimedia.org/api/rest_v1/media/math/render/svg/33d4c7be98d7eea95010a74fa10d65b7b49ae0ce)

If the properties of the system do not depend on the time at which it is operated then it is said to be **time-invariant** and h is a function only of the time difference _τ_ = _t_ − _t'_ which is zero for _τ_< 0 (namely _t_<_t'_). By redefinition of h it is then possible to write the input-output relation equivalently in any of the ways, ![Image 49: {\displaystyle y(t)=\int _{-\infty }^{t}h(t-t')x(t')dt'=\int _{-\infty }^{\infty }h(t-t')x(t')dt'=\int _{-\infty }^{\infty }h(\tau )x(t-\tau )d\tau =\int _{0}^{\infty }h(\tau )x(t-\tau )d\tau }](https://wikimedia.org/api/rest_v1/media/math/render/svg/4c4ea3252b2eebc1d36dd95f1cbb7fb0fffc0f57)

Linear time-invariant systems are most commonly characterized by the Laplace transform of the impulse response function called the _transfer function_ which is: ![Image 50: {\displaystyle H(s)=\int _{0}^{\infty }h(t)e^{-st}\,dt.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/4385ed3642b2b896fcd5df4805435321d8d0d958)

In applications this is usually a rational algebraic function of s. Because _h_(_t_) is zero for negative t, the integral may equally be written over the doubly infinite range and putting _s_ = _iω_ follows the formula for the _frequency response function_: ![Image 51: {\displaystyle H(i\omega )=\int _{-\infty }^{\infty }h(t)e^{-i\omega t}dt}](https://wikimedia.org/api/rest_v1/media/math/render/svg/705a41bb558c3de4a5e15e032718f8e264687dad)

Discrete-time systems
---------------------

[[edit](https://en.wikipedia.org/w/index.php?title=Linear_system&action=edit&section=5 "Edit section: Discrete-time systems")]

The output of any discrete time linear system is related to the input by the time-varying convolution sum: ![Image 52: {\displaystyle y[n]=\sum _{m=-\infty }^{n}{h[n,m]x[m]}=\sum _{m=-\infty }^{\infty }{h[n,m]x[m]}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/a8de98ac398c3891590682a4fa7ffe2b38e09263) or equivalently for a time-invariant system on redefining _h_, ![Image 53: {\displaystyle y[n]=\sum _{k=0}^{\infty }{h[k]x[n-k]}=\sum _{k=-\infty }^{\infty }{h[k]x[n-k]}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/05629aebe6564c41b4c709875918593e99f0bcbd) where ![Image 54: {\displaystyle k=n-m}](https://wikimedia.org/api/rest_v1/media/math/render/svg/a9b797cfa06598ecf97c060c83702d58c212f70e) represents the lag time between the stimulus at time _m_ and the response at time _n_.
