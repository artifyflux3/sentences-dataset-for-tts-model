From Wikipedia, the free encyclopedia

Data analysis process

In [statistics](https://en.wikipedia.org/wiki/Statistics "Statistics"), [econometrics](https://en.wikipedia.org/wiki/Econometrics "Econometrics") and related fields, **multidimensional analysis** (**MDA**) is a [data analysis](https://en.wikipedia.org/wiki/Data_analysis "Data analysis") process that groups data into two categories: [data dimensions](https://en.wikipedia.org/wiki/Data_dimension "Data dimension") and measurements. For example, a [data set](https://en.wikipedia.org/wiki/Data_set "Data set") consisting of the number of wins for a single football team at each of several years is a single-dimensional (in this case, longitudinal) data set. A data set consisting of the number of wins for several football teams in a single year is also a single-dimensional (in this case, cross-sectional) data set. A data set consisting of the number of wins for several football teams over several years is a two-dimensional data set.

Higher dimensions
-----------------

[[edit](https://en.wikipedia.org/w/index.php?title=Multidimensional_analysis&action=edit&section=1 "Edit section: Higher dimensions")]

In many disciplines, two-dimensional data sets are also called [panel data](https://en.wikipedia.org/wiki/Panel_data "Panel data").[[1]](https://en.wikipedia.org/wiki/Multidimensional_analysis#cite_note-1) While, strictly speaking, two- and higher-dimensional data sets are "multi-dimensional", the term "multidimensional" tends to be applied only to data sets with three or more dimensions.[[2]](https://en.wikipedia.org/wiki/Multidimensional_analysis#cite_note-2) For example, some forecast data sets provide forecasts for multiple target periods, conducted by multiple forecasters, and made at multiple horizons. The three dimensions provide more information than can be gleaned from two-dimensional panel data sets.

Software
--------

[[edit](https://en.wikipedia.org/w/index.php?title=Multidimensional_analysis&action=edit&section=2 "Edit section: Software")]

Computer software for MDA include [Online analytical processing](https://en.wikipedia.org/wiki/Online_analytical_processing "Online analytical processing") (OLAP) for data in [relational databases](https://en.wikipedia.org/wiki/Relational_database "Relational database"), [pivot tables](https://en.wikipedia.org/wiki/Pivot_table "Pivot table") for data in [spreadsheets](https://en.wikipedia.org/wiki/Spreadsheet "Spreadsheet"), and [Array DBMSs](https://en.wikipedia.org/wiki/Array_DBMSs "Array DBMSs") for general multi-dimensional data (such as [raster data](https://en.wikipedia.org/wiki/Raster_data "Raster data")) in science, engineering, and business.
