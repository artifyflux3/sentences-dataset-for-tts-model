From Wikipedia, the free encyclopedia

Point of a subset S around which there are no other points of S

[![Image 24](https://upload.wikimedia.org/wikipedia/commons/thumb/2/2a/%C4%90i%E1%BB%83m_c%C3%B4_l%E1%BA%ADp-Isolated_point.svg/500px-%C4%90i%E1%BB%83m_c%C3%B4_l%E1%BA%ADp-Isolated_point.svg.png)](https://en.wikipedia.org/wiki/File:%C4%90i%E1%BB%83m_c%C3%B4_l%E1%BA%ADp-Isolated_point.svg)

"0" is an isolated point of A={0}∪[1,2]{\displaystyle A=\{0\}\cup [1,2]}![Image 25: {\displaystyle A=\{0\}\cup [1,2]}](https://wikimedia.org/api/rest_v1/media/math/render/svg/0bbddfd3db81d8d0a085f79b267d9f99a15efc5c)

In [mathematics](https://en.wikipedia.org/wiki/Mathematics "Mathematics"), a [point](https://en.wikipedia.org/wiki/Point_(topology) "Point (topology)")x is called an **isolated point** of a subset S (in a [topological space](https://en.wikipedia.org/wiki/Topological_space "Topological space")X) if x is an element of S and there exists a [neighborhood](https://en.wikipedia.org/wiki/Neighborhood_(mathematics) "Neighborhood (mathematics)") of x that does not contain any other points of S. This is equivalent to saying that the [singleton](https://en.wikipedia.org/wiki/Singleton_(mathematics) "Singleton (mathematics)"){_x_}  is an [open set](https://en.wikipedia.org/wiki/Open_set "Open set") in the topological space S (considered as a [subspace](https://en.wikipedia.org/wiki/Subspace_topology "Subspace topology") of X). Another equivalent formulation is: an element x of S is an isolated point of S if and only if it is not a [limit point](https://en.wikipedia.org/wiki/Limit_point "Limit point") of S.

If the space X is a [metric space](https://en.wikipedia.org/wiki/Metric_space "Metric space"), for example a [Euclidean space](https://en.wikipedia.org/wiki/Euclidean_space "Euclidean space"), then an element x of S is an isolated point of S if there exists an [open ball](https://en.wikipedia.org/wiki/Open_ball "Open ball") around x that contains only finitely many elements of S. A [point set](https://en.wikipedia.org/wiki/Point_set "Point set") that is made up only of isolated points is called a **discrete set** or **discrete point set** (see also [discrete space](https://en.wikipedia.org/wiki/Discrete_space "Discrete space")).

Related notions
---------------

[[edit](https://en.wikipedia.org/w/index.php?title=Isolated_point&action=edit&section=1 "Edit section: Related notions")]

Any discrete subset S of Euclidean space must be [countable](https://en.wikipedia.org/wiki/Countable "Countable"), since the isolation of each of its points together with the fact that [rationals](https://en.wikipedia.org/wiki/Rational_number "Rational number") are [dense](https://en.wikipedia.org/wiki/Dense_set "Dense set") in the [reals](https://en.wikipedia.org/wiki/Real_number "Real number") means that the points of S may be mapped injectively onto a set of points with rational coordinates, of which there are only countably many. However, not every countable set is discrete, of which the rational numbers under the usual Euclidean metric are the canonical example.

A set with no isolated point is said to be _[dense-in-itself](https://en.wikipedia.org/wiki/Dense-in-itself "Dense-in-itself")_ (every neighbourhood of a point contains other points of the set). A [closed set](https://en.wikipedia.org/wiki/Closed_set "Closed set") with no isolated point is called a _[perfect set](https://en.wikipedia.org/wiki/Perfect\_set "Perfect set")_ (it contains all its limit points and no isolated points).

The number of isolated points is a [topological invariant](https://en.wikipedia.org/wiki/Topological_invariant "Topological invariant"), i.e. if two [topological spaces](https://en.wikipedia.org/wiki/Topological_spaces "Topological spaces")X, Y are [homeomorphic](https://en.wikipedia.org/wiki/Homeomorphic "Homeomorphic"), the number of isolated points in each is equal.

Examples
--------

[[edit](https://en.wikipedia.org/w/index.php?title=Isolated_point&action=edit&section=2 "Edit section: Examples")]

### Standard examples

[[edit](https://en.wikipedia.org/w/index.php?title=Isolated_point&action=edit&section=3 "Edit section: Standard examples")]

[Topological spaces](https://en.wikipedia.org/wiki/Topological_space "Topological space") in the following three examples are considered as [subspaces](https://en.wikipedia.org/wiki/Subspace_topology "Subspace topology") of the [real line](https://en.wikipedia.org/wiki/Real_line "Real line") with the standard topology.

*   For the set S={0}∪[1,2],{\displaystyle S=\{0\}\cup [1,2],}![Image 26: {\displaystyle S=\{0\}\cup [1,2],}](https://wikimedia.org/api/rest_v1/media/math/render/svg/3d6ebff649777113c6ceef42f8818f038af64c38) the point 0 is an isolated point.
*   For the set S={0}∪{1,1 2,1 3,…},{\displaystyle S=\{0\}\cup \{1,{\tfrac {1}{2}},{\tfrac {1}{3}},\dots \},}![Image 27: {\displaystyle S=\{0\}\cup \{1,{\tfrac {1}{2}},{\tfrac {1}{3}},\dots \},}](https://wikimedia.org/api/rest_v1/media/math/render/svg/b3986f11a6874923c0d36f3d9634c5c7bad0a2dc) each of the points ⁠1 k{\displaystyle {\tfrac {1}{k}}}![Image 28: {\displaystyle {\tfrac {1}{k}}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/5ef726cad1d26aea3b5d999f914d28e8d88d5071)⁠ is an isolated point, but 0 is not an isolated point because there are other points in S as close to 0 as desired.
*   The set N={0,1,2,…}{\displaystyle \mathbb {N} =\{0,1,2,\ldots \}}![Image 29: {\displaystyle \mathbb {N} =\{0,1,2,\ldots \}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/6a6259fa720c155df5af3c96fda39822267e8bf0) of [natural numbers](https://en.wikipedia.org/wiki/Natural_number "Natural number") is a discrete set.

In the topological space X={a,b}{\displaystyle X=\{a,b\}}![Image 30: {\displaystyle X=\{a,b\}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/fd49abd3a2f5f1fa81958c55cc0150e83d08c9fa) with topology τ={∅,{a},X},{\displaystyle \tau =\{\emptyset ,\{a\},X\},}![Image 31: {\displaystyle \tau =\{\emptyset ,\{a\},X\},}](https://wikimedia.org/api/rest_v1/media/math/render/svg/85fc8b5b38b124d0dc4c839685a6dc16bef9c9c1) the element a is an isolated point, even though b{\displaystyle b}![Image 32: {\displaystyle b}](https://wikimedia.org/api/rest_v1/media/math/render/svg/f11423fbb2e967f986e36804a8ae4271734917c3) belongs to the [closure](https://en.wikipedia.org/wiki/Closure_(mathematics) "Closure (mathematics)") of {a}{\displaystyle \{a\}}![Image 33: {\displaystyle \{a\}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/ae00d3f70c2221d5933b6b03d14b1df54756c8ee) (and is therefore, in some sense, "close" to a). Such a situation is not possible in a [Hausdorff space](https://en.wikipedia.org/wiki/Hausdorff_space "Hausdorff space").

The [Morse lemma](https://en.wikipedia.org/wiki/Morse_theory#Morse_lemma "Morse theory") states that [non-degenerate critical points](https://en.wikipedia.org/wiki/Non-degenerate_critical_point "Non-degenerate critical point") of certain functions are isolated.

### Two counter-intuitive examples

[[edit](https://en.wikipedia.org/w/index.php?title=Isolated_point&action=edit&section=4 "Edit section: Two counter-intuitive examples")]

Consider the set F of points x in the real interval (0,1) such that every digit x i of their [binary](https://en.wikipedia.org/wiki/Binary_number "Binary number") representation fulfills the following conditions:

*   Either x i=0{\displaystyle x_{i}=0}![Image 34: {\displaystyle x_{i}=0}](https://wikimedia.org/api/rest_v1/media/math/render/svg/491e215fe273840aab63acab32e73cb14f2fc8e8) or x i=1.{\displaystyle x_{i}=1.}![Image 35: {\displaystyle x_{i}=1.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/b3b56d61cc8dac136f829b93b24ba5c15c060eb6)
*   x i=1{\displaystyle x_{i}=1}![Image 36: {\displaystyle x_{i}=1}](https://wikimedia.org/api/rest_v1/media/math/render/svg/a230692f2eb10d5fc1ead43dd83d75f3b592a09b) only for finitely many indices i.
*   If m denotes the largest index such that x m=1,{\displaystyle x_{m}=1,}![Image 37: {\displaystyle x_{m}=1,}](https://wikimedia.org/api/rest_v1/media/math/render/svg/a3c96b442ba4a3057621aca0f587e7c8464663d1) then x m−1=0.{\displaystyle x_{m-1}=0.}![Image 38: {\displaystyle x_{m-1}=0.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/1ddc2cd07d8c0f5981cb6607b0e1a0ae851a48af)
*   If x i=1{\displaystyle x_{i}=1}![Image 39: {\displaystyle x_{i}=1}](https://wikimedia.org/api/rest_v1/media/math/render/svg/a230692f2eb10d5fc1ead43dd83d75f3b592a09b) and i<m,{\displaystyle i<m,}![Image 40: {\displaystyle i<m,}](https://wikimedia.org/api/rest_v1/media/math/render/svg/e494980a471ef529ea3a56d240c1ad8bcefc2321) then exactly one of the following two conditions holds: x i−1=1{\displaystyle x_{i-1}=1}![Image 41: {\displaystyle x_{i-1}=1}](https://wikimedia.org/api/rest_v1/media/math/render/svg/d79b6ed190df1f9c9198d311da937a9afd10b025) or x i+1=1.{\displaystyle x_{i+1}=1.}![Image 42: {\displaystyle x_{i+1}=1.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/77fc5f62ca1bd647e0102c763baad164bca79555)

Informally, these conditions means that every digit of the binary representation of x{\displaystyle x}![Image 43: {\displaystyle x}](https://wikimedia.org/api/rest_v1/media/math/render/svg/87f9e315fd7e2ba406057a97300593c4802b53e4) that equals 1 belongs to a pair ...0110..., except for ...010... at the very end.

Now, F is an explicit set consisting entirely of isolated points but has the counter-intuitive property that its [closure](https://en.wikipedia.org/wiki/Closure_(topology) "Closure (topology)") is an [uncountable set](https://en.wikipedia.org/wiki/Uncountable_set "Uncountable set").[[1]](https://en.wikipedia.org/wiki/Isolated_point#cite_note-1)

Another set F with the same properties can be obtained as follows. Let C be the middle-thirds [Cantor set](https://en.wikipedia.org/wiki/Cantor_set "Cantor set"), let I 1,I 2,I 3,…,I k,…{\displaystyle I_{1},I_{2},I_{3},\ldots ,I_{k},\ldots }![Image 44: {\displaystyle I_{1},I_{2},I_{3},\ldots ,I_{k},\ldots }](https://wikimedia.org/api/rest_v1/media/math/render/svg/70ea2f8aa702d5d2ae33141e5fdef77cbd7131d3) be the [component](https://en.wikipedia.org/wiki/Connected_space#Connected_components "Connected space") intervals of [0,1]−C{\displaystyle [0,1]-C}![Image 45: {\displaystyle [0,1]-C}](https://wikimedia.org/api/rest_v1/media/math/render/svg/2dc84e2f8bf626b9355157a77012d6d4878855e9), and let F be a set consisting of one point from each I k. Since each I k contains only one point from F, every point of F is an isolated point. However, if p is any point in the Cantor set, then every neighborhood of p contains at least one I k, and hence at least one point of F. It follows that each point of the Cantor set lies in the closure of F, and therefore F has uncountable closure.
