( February 2011 ) ( Learn how and when to remove this message ) The Routh array is a tabular method permitting one to establish the stability of a system using only the coefficients of the characteristic polynomial .  Central to the field of control systems design , the Routh–Hurwitz theorem and Routh array emerge by using the Euclidean algorithm and Sturm's theorem in evaluating Cauchy indices .
