A one-to-one correspondence, comparing a set of apples to a set of oranges, showing they have the same cardinality.

In [mathematics](https://en.wikipedia.org/wiki/Mathematics "Mathematics"), **cardinality** is an intrinsic property of [sets](https://en.wikipedia.org/wiki/Set_(mathematics) "Set (mathematics)"), roughly meaning the number of individual [objects](https://en.wikipedia.org/wiki/Mathematical_object "Mathematical object") they contain, which may be [infinite](https://en.wikipedia.org/wiki/Infinity "Infinity"). The cardinal number corresponding to a set ![Image 2: {\displaystyle A}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7daff47fa58cdfd29dc333def748ff5fa4c923e3) is written as ![Image 3: {\displaystyle |A|}](https://wikimedia.org/api/rest_v1/media/math/render/svg/648fce92f29d925f04d39244ccfe435320dfc6de) between two vertical bars. For [finite sets](https://en.wikipedia.org/wiki/Finite_sets "Finite sets"), cardinality coincides with the [natural number](https://en.wikipedia.org/wiki/Natural_number "Natural number") found by counting its elements. Beginning in the late 19th century, this concept of cardinality was generalized to [infinite sets](https://en.wikipedia.org/wiki/Infinite_set "Infinite set").

Two sets are said to be _**equinumerous**_ or _have the same cardinality_ if there exists a [one-to-one correspondence](https://en.wikipedia.org/wiki/One-to-one_correspondence "One-to-one correspondence") between them. That is, if their objects can be paired such that each object has a pair, and no object is paired more than once (see image). A set is [countably infinite](https://en.wikipedia.org/wiki/Countable_set "Countable set") if it can be placed in one-to-one correspondence with the set of natural numbers ![Image 4: {\displaystyle \{1,2,3,4,\cdots \}.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/776c32433681adce16d3071f28ce2790a012320b) For example, the set of even numbers ![Image 5: {\displaystyle \{2,4,6,..\}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/44b3e9c185385084ef003416467905f8e639cb3f), the set of [prime numbers](https://en.wikipedia.org/wiki/Prime_numbers "Prime numbers")![Image 6: {\displaystyle \{2,3,5,\cdots \}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/5339893f6cb0a33283ffbc11d596e51a2a266ec1), and the set of [rational numbers](https://en.wikipedia.org/wiki/Rational_numbers "Rational numbers") are all countable. A set is [uncountable](https://en.wikipedia.org/wiki/Uncountable_set "Uncountable set") if it is both infinite and cannot be put in correspondence with the set of natural numbers—for example, the set of [real numbers](https://en.wikipedia.org/wiki/Real_numbers "Real numbers") or the [powerset](https://en.wikipedia.org/wiki/Powerset "Powerset") of the set of natural numbers.

[Cardinal numbers](https://en.wikipedia.org/wiki/Cardinal_number "Cardinal number") extend the natural numbers as representatives of size. Most commonly, the [aleph numbers](https://en.wikipedia.org/wiki/Aleph_numbers "Aleph numbers") are defined via [ordinal numbers](https://en.wikipedia.org/wiki/Ordinal_number "Ordinal number"), and represent a large class of sets. The question of whether there is a set whose cardinality is greater than that of the integers but less than that of the real numbers, is known as the [continuum hypothesis](https://en.wikipedia.org/wiki/Continuum_hypothesis "Continuum hypothesis"), which has been shown to be [unprovable](https://en.wikipedia.org/wiki/Unprovable "Unprovable") in standard set theories such as [Zermelo–Fraenkel set theory](https://en.wikipedia.org/wiki/Zermelo%E2%80%93Fraenkel_set_theory "Zermelo–Fraenkel set theory").

Definition and etymology
------------------------

[[edit](https://en.wikipedia.org/w/index.php?title=Cardinality&action=edit&section=1 "Edit section: Definition and etymology")]

Cardinality is an intrinsic property of [sets](https://en.wikipedia.org/wiki/Set_(mathematics) "Set (mathematics)") which defines their size, roughly corresponding to the number of individual [objects](https://en.wikipedia.org/wiki/Mathematical_object "Mathematical object") they contain.[[1]](https://en.wikipedia.org/wiki/Cardinality#cite_note-1) Fundamentally however, it is different from the concepts of [number](https://en.wikipedia.org/wiki/Number "Number") or [counting](https://en.wikipedia.org/wiki/Counting "Counting") as the cardinalities of two sets can be compared without referring to their number of elements, or defining number at all. For example, in the image [above](https://en.wikipedia.org/wiki/Cardinality#top), a set of apples is compared to a set of oranges such that every fruit is used exactly once which shows these two sets have the same cardinality, even if one doesn't know how many of each there are.[[2]](https://en.wikipedia.org/wiki/Cardinality#cite_note-2)[[3]](https://en.wikipedia.org/wiki/Cardinality#cite_note-3) Thus, cardinality is measured by putting sets in [one-to-one correspondence](https://en.wikipedia.org/wiki/One-to-one_correspondence "One-to-one correspondence"). If it is possible, the sets are said to have the _same cardinality_, and if not, one set is said to be _strictly larger_ or _strictly smaller_ than the other.[[4]](https://en.wikipedia.org/wiki/Cardinality#cite_note-4)

[Georg Cantor](https://en.wikipedia.org/wiki/Georg_Cantor "Georg Cantor"), the originator of the concept, defined cardinality as "the general concept which, with the aid of our intelligence, results from a set when we abstract from the nature of its various elements and from the order of their being given."[[5]](https://en.wikipedia.org/wiki/Cardinality#cite_note-5) This definition was considered to be imprecise, unclear, and purely psychological.[[6]](https://en.wikipedia.org/wiki/Cardinality#cite_note-6) Thus, [cardinal numbers](https://en.wikipedia.org/wiki/Cardinality#Cardinal_numbers), a means of measuring cardinality, became the main way of presenting the concept. The distinction between the two is roughly analogous to the difference between an object's [mass](https://en.wikipedia.org/wiki/Mass "Mass") and its mass _in [kilograms](https://en.wikipedia.org/wiki/Kilograms "Kilograms")_.[[7]](https://en.wikipedia.org/wiki/Cardinality#cite_note-7) However, somewhat confusingly, the phrases "The cardinality of M" and "The cardinal number of M" are used interchangeably.

In English, the term _cardinality_ originates from the [post-classical Latin](https://en.wikipedia.org/wiki/Post-Classical_Latin_language "Post-Classical Latin language")_cardinalis_, meaning "principal" or "chief", which derives from _cardo_, a noun meaning "hinge". In Latin, _cardo_ referred to something central or pivotal, both literally and metaphorically. This concept of centrality passed into [medieval Latin](https://en.wikipedia.org/wiki/Medieval_Latin "Medieval Latin") and then into English, where _cardinal_ came to describe things considered to be, in some sense, fundamental, such as _[cardinal virtues](https://en.wikipedia.org/wiki/Cardinal\_virtues "Cardinal virtues")_, _[cardinal sins](https://en.wikipedia.org/wiki/Cardinal\_sins "Cardinal sins")_, _[cardinal directions](https://en.wikipedia.org/wiki/Cardinal\_directions "Cardinal directions")_, and (grammatically defined) _[cardinal numbers](https://en.wikipedia.org/wiki/Cardinal\_numbers\_(linguistics) "Cardinal numbers (linguistics)")_.[[8]](https://en.wikipedia.org/wiki/Cardinality#cite_note-8)[[9]](https://en.wikipedia.org/wiki/Cardinality#cite_note-9) The last of which referred to numbers used for counting (e.g., _one_, _two_, _three_),[[10]](https://en.wikipedia.org/wiki/Cardinality#cite_note-10) as opposed to _[ordinal numbers](https://en.wikipedia.org/wiki/Ordinal\_numbers\_(linguistics) "Ordinal numbers (linguistics)")_, which express order (e.g., _first, second, third_),[[11]](https://en.wikipedia.org/wiki/Cardinality#cite_note-11) and _[nominal numbers](https://en.wikipedia.org/wiki/Nominal\_number "Nominal number")_ used for labeling without meaning (e.g., [jersey numbers](https://en.wikipedia.org/wiki/Jersey_numbers "Jersey numbers") and [serial numbers](https://en.wikipedia.org/wiki/Serial_numbers "Serial numbers")).[[12]](https://en.wikipedia.org/wiki/Cardinality#cite_note-12) In mathematics, the notion of cardinality was first introduced by [Georg Cantor](https://en.wikipedia.org/wiki/Georg_Cantor "Georg Cantor") in the late 19th century, wherein he used the used the term _Mächtigkeit_, which may be translated as "magnitude" or "power", though Cantor credited the term to a work by [Jakob Steiner](https://en.wikipedia.org/wiki/Jakob_Steiner "Jakob Steiner") on [projective geometry](https://en.wikipedia.org/wiki/Projective_geometry "Projective geometry").[[13]](https://en.wikipedia.org/wiki/Cardinality#cite_note-13)[[14]](https://en.wikipedia.org/wiki/Cardinality#cite_note-:3-14)[[15]](https://en.wikipedia.org/wiki/Cardinality#cite_note-:4-15) The terms _cardinality_ and _cardinal number_ were eventually adopted from the grammatical sense, and later translations would use these terms.[[16]](https://en.wikipedia.org/wiki/Cardinality#cite_note-16)[[17]](https://en.wikipedia.org/wiki/Cardinality#cite_note-17)

[![Image 7](https://upload.wikimedia.org/wikipedia/commons/thumb/9/9c/AristotlesWheelLabeledDiagram.svg/330px-AristotlesWheelLabeledDiagram.svg.png)](https://en.wikipedia.org/wiki/File:AristotlesWheelLabeledDiagram.svg)

Diagram of Aristotle's wheel as described in _Mechanica_

From the 6th century BCE, the writings of [Greek philosophers](https://en.wikipedia.org/wiki/Greek_philosophers "Greek philosophers"), such as [Anaximander](https://en.wikipedia.org/wiki/Anaximander "Anaximander"), show hints of comparing infinite sets or shapes, however, it was generally viewed as paradoxical and imperfect (cf. _[Zeno's paradoxes](https://en.wikipedia.org/wiki/Zeno%27s\_paradoxes "Zeno's paradoxes")_).[[18]](https://en.wikipedia.org/wiki/Cardinality#cite_note-Allen22-18)[Aristotle](https://en.wikipedia.org/wiki/Aristotle "Aristotle") distinguished between the notions of [actual infinity](https://en.wikipedia.org/wiki/Actual_infinity "Actual infinity") and potential infinity, arguing that Greek mathematicians understood the difference, and that they "do not need the [actual] infinite and do not use it."[[19]](https://en.wikipedia.org/wiki/Cardinality#cite_note-19) The Greek notion of number (_αριθμός_, _arithmos_) was used exclusively for a definite number of definite objects (i.e. finite numbers).[[20]](https://en.wikipedia.org/wiki/Cardinality#cite_note-20) This would be codified in [Euclid's _Elements_](https://en.wikipedia.org/wiki/Euclid%27s_Elements "Euclid's Elements"), where the [fifth common notion](https://en.wikipedia.org/wiki/Euclidean_geometry#common_notions "Euclidean geometry") states "The whole is greater than the part", often called the _Euclidean principle_. This principle would be the dominating philosophy in mathematics until the 19th century.[[18]](https://en.wikipedia.org/wiki/Cardinality#cite_note-Allen22-18)[[21]](https://en.wikipedia.org/wiki/Cardinality#cite_note-21)

Around the 4th century BCE, [Jaina mathematics](https://en.wikipedia.org/wiki/Jaina_mathematics "Jaina mathematics") would be the first to discuss different sizes of infinity. They defined three major classes of number: enumerable (finite numbers), unenumerable (_asamkhyata_, roughly, [countably infinite](https://en.wikipedia.org/wiki/Cardinality#Countable_sets)), and infinite (_ananta_). Then they had five classes of infinite numbers: infinite in one direction, infinite in both directions, infinite in area, infinite everywhere, and infinite perpetually.[[22]](https://en.wikipedia.org/wiki/Cardinality#cite_note-22)[[23]](https://en.wikipedia.org/wiki/Cardinality#cite_note-23)

One of the earliest explicit uses of a one-to-one correspondence is recorded in [Aristotle's _Mechanics_](https://en.wikipedia.org/wiki/Mechanics_(Aristotle) "Mechanics (Aristotle)") (c. 350 BCE), known as [Aristotle's wheel paradox](https://en.wikipedia.org/wiki/Aristotle%27s_wheel_paradox "Aristotle's wheel paradox"). The paradox can be briefly described as follows: A wheel is depicted as two [concentric circles](https://en.wikipedia.org/wiki/Concentric_circles "Concentric circles"). The larger, outer circle is tangent to a horizontal line (e.g. a road that it rolls on), while the smaller, inner circle is rigidly affixed to the larger. Assuming the larger circle rolls along the line without slipping (or skidding) for one full revolution, the distances moved by both circles are the same: the [circumference](https://en.wikipedia.org/wiki/Circumference "Circumference") of the larger circle. Further, the lines traced by the bottom-most point of each is the same length.[[24]](https://en.wikipedia.org/wiki/Cardinality#cite_note-:0-24) Since the smaller wheel does not skip any points, and no point on the smaller wheel is used more than once, there is a one-to-one correspondence between the two circles.[[25]](https://en.wikipedia.org/wiki/Cardinality#cite_note-25)[[26]](https://en.wikipedia.org/wiki/Cardinality#cite_note-26)[[27]](https://en.wikipedia.org/wiki/Cardinality#cite_note-27)

### Pre-Cantorian set theory

[[edit](https://en.wikipedia.org/w/index.php?title=Cardinality&action=edit&section=4 "Edit section: Pre-Cantorian set theory")]

[![Image 8](https://upload.wikimedia.org/wikipedia/commons/thumb/f/fc/Galileo_Galilei_%281564-1642%29_RMG_BHC2700.tiff/lossy-page1-168px-Galileo_Galilei_%281564-1642%29_RMG_BHC2700.tiff.jpg)](https://en.wikipedia.org/wiki/File:Galileo_Galilei_(1564-1642)_RMG_BHC2700.tiff)

[![Image 9](https://upload.wikimedia.org/wikipedia/commons/thumb/9/9b/Bernard_Bolzano.jpg/250px-Bernard_Bolzano.jpg)](https://en.wikipedia.org/wiki/File:Bernard_Bolzano.jpg)

[Galileo Galilei](https://en.wikipedia.org/wiki/Galileo_Galilei "Galileo Galilei") presented what was later coined [Galileo's paradox](https://en.wikipedia.org/wiki/Galileo%27s_paradox "Galileo's paradox") in his book _[Two New Sciences](https://en.wikipedia.org/wiki/Two\_New\_Sciences "Two New Sciences")_ (1638),[[28]](https://en.wikipedia.org/wiki/Cardinality#cite_note-28) where he presents a seeming paradox in infinite sequences of numbers. It goes roughly as follows: for each [perfect square](https://en.wikipedia.org/wiki/Square_number "Square number")![Image 10: {\displaystyle (n^{2})}](https://wikimedia.org/api/rest_v1/media/math/render/svg/52376d02d45e1ac7c4cd8d8208c2f4ae104e3098) 1, 4, 9, 16, and so on, there is a unique [square root](https://en.wikipedia.org/wiki/Square_root "Square root")![Image 11: {\textstyle ({\sqrt {n^{2}}}=n)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/e8070f703090fba8366932a60f34f360ae518433) 1, 2, 3, 4, and so on. Therefore, there are as many perfect squares as there are square roots. However, every number is a square root, since it can be [squared](https://en.wikipedia.org/wiki/Square_(algebra) "Square (algebra)"), but not every number is a perfect square. Moreover, the proportion of perfect squares as one passes larger values diminishes, and is eventually smaller than any given fraction. Galileo denied that this was fundamentally contradictory, however he concluded that this meant we could not compare the sizes of infinite sets, missing the opportunity to discover cardinality.[[29]](https://en.wikipedia.org/wiki/Cardinality#cite_note-29)

In _[A Treatise of Human Nature](https://en.wikipedia.org/wiki/A\_Treatise\_of\_Human\_Nature "A Treatise of Human Nature")_ (1739), [David Hume](https://en.wikipedia.org/wiki/David_Hume "David Hume") is quoted for saying _"When two numbers are so combined, as that the one has always a unit answering to every unit of the other, we pronounce them equal",[[30]](https://en.wikipedia.org/wiki/Cardinality#cite\_note-30)_ now called _[Hume's principle](https://en.wikipedia.org/wiki/Hume%27s\_principle "Hume's principle")_, which was used extensively by [Gottlob Frege](https://en.wikipedia.org/wiki/Gottlob_Frege "Gottlob Frege") later during the rise of set theory.[[31]](https://en.wikipedia.org/wiki/Cardinality#cite_note-:1-31)[[32]](https://en.wikipedia.org/wiki/Cardinality#cite_note-:2-32)[[33]](https://en.wikipedia.org/wiki/Cardinality#cite_note-:8-33)

[Bernard Bolzano](https://en.wikipedia.org/wiki/Bernard_Bolzano "Bernard Bolzano")'s _[Paradoxes of the Infinite](https://en.wikipedia.org/wiki/Paradoxes\_of\_the\_Infinite "Paradoxes of the Infinite")_ (_Paradoxien des Unendlichen_, 1851) is often considered the first systematic attempt to introduce the concept of sets into [mathematical analysis](https://en.wikipedia.org/wiki/Mathematical_analysis "Mathematical analysis"). In this work, Bolzano defended the notion of [actual infinity](https://en.wikipedia.org/wiki/Actual_infinity "Actual infinity"), presented an early formulation of what would later be recognized as one-to-one correspondence between infinite sets. He discussed examples such as the pairing between the [intervals](https://en.wikipedia.org/wiki/Interval_(mathematics) "Interval (mathematics)")![Image 12: {\displaystyle [0,5]}](https://wikimedia.org/api/rest_v1/media/math/render/svg/466c74a63ec8b8422ccb96c6bc5140dca45b5593) and ![Image 13: {\displaystyle [0,12]}](https://wikimedia.org/api/rest_v1/media/math/render/svg/89aabd85b3e389da4856956942ab5320c9bc586f) by the relation ![Image 14: {\displaystyle 5y=12x,}](https://wikimedia.org/api/rest_v1/media/math/render/svg/c0edc5eccae88ae44cd27320bcaf0ea6ab0465b5) and revisited Galileo's paradox. However, he too resisted saying that these sets were, in that sense, the same size. While _Paradoxes of the Infinite_ anticipated several ideas central to later set theory, the work had little influence on contemporary mathematics, in part due to its [posthumous publication](https://en.wikipedia.org/wiki/Posthumous_publication "Posthumous publication") and limited circulation.[[34]](https://en.wikipedia.org/wiki/Cardinality#cite_note-:9-34)[[35]](https://en.wikipedia.org/wiki/Cardinality#cite_note-35)[[36]](https://en.wikipedia.org/wiki/Cardinality#cite_note-36)

[![Image 15: refer to caption](https://upload.wikimedia.org/wikipedia/commons/thumb/6/67/Georg_Cantor3.jpg/250px-Georg_Cantor3.jpg)](https://en.wikipedia.org/wiki/File:Georg_Cantor3.jpg)

[Georg Cantor](https://en.wikipedia.org/wiki/Georg_Cantor "Georg Cantor"), c. 1870

The concept of cardinality emerged nearly fully formed in the work of Georg Cantor during the 1870s and 1880s, in the context of [mathematical analysis](https://en.wikipedia.org/wiki/Mathematical_analysis "Mathematical analysis"). In a series of papers beginning with _[On a Property of the Collection of All Real Algebraic Numbers](https://en.wikipedia.org/wiki/Cantor%27s\_first\_set\_theory\_article "Cantor's first set theory article")_ (1874),[[37]](https://en.wikipedia.org/wiki/Cardinality#cite_note-37) Cantor introduced the idea of comparing the sizes of infinite sets, through the notion of one-to-one correspondence.[[38]](https://en.wikipedia.org/wiki/Cardinality#cite_note-38) He showed that the set of [real numbers](https://en.wikipedia.org/wiki/Real_numbers "Real numbers") was, in this sense, strictly larger than the set of natural numbers [using a nested intervals argument](https://en.wikipedia.org/wiki/Cantor%27s_first_set_theory_article#Second_theorem "Cantor's first set theory article").[[39]](https://en.wikipedia.org/wiki/Cardinality#cite_note-39) This result was later refined into the more widely known [diagonal argument](https://en.wikipedia.org/wiki/Cantor%27s_diagonal_argument "Cantor's diagonal argument") of 1891, published in _Über eine elementare Frage der Mannigfaltigkeitslehre,_[[40]](https://en.wikipedia.org/wiki/Cardinality#cite_note-40) where he also proved the more general result (now called [Cantor's Theorem](https://en.wikipedia.org/wiki/Cantor%27s_Theorem "Cantor's Theorem")) that the [power set](https://en.wikipedia.org/wiki/Power_set "Power set") of any set is strictly larger than the set itself.[[41]](https://en.wikipedia.org/wiki/Cardinality#cite_note-41)

Cantor introduced the notion [cardinal numbers](https://en.wikipedia.org/wiki/Cardinal_numbers "Cardinal numbers") in terms of [ordinal numbers](https://en.wikipedia.org/wiki/Ordinal_numbers "Ordinal numbers"). He viewed cardinal numbers as an abstraction of sets, introduced the notations, where, for a given set ![Image 16: {\textstyle M}](https://wikimedia.org/api/rest_v1/media/math/render/svg/913ace920108f7552777e36ac0b7ee3f5093a088), the [order type](https://en.wikipedia.org/wiki/Order_type "Order type") of that set was written ![Image 17: {\textstyle {\overline {M}}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/33132e88c5b38d8848c1e06547ca5cb180c9ff2e), and the cardinal number was ![Image 18: {\textstyle M}](https://wikimedia.org/api/rest_v1/media/math/render/svg/913ace920108f7552777e36ac0b7ee3f5093a088), a double abstraction.[[42]](https://en.wikipedia.org/wiki/Cardinality#cite_note-42) He also introduced the [Aleph sequence](https://en.wikipedia.org/wiki/Cardinality#Aleph_numbers) for infinite cardinal numbers. These notations appeared in correspondence and were formalized in his later writings, particularly the series _Beiträge zur Begründung der transfiniten Mengenlehre_ (1895–1897).[[43]](https://en.wikipedia.org/wiki/Cardinality#cite_note-43) In these works, Cantor developed an [arithmetic of cardinal numbers](https://en.wikipedia.org/wiki/Cardinal_arithmetic "Cardinal arithmetic"), defining addition, multiplication, and exponentiation of cardinal numbers based on set-theoretic constructions. This led to the formulation of the [Continuum Hypothesis](https://en.wikipedia.org/wiki/Continuum_Hypothesis "Continuum Hypothesis") (CH), the proposition that no set has cardinality strictly between the cardinality of the natural numbers ![Image 19: {\displaystyle \aleph _{0}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/721cd7f8c15a2e72ad162bdfa5baea8eef98aab1) and the [cardinality of the continuum](https://en.wikipedia.org/wiki/Cardinality_of_the_continuum "Cardinality of the continuum")![Image 20: {\displaystyle |\mathbb {R} |}](https://wikimedia.org/api/rest_v1/media/math/render/svg/a69d629bc7c9f8d0b5d6ea4cd6fad0cd4330ca0b), that is whether ![Image 21: {\displaystyle |\mathbb {R} |=\aleph _{1}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/948a68092d411cf2e7491f37f24fbd4c2348f458). Cantor was unable to resolve CH and left it as an [open problem](https://en.wikipedia.org/wiki/Open_problem "Open problem").[[44]](https://en.wikipedia.org/wiki/Cardinality#cite_note-44)

Parallel to Cantor’s development, [Richard Dedekind](https://en.wikipedia.org/wiki/Richard_Dedekind "Richard Dedekind") independently formulated many advanced theorems of set theory, and helped establish set-theoretic foundations of algebra and arithmetic.[[45]](https://en.wikipedia.org/wiki/Cardinality#cite_note-45) Dedekind’s _[Was sind und was sollen die Zahlen?](https://en.wikipedia.org/w/index.php?title=Was\_sind\_und\_was\_sollen\_die\_Zahlen%3F&action=edit&redlink=1 "Was sind und was sollen die Zahlen? (page does not exist)")_ (1888)[[46]](https://en.wikipedia.org/wiki/Cardinality#cite_note-46) emphasized structural properties over extensional definitions, and supported the bijective formulation of size and number. Dedekind was in correspondence with Cantor during the development of set theory; he supplied Cantor with a proof of the countability of the [algebraic numbers](https://en.wikipedia.org/wiki/Algebraic_numbers "Algebraic numbers"), and gave feedback and modifications on Cantor's proofs before publishing.[[47]](https://en.wikipedia.org/wiki/Cardinality#cite_note-47)[[34]](https://en.wikipedia.org/wiki/Cardinality#cite_note-:9-34)[[48]](https://en.wikipedia.org/wiki/Cardinality#cite_note-48)

After Cantor's 1883 proof that all finite-dimensional spaces ![Image 22: {\displaystyle (\mathbb {R} ^{n})}](https://wikimedia.org/api/rest_v1/media/math/render/svg/2f345c7fdd008f32d9470b7e68e726efe3fec855) have the same cardinality,[[49]](https://en.wikipedia.org/wiki/Cardinality#cite_note-49) in 1890, [Giuseppe Peano](https://en.wikipedia.org/wiki/Giuseppe_Peano "Giuseppe Peano") introduced the [Peano curve](https://en.wikipedia.org/wiki/Peano_curve "Peano curve"), which was a more visual proof that the [unit interval](https://en.wikipedia.org/wiki/Unit_interval "Unit interval")![Image 23: {\displaystyle [0,1]}](https://wikimedia.org/api/rest_v1/media/math/render/svg/738f7d23bb2d9642bab520020873cccbef49768d) has the same cardinality as the [unit square](https://en.wikipedia.org/wiki/Unit_square "Unit square") on ![Image 24: {\displaystyle \mathbb {R} ^{2}.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/066b155c535a38739cc0c4b288324cbb7a4a227a)[[50]](https://en.wikipedia.org/wiki/Cardinality#cite_note-50) This created a new area of mathematical analysis studying what is now called [space-filling curves](https://en.wikipedia.org/wiki/Space-filling_curves "Space-filling curves").[[51]](https://en.wikipedia.org/wiki/Cardinality#cite_note-51)

German logician [Gottlob Frege](https://en.wikipedia.org/wiki/Gottlob_Frege "Gottlob Frege") attempted to ground the concepts of number and arithmetic in logic using Cantor's theory of cardinality and [Hume's principle](https://en.wikipedia.org/wiki/Hume%27s_principle "Hume's principle") in _[Die Grundlagen der Arithmetik](https://en.wikipedia.org/wiki/Die\_Grundlagen\_der\_Arithmetik "Die Grundlagen der Arithmetik")_ (1884) and the subsequent _Grundgesetze der Arithmetik_ (1893, 1903).[[31]](https://en.wikipedia.org/wiki/Cardinality#cite_note-:1-31)[[32]](https://en.wikipedia.org/wiki/Cardinality#cite_note-:2-32)[[33]](https://en.wikipedia.org/wiki/Cardinality#cite_note-:8-33) Frege defined cardinal numbers as [equivalence classes](https://en.wikipedia.org/wiki/Equivalence_classes "Equivalence classes") of sets under equinumerosity. However, Frege's approach to set theory was later shown to be flawed. His approach was eventually reformalized by [Bertrand Russell](https://en.wikipedia.org/wiki/Bertrand_Russell_Peace_Foundation "Bertrand Russell Peace Foundation") and [Alfred Whitehead](https://en.wikipedia.org/wiki/Alfred_North_Whitehead "Alfred North Whitehead") in _[Principia Mathematica](https://en.wikipedia.org/wiki/Principia\_Mathematica "Principia Mathematica")_ (1910–1913, vol. II)[[52]](https://en.wikipedia.org/wiki/Cardinality#cite_note-FOOTNOTERussellWhitehead1973-52) using a [theory of types](https://en.wikipedia.org/wiki/Type_theory#History "Type theory").[[53]](https://en.wikipedia.org/wiki/Cardinality#cite_note-53) Though Russell initially had difficulties understanding Cantor's and Frege’s intuitions of cardinality.[[54]](https://en.wikipedia.org/wiki/Cardinality#cite_note-54)[[55]](https://en.wikipedia.org/wiki/Cardinality#cite_note-55) This definition of cardinal numbers is now referred to as the _Frege–Russell_ definition.[[56]](https://en.wikipedia.org/wiki/Cardinality#cite_note-56) This definition was eventually superseded by the convention established by [John von Neumann](https://en.wikipedia.org/wiki/John_von_Neumann "John von Neumann") in 1928 which uses representatives to define cardinal numbers.[[57]](https://en.wikipedia.org/wiki/Cardinality#cite_note-57)

At the [Paris](https://en.wikipedia.org/wiki/Paris "Paris") conference of the [International Congress of Mathematicians](https://en.wikipedia.org/wiki/International_Congress_of_Mathematicians "International Congress of Mathematicians") in 1900, [David Hilbert](https://en.wikipedia.org/wiki/David_Hilbert "David Hilbert"), one of the most influential mathematicians of the time, gave a speech wherein he presented ten unsolved problems (of a total of 23, later published, now called [_Hilbert's problems_](https://en.wikipedia.org/wiki/Hilbert%27s_problems "Hilbert's problems")). Of these, he placed "Cantor's problem" (now called the Continuum Hypothesis) as the first on the list. This list of problems would prove to be very influential in 20th century mathematics, and attracted a lot of attention from other mathematicians toward Cantor's theory of cardinality.[[58]](https://en.wikipedia.org/wiki/Cardinality#cite_note-58)[[34]](https://en.wikipedia.org/wiki/Cardinality#cite_note-:9-34)

### Axiomatic set theory

[[edit](https://en.wikipedia.org/w/index.php?title=Cardinality&action=edit&section=8 "Edit section: Axiomatic set theory")]

[![Image 25](https://upload.wikimedia.org/wikipedia/commons/thumb/e/ed/Ernst_Zermelo_1900s.jpg/250px-Ernst_Zermelo_1900s.jpg)](https://en.wikipedia.org/wiki/File:Ernst_Zermelo_1900s.jpg)

[![Image 26](https://upload.wikimedia.org/wikipedia/commons/thumb/4/42/Kurt_g%C3%B6del.jpg/171px-Kurt_g%C3%B6del.jpg)](https://en.wikipedia.org/wiki/File:Kurt_g%C3%B6del.jpg)

In 1908, [Ernst Zermelo](https://en.wikipedia.org/wiki/Ernst_Zermelo "Ernst Zermelo") proposed the first [axiomatization](https://en.wikipedia.org/wiki/Axiomatization "Axiomatization") of set theory, now called [Zermelo set theory](https://en.wikipedia.org/wiki/Zermelo_set_theory "Zermelo set theory"), primarily to support his earlier (1904) proof of the [Well-ordering theorem](https://en.wikipedia.org/wiki/Well-ordering_theorem "Well-ordering theorem"), which showed that all cardinal numbers could be represented as [Alephs](https://en.wikipedia.org/wiki/Cardinality#Aleph_numbers), though the proof required a controversial principle now known as the [Axiom of Choice](https://en.wikipedia.org/wiki/Axiom_of_Choice "Axiom of Choice") (AC).[[59]](https://en.wikipedia.org/wiki/Cardinality#cite_note-59) Zermelo's system would later be extended by [Abraham Fraenkel](https://en.wikipedia.org/wiki/Abraham_Fraenkel "Abraham Fraenkel") and [Thoralf Skolem](https://en.wikipedia.org/wiki/Thoralf_Skolem "Thoralf Skolem") in the 1920s to create the standard foundation of set theory, called [Zermelo–Fraenkel set theory](https://en.wikipedia.org/wiki/Zermelo%E2%80%93Fraenkel_set_theory "Zermelo–Fraenkel set theory") (ZFC, "C" for the Axiom of Choice). ZFC provided a rigorous foundation through which infinite cardinals could be systematically studied while avoiding the [paradoxes of naive set theory](https://en.wikipedia.org/wiki/Paradoxes_of_set_theory "Paradoxes of set theory").[[34]](https://en.wikipedia.org/wiki/Cardinality#cite_note-:9-34)[[60]](https://en.wikipedia.org/wiki/Cardinality#cite_note-60)

In 1940, [Kurt Gödel](https://en.wikipedia.org/wiki/Kurt_G%C3%B6del "Kurt Gödel") showed that CH cannot be _disproved_ from the axioms of ZFC. Gödel's proof shows that both CH and AC hold in his [constructible universe](https://en.wikipedia.org/wiki/Constructible_universe "Constructible universe"): an [inner model](https://en.wikipedia.org/wiki/Inner_model "Inner model") of ZFC where CH holds. The existence of an inner model of ZFC in which additional axioms hold shows that the additional axioms are (relatively) [consistent](https://en.wikipedia.org/wiki/Consistent "Consistent") with ZFC.[[61]](https://en.wikipedia.org/wiki/Cardinality#cite_note-61) In 1963, [Paul Cohen](https://en.wikipedia.org/wiki/Paul_Cohen "Paul Cohen") showed that CH cannot be _proven_ from the ZFC axioms, which showed that CH was [independent](https://en.wikipedia.org/wiki/Independence_(mathematical_logic) "Independence (mathematical logic)") from ZFC. To prove his result, Cohen developed the method of [forcing](https://en.wikipedia.org/wiki/Forcing_(mathematics) "Forcing (mathematics)"), which has become a standard tool in set theory. Essentially, this method begins with Gödel's model of ZFC in which CH holds and constructs another model which contains more sets than the original in a way that CH does not hold. Cohen was awarded the [Fields Medal](https://en.wikipedia.org/wiki/Fields_Medal "Fields Medal") in 1966 for his proof.[[62]](https://en.wikipedia.org/wiki/Cardinality#cite_note-62)[[63]](https://en.wikipedia.org/wiki/Cardinality#cite_note-63)

[![Image 27](https://upload.wikimedia.org/wikipedia/commons/thumb/f/ff/Aplicaci%C3%B3n_2_inyectiva_sobreyectiva04.svg/250px-Aplicaci%C3%B3n_2_inyectiva_sobreyectiva04.svg.png)](https://en.wikipedia.org/wiki/File:Aplicaci%C3%B3n_2_inyectiva_sobreyectiva04.svg)

A one-to-one correspondence from **N**, the set of all non-negative integers, to the set **E** of non-negative [even numbers](https://en.wikipedia.org/wiki/Even_number "Even number"). Although **E** is a proper subset of **N**, both sets have the same cardinality.

The basic notions of [sets](https://en.wikipedia.org/wiki/Set_(mathematics) "Set (mathematics)") and [functions](https://en.wikipedia.org/wiki/Function_(mathematics) "Function (mathematics)") are used to develop the concept of cardinality, and technical terms therein are used throughout this article. A [set](https://en.wikipedia.org/wiki/Set_(mathematics) "Set (mathematics)") can be understood as any collection of objects, usually represented with [curly braces](https://en.wikipedia.org/wiki/Curly_braces "Curly braces"). The symbol ![Image 30: {\displaystyle \in }](https://wikimedia.org/api/rest_v1/media/math/render/svg/6fe4d5b0a594c1da89b5e78e7dfbeed90bdcc32f) represents set membership, for example ![Image 31: {\displaystyle 1\in S}](https://wikimedia.org/api/rest_v1/media/math/render/svg/fc2e3bdffbb16f850a092243fdd03d484174ed37) says "1 is a member of the set ![Image 32: {\displaystyle S}](https://wikimedia.org/api/rest_v1/media/math/render/svg/4611d85173cd3b508e67077d4a1252c9c05abca2)" which is true by the definition of ![Image 33: {\displaystyle S}](https://wikimedia.org/api/rest_v1/media/math/render/svg/4611d85173cd3b508e67077d4a1252c9c05abca2) above.

A [function](https://en.wikipedia.org/wiki/Function_(mathematics) "Function (mathematics)") is an association that maps elements of one set to the elements of another, often represented with an arrow diagram. For example, the adjacent image depicts a function which maps the set of [natural numbers](https://en.wikipedia.org/wiki/Natural_numbers "Natural numbers") to the set of [even numbers](https://en.wikipedia.org/wiki/Even_numbers "Even numbers") by multiplying by 2. If a function does not map two elements to the same place, it is called [injective](https://en.wikipedia.org/wiki/Injective "Injective"). If a function covers every element in the output space, it is called [surjective](https://en.wikipedia.org/wiki/Surjective "Surjective"). If a function is both injective and surjective, it is called [bijective](https://en.wikipedia.org/wiki/Bijective "Bijective"). (For further clarification, see _[Bijection, injection and surjection](https://en.wikipedia.org/wiki/Bijection,\_injection\_and\_surjection "Bijection, injection and surjection")_.)

The intuitive property of two sets having the "same size" is that their objects can be paired one-to-one. A one-to-one pairing between two sets defines a bijective function between them by mapping each object to its pair. Similarly, a bijection between two sets defines a pairing of their elements by pairing each object with the one it maps to. A fundamental result necessary in developing a theory of cardinality is relating it to an [equivalence relation](https://en.wikipedia.org/wiki/Equivalence_relation "Equivalence relation"). A binary [relation](https://en.wikipedia.org/wiki/Relation_(mathematics) "Relation (mathematics)") is an equivalence relation if it satisfies the three basic properties of equality: [reflexivity](https://en.wikipedia.org/wiki/Reflexive_relation "Reflexive relation"), [symmetry](https://en.wikipedia.org/wiki/Symmetric_relation "Symmetric relation"), and [transitivity](https://en.wikipedia.org/wiki/Transitive_relation "Transitive relation").[[82]](https://en.wikipedia.org/wiki/Cardinality#cite_note-:10-82)

Since equinumerosity satisfies these three properties, it forms an equivalence relation. This means that cardinality, in some sense, [partitions sets](https://en.wikipedia.org/wiki/Partition_of_a_set "Partition of a set") into [equivalence classes](https://en.wikipedia.org/wiki/Equivalence_classes "Equivalence classes"), and one may assign a representative to denote this class. This motivates the notion of a [cardinal number](https://en.wikipedia.org/wiki/Cardinality#Cardinal_numbers).[[83]](https://en.wikipedia.org/wiki/Cardinality#cite_note-83) Somewhat more formally, a relation must be a certain set of [ordered pairs](https://en.wikipedia.org/wiki/Ordered_pairs "Ordered pairs"). Since there is no [set of all sets](https://en.wikipedia.org/wiki/Set_of_all_sets "Set of all sets") in standard set theory, equinumerosity is not a relation in the usual sense, but a [predicate](https://en.wikipedia.org/wiki/Predicate_(logic) "Predicate (logic)") or a relation over [classes](https://en.wikipedia.org/wiki/Class_(set_theory) "Class (set theory)").

If ![Image 77: {\displaystyle A\preceq B,}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7bae8c40eb134bf6daeadb7aa0c90b4efe79a7ac) but there is no injection from ![Image 78: {\displaystyle B}](https://wikimedia.org/api/rest_v1/media/math/render/svg/47136aad860d145f75f3eed3022df827cee94d7a) to ![Image 79: {\displaystyle A,}](https://wikimedia.org/api/rest_v1/media/math/render/svg/2746026864cc5896e3e52443a1c917be2df9d8ea) then ![Image 80: {\displaystyle A}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7daff47fa58cdfd29dc333def748ff5fa4c923e3) is said to be _strictly_ smaller than ![Image 81: {\displaystyle B,}](https://wikimedia.org/api/rest_v1/media/math/render/svg/075d661417b8ca5a991a2a7bd4991cc1ab856d9d) or be _dom_ written without the underline as ![Image 82: {\displaystyle A\prec B}](https://wikimedia.org/api/rest_v1/media/math/render/svg/d048e2bd00066ee5397ed7c98e7c8776b2ab354c) or ![Image 83: {\displaystyle |A|<|B|.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/3f83b6dcae644912db51952ed8fefc1a8777033b)[[93]](https://en.wikipedia.org/wiki/Cardinality#cite_note-93) For example, if ![Image 84: {\displaystyle A}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7daff47fa58cdfd29dc333def748ff5fa4c923e3) has four elements and ![Image 85: {\displaystyle B}](https://wikimedia.org/api/rest_v1/media/math/render/svg/47136aad860d145f75f3eed3022df827cee94d7a) has five, then the following are true ![Image 86: {\displaystyle A\preceq A,}](https://wikimedia.org/api/rest_v1/media/math/render/svg/c8a782a20ee726af955426dcf3b661f4f3c3fd80)![Image 87: {\displaystyle A\preceq B,}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7bae8c40eb134bf6daeadb7aa0c90b4efe79a7ac) and ![Image 88: {\displaystyle A\prec B.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/18f522d651e993896484206350519faec198255e)

The basic properties of an inequality are reflexivity (for any ![Image 89: {\displaystyle a,}](https://wikimedia.org/api/rest_v1/media/math/render/svg/f059f053fcf9f421b7c74362cf3bd5ed024e19d1)![Image 90: {\displaystyle a\leq a}](https://wikimedia.org/api/rest_v1/media/math/render/svg/5f33338c098fb2384f5c7f4615729d0e0f8d744a)), transitivity (if ![Image 91: {\displaystyle a\leq b}](https://wikimedia.org/api/rest_v1/media/math/render/svg/41558abc50886fdf38817495b243958d7b3dd39b) and ![Image 92: {\displaystyle b\leq c,}](https://wikimedia.org/api/rest_v1/media/math/render/svg/28ec3ad88e4e788783f6e80afc4b05f652cfc37d) then ![Image 93: {\displaystyle a\leq c}](https://wikimedia.org/api/rest_v1/media/math/render/svg/fb1c962997d8a303e076777cd6d6bc732f360ac8)) and [antisymmetry](https://en.wikipedia.org/wiki/Antisymmetric_relation "Antisymmetric relation") (if ![Image 94: {\displaystyle a\leq b}](https://wikimedia.org/api/rest_v1/media/math/render/svg/41558abc50886fdf38817495b243958d7b3dd39b) and ![Image 95: {\displaystyle b\leq a,}](https://wikimedia.org/api/rest_v1/media/math/render/svg/e05af0ccc5a6e91bf711507150765f14a209b592) then ![Image 96: {\displaystyle a=b}](https://wikimedia.org/api/rest_v1/media/math/render/svg/1956b03d1314c7071ac1f45ed7b1e29422dcfcc4)) (See [_Inequality § Formal definitions_](https://en.wikipedia.org/wiki/Inequality_(mathematics)#Formal_definitions_and_generalizations "Inequality (mathematics)")). Cardinal inequality ![Image 97: {\displaystyle (\preceq )}](https://wikimedia.org/api/rest_v1/media/math/render/svg/31e7fa28d3a55d818bf10cecfdf83d2e68aec2d7) as defined above is reflexive since the [identity function](https://en.wikipedia.org/wiki/Identity_function "Identity function") is injective, and is transitive by function composition.[[94]](https://en.wikipedia.org/wiki/Cardinality#cite_note-94) Antisymmetry is established by the [Schröder–Bernstein theorem](https://en.wikipedia.org/wiki/Schr%C3%B6der%E2%80%93Bernstein_theorem "Schröder–Bernstein theorem"). The proof roughly goes as follows.[[95]](https://en.wikipedia.org/wiki/Cardinality#cite_note-:6-95)

Given sets ![Image 98: {\displaystyle A}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7daff47fa58cdfd29dc333def748ff5fa4c923e3) and ![Image 99: {\displaystyle B}](https://wikimedia.org/api/rest_v1/media/math/render/svg/47136aad860d145f75f3eed3022df827cee94d7a), where ![Image 100: {\displaystyle f:A\mapsto B}](https://wikimedia.org/api/rest_v1/media/math/render/svg/95138b7a1c4e935668228a44c617bfe09e7cce74) is the function that proves ![Image 101: {\displaystyle A\preceq B}](https://wikimedia.org/api/rest_v1/media/math/render/svg/bd084bf1b0042f6422a02013f8051a9d2eaee18b) and ![Image 102: {\displaystyle g:B\mapsto A}](https://wikimedia.org/api/rest_v1/media/math/render/svg/05b40fa1e75d7c920005d4b910176377be4fabc2) proves ![Image 103: {\displaystyle B\preceq A}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7630f81ffd3df39067ccda5c1c69dba497043a8e), then consider the sequence of points given by applying ![Image 104: {\displaystyle f}](https://wikimedia.org/api/rest_v1/media/math/render/svg/132e57acb643253e7810ee9702d9581f159a1c61) then ![Image 105: {\displaystyle g}](https://wikimedia.org/api/rest_v1/media/math/render/svg/d3556280e66fe2c0d0140df20935a6f057381d77) to each element over and over. Then we can define a bijection ![Image 106: {\displaystyle h:A\mapsto B}](https://wikimedia.org/api/rest_v1/media/math/render/svg/63719b1b0327d1bf49ef2db454ef9382733b2e99) as follows: If a sequence forms a cycle, begins with an element ![Image 107: {\displaystyle a\in A}](https://wikimedia.org/api/rest_v1/media/math/render/svg/a97387981adb5d65f74518e20b6785a284d7abd5) not mapped to by ![Image 108: {\displaystyle g}](https://wikimedia.org/api/rest_v1/media/math/render/svg/d3556280e66fe2c0d0140df20935a6f057381d77), or extends infinitely in both directions, define ![Image 109: {\displaystyle h(x)=f(x)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/1f7ee450fe0fd3b8f6002fc97875396759f932df) for each ![Image 110: {\displaystyle x\in A}](https://wikimedia.org/api/rest_v1/media/math/render/svg/27bcc9b2afb295d4234bc294860cd0c63bcad2ca) in those sequences. The last case, if a sequence begins with an element ![Image 111: {\displaystyle b\in B}](https://wikimedia.org/api/rest_v1/media/math/render/svg/61dbfba9ff608c8700a30596649d98dcc6147d86), not mapped to by ![Image 112: {\displaystyle f}](https://wikimedia.org/api/rest_v1/media/math/render/svg/132e57acb643253e7810ee9702d9581f159a1c61), define ![Image 113: {\displaystyle h(x)=g^{-1}(x)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/cb7b4dec703dc658787fcd868121772274cdc7f0) for each ![Image 114: {\displaystyle x\in A}](https://wikimedia.org/api/rest_v1/media/math/render/svg/27bcc9b2afb295d4234bc294860cd0c63bcad2ca) in that sequence. Then ![Image 115: {\displaystyle h}](https://wikimedia.org/api/rest_v1/media/math/render/svg/b26be3e694314bc90c3215047e4a2010c6ee184a) is a bijection.[[95]](https://en.wikipedia.org/wiki/Cardinality#cite_note-:6-95)

The above shows that cardinal inequality is a [partial order](https://en.wikipedia.org/wiki/Partial_order "Partial order"). A [total order](https://en.wikipedia.org/wiki/Total_order "Total order") has the additional property that, for any ![Image 116: {\displaystyle a}](https://wikimedia.org/api/rest_v1/media/math/render/svg/ffd2487510aa438433a2579450ab2b3d557e5edc) and ![Image 117: {\displaystyle b}](https://wikimedia.org/api/rest_v1/media/math/render/svg/f11423fbb2e967f986e36804a8ae4271734917c3), either ![Image 118: {\displaystyle a\leq b}](https://wikimedia.org/api/rest_v1/media/math/render/svg/41558abc50886fdf38817495b243958d7b3dd39b) or ![Image 119: {\displaystyle b\leq a.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/88d1f6178413e25517396277e7a8385468eb637f) This can be established by the [well-ordering theorem](https://en.wikipedia.org/wiki/Well-ordering_theorem "Well-ordering theorem"). Every well-ordered set is [isomorphic](https://en.wikipedia.org/wiki/Order_isomorphic "Order isomorphic") to a unique [ordinal number](https://en.wikipedia.org/wiki/Ordinal_number "Ordinal number"), called the [order type](https://en.wikipedia.org/wiki/Order_type "Order type") of the set. Then, by comparing their order types, one can show that ![Image 120: {\displaystyle A\preceq B}](https://wikimedia.org/api/rest_v1/media/math/render/svg/bd084bf1b0042f6422a02013f8051a9d2eaee18b) or ![Image 121: {\displaystyle B\preceq A}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7630f81ffd3df39067ccda5c1c69dba497043a8e). This result is equivalent to the [axiom of choice](https://en.wikipedia.org/wiki/Axiom_of_choice "Axiom of choice").[[96]](https://en.wikipedia.org/wiki/Cardinality#cite_note-96)[[97]](https://en.wikipedia.org/wiki/Cardinality#cite_note-97)[[98]](https://en.wikipedia.org/wiki/Cardinality#cite_note-98)

A set is called _[countable](https://en.wikipedia.org/wiki/Countable "Countable")_ if it is [finite](https://en.wikipedia.org/wiki/Finite_set "Finite set") or has a bijection with the set of [natural numbers](https://en.wikipedia.org/wiki/Natural_number "Natural number")![Image 122: {\displaystyle (\mathbb {N} ),}](https://wikimedia.org/api/rest_v1/media/math/render/svg/bea55fe9f7155b2404c5e56b357334e6c303740b) in which case it is called _countably infinite_. The term _[denumerable](https://en.wikipedia.org/wiki/Denumerable "Denumerable")_ is also sometimes used for countably infinite sets.[[99]](https://en.wikipedia.org/wiki/Cardinality#cite_note-99) For example, the set of all even natural numbers is countable, and therefore has the same cardinality as the whole set of natural numbers, even though it is a [proper subset](https://en.wikipedia.org/wiki/Proper_subset "Proper subset"). Similarly, the set of [square numbers](https://en.wikipedia.org/wiki/Square_numbers "Square numbers") is countable, which was considered paradoxical for hundreds of years before modern set theory (cf. _[§Pre-Cantorian set theory](https://en.wikipedia.org/wiki/Cardinality#Pre-Cantorian\_set\_theory)_). However, several other examples have historically been considered surprising or initially unintuitive since the rise of set theory.[[100]](https://en.wikipedia.org/wiki/Cardinality#cite_note-100)

[![Image 124](https://upload.wikimedia.org/wikipedia/commons/thumb/b/b1/Straight_counter-clockwise_spiral_path_in_square_grid.png/250px-Straight_counter-clockwise_spiral_path_in_square_grid.png)](https://en.wikipedia.org/wiki/File:Straight_counter-clockwise_spiral_path_in_square_grid.png)

The [rational numbers](https://en.wikipedia.org/wiki/Rational_numbers "Rational numbers")![Image 125: {\displaystyle (\mathbb {Q} )}](https://wikimedia.org/api/rest_v1/media/math/render/svg/a721495ab8dcae6b57e2b3e33402f94d57f7b9b6) are those which can be expressed as the [quotient](https://en.wikipedia.org/wiki/Quotient "Quotient") or [fraction](https://en.wikipedia.org/wiki/Fraction_(mathematics) "Fraction (mathematics)")⁠![Image 126: {\displaystyle {\tfrac {p}{q}}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/4b38d2684323653daafdd152b7e988594003897d)⁠ of two [integers](https://en.wikipedia.org/wiki/Integer "Integer"). The rational numbers can be shown to be countable by considering the set of fractions as the set of all [ordered pairs](https://en.wikipedia.org/wiki/Ordered_pairs "Ordered pairs") of integers, denoted ![Image 127: {\displaystyle \mathbb {Z} \times \mathbb {Z} ,}](https://wikimedia.org/api/rest_v1/media/math/render/svg/5fdf4c841a073390b8b1bddf7ae8e3494fed67fc) which can be visualized as the set of all [integer points](https://en.wikipedia.org/wiki/Integer_lattice "Integer lattice") on a grid. Then, an intuitive function can be described by drawing a line in a repeating pattern, or spiral, which eventually goes through each point in the grid. For example, going through each diagonal on the grid for positive fractions, or through a lattice spiral for all integer pairs.[[101]](https://en.wikipedia.org/wiki/Cardinality#cite_note-101) These technically over cover the rationals, since, for example, the rational number ![Image 128: {\textstyle {\frac {1}{2}}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/990274ef9b8937f955b175041b7cd0d5a2d482ec) gets mapped to by all the fractions ![Image 129: {\textstyle {\frac {2}{4}},\,{\frac {3}{6}},\,{\frac {4}{8}},\,\dots ,}](https://wikimedia.org/api/rest_v1/media/math/render/svg/2c80dda73e7a9e4473f8fbff82e513f8bda80bd6) as the grid method treats these all as distinct ordered pairs. So this function shows ![Image 130: {\displaystyle |\mathbb {Q} |\leq |\mathbb {N} |}](https://wikimedia.org/api/rest_v1/media/math/render/svg/80b91c99af85ac211d9083074fdac5235d09b727) not ![Image 131: {\displaystyle |\mathbb {Q} |=|\mathbb {N} |.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/0c881b122cebe7957d81d2553ca302a82859d0bb) This can be corrected by "skipping over" these numbers in the grid, using the Schröder–Bernstein theorem, or by designing a function which does this naturally, for example using the [Calkin–Wilf tree](https://en.wikipedia.org/wiki/Calkin%E2%80%93Wilf_tree "Calkin–Wilf tree").[[102]](https://en.wikipedia.org/wiki/Cardinality#cite_note-102)

[![Image 132](https://upload.wikimedia.org/wikipedia/commons/thumb/d/d1/Algebraicszoom.png/330px-Algebraicszoom.png)](https://en.wikipedia.org/wiki/File:Algebraicszoom.png)

Algebraic numbers on the [complex plane](https://en.wikipedia.org/wiki/Complex_plane "Complex plane"), colored by degree

A number is called [algebraic](https://en.wikipedia.org/wiki/Algebraic_number "Algebraic number") if it is a solution of some [polynomial](https://en.wikipedia.org/wiki/Polynomial "Polynomial") equation (with integer [coefficients](https://en.wikipedia.org/wiki/Coefficient "Coefficient")). For example, the [square root of two](https://en.wikipedia.org/wiki/Square_root_of_two "Square root of two")![Image 133: {\displaystyle {\sqrt {2}}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/b4afc1e27d418021bf10898eb44a7f5f315735ff) is a solution to ![Image 134: {\displaystyle x^{2}-2=0,}](https://wikimedia.org/api/rest_v1/media/math/render/svg/af07a621d005ee8fabacdc8df5966d49828e14d0) and the rational number ![Image 135: {\displaystyle p/q}](https://wikimedia.org/api/rest_v1/media/math/render/svg/8fa5bd4cf049744deac0ac4a04c07998bd6befa9) is the solution to ![Image 136: {\displaystyle qx-p=0.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/70354ae6a1d367d02ee48c44b1a6f95d37d9daaa) Conversely, a number which cannot be the root of any polynomial is called [transcendental](https://en.wikipedia.org/wiki/Transcendental_number "Transcendental number"). Two examples include [Euler's number](https://en.wikipedia.org/wiki/Euler%27s_number "Euler's number") (_e_) and [pi (π)](https://en.wikipedia.org/wiki/Pi "Pi"). In general, proving a number is transcendental is considered to be very difficult, and only a few classes of transcendental numbers are known. However, it can be shown that the set of algebraic numbers is countable by ordering the polynomials [lexicographically](https://en.wikipedia.org/wiki/Lexicographically "Lexicographically") (for example, see _[Cantor's first set theory article §The proofs](https://en.wikipedia.org/wiki/Cantor%27s\_first\_set\_theory\_article#The\_proofs "Cantor's first set theory article")_). Since the set of algebraic numbers is countable while the real numbers are uncountable (shown in the following section), the transcendental numbers must form the vast majority of real numbers, even though they are individually much harder to identify. That is to say, [almost all](https://en.wikipedia.org/wiki/Almost_all "Almost all") real numbers are transcendental.[[103]](https://en.wikipedia.org/wiki/Cardinality#cite_note-103)

[![Image 137](https://upload.wikimedia.org/wikipedia/commons/thumb/0/07/Hilbert%27s_Hotel.png/330px-Hilbert%27s_Hotel.png)](https://en.wikipedia.org/wiki/File:Hilbert%27s_Hotel.png)

Visual representation of [Hilbert's hotel](https://en.wikipedia.org/wiki/Hilbert%27s_hotel "Hilbert's hotel"). Each guest goes to the room with a number that is double their room number, leaving the odd-numbered rooms vacant. 

[Hilbert's paradox of the Grand Hotel](https://en.wikipedia.org/wiki/Hilbert%27s_paradox_of_the_Grand_Hotel "Hilbert's paradox of the Grand Hotel") is a popular [thought experiment](https://en.wikipedia.org/wiki/Thought_experiment "Thought experiment") devised by the German mathematician [David Hilbert](https://en.wikipedia.org/wiki/David_Hilbert "David Hilbert") to illustrate a counterintuitive property of countably infinite sets, allowing them to have the same cardinality as a [proper subset](https://en.wikipedia.org/wiki/Proper_subset "Proper subset") of themselves. The scenario begins by imagining a hotel with an infinite number of rooms, one for each natural number, all of which are occupied. But then a new guest walks in asking for a room. The hotel accommodates by moving the occupant of room 1 to room 2, the occupant of room 2 to room 3, room 3 to room 4, and in general, room n to room n+1. Then every guest still has a room, but room 1 is open for the new guest.[[104]](https://en.wikipedia.org/wiki/Cardinality#cite_note-:5-104)[[105]](https://en.wikipedia.org/wiki/Cardinality#cite_note-:11-105)

Then, the scenario continues by imagining an infinitely long bus of new guests seeking a room. The hotel accommodates by moving the person in room 1 to room 2, room 2 to room 4, and in general, room n to room 2n. Thus, all the even-numbered rooms are occupied, but all the odd-numbered rooms are vacant, leaving room for the infinite bus of new guests. The scenario continues further by assuming an infinite number of these infinite buses arrive at the hotel, and showing that the hotel is still able to accommodate. Finally, an infinite bus which has a seat for every [real number](https://en.wikipedia.org/wiki/Real_number "Real number") arrives, and the hotel is no longer able to accommodate.[[104]](https://en.wikipedia.org/wiki/Cardinality#cite_note-:5-104)[[105]](https://en.wikipedia.org/wiki/Cardinality#cite_note-:11-105)

Further information: [§Cardinality of the continuum](https://en.wikipedia.org/wiki/Cardinality#Cardinality_of_the_continuum)

A set is called _[uncountable](https://en.wikipedia.org/wiki/Uncountable "Uncountable")_ if it is not countable; that is, it is infinite and strictly larger than the set of natural numbers. The usual first example of this is the set of [real numbers](https://en.wikipedia.org/wiki/Real_numbers "Real numbers")![Image 138: {\displaystyle (\mathbb {R} )}](https://wikimedia.org/api/rest_v1/media/math/render/svg/66964ab8a4912f2be91dc22cceb762be1d3d8d05), which can be understood as the set of all numbers on the [number line](https://en.wikipedia.org/wiki/Number_line "Number line"). One method of proving that the reals are uncountable is called [Cantor's diagonal argument](https://en.wikipedia.org/wiki/Cantor%27s_diagonal_argument "Cantor's diagonal argument"), credited to Cantor for his 1891 proof,[[106]](https://en.wikipedia.org/wiki/Cardinality#cite_note-Cantor.1891-106) though his method differs from the more common presentation.[[107]](https://en.wikipedia.org/wiki/Cardinality#cite_note-107)

> ![Image 139: {\displaystyle {\begin{aligned}1\to &\;0.{\color {red}{\textbf {6}}}18033...\\2\to &\;0.1{\color {red}{\textbf {2}}}3456...\\3\to &\;0.60{\color {red}{\textbf {7}}}927...\\4\to &\;0.222{\color {red}{\textbf {2}}}22...\\\vdots \\&\;0.{\color {red}{\textbf {2323}}22}...\end{aligned}}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/6e2801b7e00d8340b114666f446f7b663c11c504)

It begins by assuming, [by contradiction](https://en.wikipedia.org/wiki/Proof_by_contradiction "Proof by contradiction"), that there is some one-to-one mapping between the natural numbers and the set of real numbers between 0 and 1 (the interval ![Image 140: {\displaystyle [0,1]}](https://wikimedia.org/api/rest_v1/media/math/render/svg/738f7d23bb2d9642bab520020873cccbef49768d)). Then, take the [decimal representation](https://en.wikipedia.org/wiki/Decimal_representation "Decimal representation") of each real number, for example, ![Image 141: {\displaystyle 0.5772...}](https://wikimedia.org/api/rest_v1/media/math/render/svg/0fc53f42b1dad4e21d55e5799c2b85dcbb7a8caf) with a leading zero followed by any sequence of digits. The number 1 is included in this set since 1 = [0.999...](https://en.wikipedia.org/wiki/0.999... "0.999...") Considering these real numbers in a column, it is always possible to create a new number such that the first digit of the new number is different from that of the first number in the column, the second digit is different from the second number in the column, and so on. The new number must also have a unique decimal representation, that is, it cannot end in [repeating nines](https://en.wikipedia.org/wiki/0.999... "0.999...") or repeating zeros. For example, if the digit isn't 2, make the digit of the new number 2, and if it was 2, make it 3.[[108]](https://en.wikipedia.org/wiki/Cardinality#cite_note-108)[[109]](https://en.wikipedia.org/wiki/Cardinality#cite_note-109) Then, this new number will be different from each of the numbers in the list by at least one digit, and therefore must not be in the list. This shows that the real numbers cannot be put into a one-to-one correspondence with the naturals, and thus must be strictly larger.[[110]](https://en.wikipedia.org/wiki/Cardinality#cite_note-110)[[111]](https://en.wikipedia.org/wiki/Cardinality#cite_note-111)

[![Image 142](https://upload.wikimedia.org/wikipedia/commons/thumb/c/c7/Diagonal_argument_powerset_svg.svg/255px-Diagonal_argument_powerset_svg.svg.png)](https://en.wikipedia.org/wiki/File:Diagonal_argument_powerset_svg.svg)

![Image 143: {\displaystyle \mathbb {N} }](https://wikimedia.org/api/rest_v1/media/math/render/svg/fdf9a96b565ea202d0f4322e9195613fb26a9bed) does not have the same cardinality as its [power set](https://en.wikipedia.org/wiki/Power_set "Power set")![Image 144: {\displaystyle {\mathcal {P}}(\mathbb {N} )}](https://wikimedia.org/api/rest_v1/media/math/render/svg/37b896f76c785d477a86be5965fbdf3814947022): For every function _f_ from **![Image 145: {\displaystyle \mathbb {N} }](https://wikimedia.org/api/rest_v1/media/math/render/svg/fdf9a96b565ea202d0f4322e9195613fb26a9bed)** to ![Image 146: {\displaystyle {\mathcal {P}}(\mathbb {N} )}](https://wikimedia.org/api/rest_v1/media/math/render/svg/37b896f76c785d477a86be5965fbdf3814947022), the set ![Image 147: {\displaystyle T=\{n\in N:n\notin f(n)\}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/afdf995d36f045cefe2b372427c348c30e36c8a7) disagrees with every set in the [range](https://en.wikipedia.org/wiki/Range_of_a_function "Range of a function") of ![Image 148: {\displaystyle f}](https://wikimedia.org/api/rest_v1/media/math/render/svg/132e57acb643253e7810ee9702d9581f159a1c61), hence ![Image 149: {\displaystyle f}](https://wikimedia.org/api/rest_v1/media/math/render/svg/132e57acb643253e7810ee9702d9581f159a1c61) cannot be surjective. The picture shows an example ![Image 150: {\displaystyle f}](https://wikimedia.org/api/rest_v1/media/math/render/svg/132e57acb643253e7810ee9702d9581f159a1c61) and the corresponding ![Image 151: {\displaystyle T}](https://wikimedia.org/api/rest_v1/media/math/render/svg/ec7200acd984a1d3a3d7dc455e262fbe54f7f6e0); **red**: ![Image 152: {\displaystyle n\notin T}](https://wikimedia.org/api/rest_v1/media/math/render/svg/0584ad9d03b6d07a84219784d0eb481e6fc40bba), **blue**: ![Image 153: {\displaystyle n\in T}](https://wikimedia.org/api/rest_v1/media/math/render/svg/061dad73f3707935d13ec5ec52fe843409fe0f3c).

Another classical example of an uncountable set, established using a related reasoning, is the [power set](https://en.wikipedia.org/wiki/Power_set "Power set") of the natural numbers, denoted ![Image 154: {\displaystyle {\mathcal {P}}(\mathbb {N} )}](https://wikimedia.org/api/rest_v1/media/math/render/svg/37b896f76c785d477a86be5965fbdf3814947022). This is the set of all [subsets](https://en.wikipedia.org/wiki/Subsets "Subsets") of ![Image 155: {\displaystyle \mathbb {N} }](https://wikimedia.org/api/rest_v1/media/math/render/svg/fdf9a96b565ea202d0f4322e9195613fb26a9bed), including the [empty set](https://en.wikipedia.org/wiki/Empty_set "Empty set") and ![Image 156: {\displaystyle \mathbb {N} }](https://wikimedia.org/api/rest_v1/media/math/render/svg/fdf9a96b565ea202d0f4322e9195613fb26a9bed) itself. The method is much closer to Cantor's original diagonal argument. Again, assume by contradiction that there exists a one-to-one correspondence ![Image 157: {\displaystyle f}](https://wikimedia.org/api/rest_v1/media/math/render/svg/132e57acb643253e7810ee9702d9581f159a1c61) between ![Image 158: {\displaystyle \mathbb {N} }](https://wikimedia.org/api/rest_v1/media/math/render/svg/fdf9a96b565ea202d0f4322e9195613fb26a9bed) and ![Image 159: {\displaystyle {\mathcal {P}}(\mathbb {N} )}](https://wikimedia.org/api/rest_v1/media/math/render/svg/37b896f76c785d477a86be5965fbdf3814947022), so that every subset of ![Image 160: {\displaystyle \mathbb {N} }](https://wikimedia.org/api/rest_v1/media/math/render/svg/fdf9a96b565ea202d0f4322e9195613fb26a9bed) is assigned to some natural number. These subsets are then placed in a column, in the order defined by ![Image 161: {\displaystyle f}](https://wikimedia.org/api/rest_v1/media/math/render/svg/132e57acb643253e7810ee9702d9581f159a1c61) (see image). Now, one may define a subset ![Image 162: {\displaystyle T}](https://wikimedia.org/api/rest_v1/media/math/render/svg/ec7200acd984a1d3a3d7dc455e262fbe54f7f6e0) of ![Image 163: {\displaystyle \mathbb {N} }](https://wikimedia.org/api/rest_v1/media/math/render/svg/fdf9a96b565ea202d0f4322e9195613fb26a9bed) which is not in the list by taking the negation of the "diagonal" of this column as follows:[[112]](https://en.wikipedia.org/wiki/Cardinality#cite_note-:12-112)

If ![Image 164: {\displaystyle 1\in f(1)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/30c3d929131f5e6d10be2bf80006d398a71b29cb), then ![Image 165: {\displaystyle 1\notin T}](https://wikimedia.org/api/rest_v1/media/math/render/svg/a3213029f72bbbc83d6a56a27c23bc59919f4d41), that is, if 1 is in the first subset of the list, then 1 is _not_ in the subset ![Image 166: {\displaystyle T}](https://wikimedia.org/api/rest_v1/media/math/render/svg/ec7200acd984a1d3a3d7dc455e262fbe54f7f6e0). Further, if ![Image 167: {\displaystyle 2\notin f(2)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/c76fba51da3cfda69109286860c769085a9a693b), then ![Image 168: {\displaystyle 2\in T}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7810390d204fb8a00bec89867d22349fcc84b032), that is if the number 2 is not in the second subset of the column, then 2 _is_ in the subset ![Image 169: {\displaystyle T}](https://wikimedia.org/api/rest_v1/media/math/render/svg/ec7200acd984a1d3a3d7dc455e262fbe54f7f6e0). Then in general, for each natural number ![Image 170: {\displaystyle n}](https://wikimedia.org/api/rest_v1/media/math/render/svg/a601995d55609f2d9f5e233e36fbe9ea26011b3b), ![Image 171: {\displaystyle n\in T}](https://wikimedia.org/api/rest_v1/media/math/render/svg/061dad73f3707935d13ec5ec52fe843409fe0f3c) if and only if ![Image 172: {\displaystyle n\notin f(n)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/f554df19290e7b177fb1cbfac5b1031ccfabf005), meaning ![Image 173: {\displaystyle n}](https://wikimedia.org/api/rest_v1/media/math/render/svg/a601995d55609f2d9f5e233e36fbe9ea26011b3b) is put in the subset ![Image 174: {\displaystyle T}](https://wikimedia.org/api/rest_v1/media/math/render/svg/ec7200acd984a1d3a3d7dc455e262fbe54f7f6e0) only if the nth subset in the column does not contain the number ![Image 175: {\displaystyle n}](https://wikimedia.org/api/rest_v1/media/math/render/svg/a601995d55609f2d9f5e233e36fbe9ea26011b3b). Then, for each natural number ![Image 176: {\displaystyle n}](https://wikimedia.org/api/rest_v1/media/math/render/svg/a601995d55609f2d9f5e233e36fbe9ea26011b3b), ![Image 177: {\displaystyle T\neq f(n)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/3238fc30cfe8b765d673a0952f47719dc8b4d08e), meaning, ![Image 178: {\displaystyle T}](https://wikimedia.org/api/rest_v1/media/math/render/svg/ec7200acd984a1d3a3d7dc455e262fbe54f7f6e0) is not the nth subset in the list, for any number ![Image 179: {\displaystyle n}](https://wikimedia.org/api/rest_v1/media/math/render/svg/a601995d55609f2d9f5e233e36fbe9ea26011b3b), and so it cannot appear anywhere in the list defined by ![Image 180: {\displaystyle f}](https://wikimedia.org/api/rest_v1/media/math/render/svg/132e57acb643253e7810ee9702d9581f159a1c61). Since ![Image 181: {\displaystyle f}](https://wikimedia.org/api/rest_v1/media/math/render/svg/132e57acb643253e7810ee9702d9581f159a1c61) was chosen arbitrarily, this shows that every function from ![Image 182: {\displaystyle \mathbb {N} }](https://wikimedia.org/api/rest_v1/media/math/render/svg/fdf9a96b565ea202d0f4322e9195613fb26a9bed) to ![Image 183: {\displaystyle {\mathcal {P}}(\mathbb {N} )}](https://wikimedia.org/api/rest_v1/media/math/render/svg/37b896f76c785d477a86be5965fbdf3814947022) must be missing at least one element, therefore no such bijection can exist, and so ![Image 184: {\displaystyle {\mathcal {P}}(\mathbb {N} )}](https://wikimedia.org/api/rest_v1/media/math/render/svg/37b896f76c785d477a86be5965fbdf3814947022) must be not be countable.[[112]](https://en.wikipedia.org/wiki/Cardinality#cite_note-:12-112)

These two sets, ![Image 185: {\displaystyle \mathbb {R} }](https://wikimedia.org/api/rest_v1/media/math/render/svg/786849c765da7a84dbc3cce43e96aad58a5868dc) and ![Image 186: {\displaystyle {\mathcal {P}}(\mathbb {N} )}](https://wikimedia.org/api/rest_v1/media/math/render/svg/37b896f76c785d477a86be5965fbdf3814947022) can be shown to have the same cardinality (by, for example, assigning each subset to a decimal expansion) called the [cardinality of the continuum](https://en.wikipedia.org/wiki/Cardinality#Cardinality_of_the_continuum).[[113]](https://en.wikipedia.org/wiki/Cardinality#cite_note-113) Whether there exists a set ![Image 187: {\displaystyle A}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7daff47fa58cdfd29dc333def748ff5fa4c923e3) with cardinality between these two sets ![Image 188: {\displaystyle |\mathbb {N} |<|A|<|\mathbb {R} |}](https://wikimedia.org/api/rest_v1/media/math/render/svg/49df964a808b88c294fa411344a04c363aa81368) is known as the [continuum hypothesis](https://en.wikipedia.org/wiki/Continuum_hypothesis "Continuum hypothesis").[[114]](https://en.wikipedia.org/wiki/Cardinality#cite_note-114)

[Cantor's theorem](https://en.wikipedia.org/wiki/Cantor%27s_theorem "Cantor's theorem") generalizes the second theorem above, showing that every set is strictly smaller than its powerset. The proof roughly goes as follows: Given a set ![Image 189: {\displaystyle A}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7daff47fa58cdfd29dc333def748ff5fa4c923e3), assume by contradiction that there is a bijection ![Image 190: {\displaystyle f}](https://wikimedia.org/api/rest_v1/media/math/render/svg/132e57acb643253e7810ee9702d9581f159a1c61) from ![Image 191: {\displaystyle A}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7daff47fa58cdfd29dc333def748ff5fa4c923e3) to ![Image 192: {\displaystyle {\mathcal {P}}(A)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/1757ec21abe0a22f8e91b51fe3e6ac4ea63a9122). Then, the subset ![Image 193: {\displaystyle T\subseteq A}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7ac1a05d70196815f1a4b6b8cd2ecf3f6459ef7d) given by taking the negation of the "diagonal", formally, ![Image 194: {\displaystyle T=\{a\in A:a\notin f(a)\}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/b63a08ea6fddf90dd7b0433144f41f00230b0aa9), cannot be in the list. Therefore, every function is missing at least one element, and so ![Image 195: {\displaystyle |A|<|{\mathcal {P}}(A)|}](https://wikimedia.org/api/rest_v1/media/math/render/svg/2b90834aae3801f48dcd69e220f2669907f84a48). Further, since ![Image 196: {\displaystyle {\mathcal {P}}(A)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/1757ec21abe0a22f8e91b51fe3e6ac4ea63a9122) is itself a set, the argument can be repeated to show ![Image 197: {\displaystyle |A|<|{\mathcal {P}}(A)|<|{\mathcal {P}}({\mathcal {P}}(A))|}](https://wikimedia.org/api/rest_v1/media/math/render/svg/9b7e0521fae0bc73a3fc831985c2b224ada9996d). Taking ![Image 198: {\displaystyle A=\mathbb {N} }](https://wikimedia.org/api/rest_v1/media/math/render/svg/493663f32689b436cf245289491762d8d710e156), this shows that ![Image 199: {\displaystyle {\mathcal {P}}({\mathcal {P}}(\mathbb {N} ))}](https://wikimedia.org/api/rest_v1/media/math/render/svg/c0d9ead47a83b0ab30b6b4e0ee39dc2cd3cce43e) is even larger than ![Image 200: {\displaystyle {\mathcal {P}}(\mathbb {N} )}](https://wikimedia.org/api/rest_v1/media/math/render/svg/37b896f76c785d477a86be5965fbdf3814947022), which was already shown to be uncountable. Repeating this argument shows that there are infinitely many "sizes" of infinity.[[115]](https://en.wikipedia.org/wiki/Cardinality#cite_note-115)

In the above sections, "the cardinality of a set" was described relationally. In other words, one set could be compared to another, intuitively comparing their "size". [Cardinal numbers](https://en.wikipedia.org/wiki/Cardinal_number "Cardinal number") are a means of measuring this "size" more explicitly. For finite sets, this is simply the [natural number](https://en.wikipedia.org/wiki/Natural_number "Natural number") found by counting the elements. This number is called the _cardinal number_ of that set, or simply _the cardinality_ of that set. The cardinal number of a set ![Image 201: {\displaystyle A}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7daff47fa58cdfd29dc333def748ff5fa4c923e3) is generally denoted by ![Image 202: {\displaystyle |A|,}](https://wikimedia.org/api/rest_v1/media/math/render/svg/4ad0bf27e7d0ee63510032b213fb898f262bd7cf) with a [vertical bar](https://en.wikipedia.org/wiki/Vertical_bar "Vertical bar") on each side,[[116]](https://en.wikipedia.org/wiki/Cardinality#cite_note-116) though it may also be denoted by ![Image 203: {\displaystyle A}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7daff47fa58cdfd29dc333def748ff5fa4c923e3), ![Image 204: {\displaystyle \operatorname {card} (A),}](https://wikimedia.org/api/rest_v1/media/math/render/svg/1c7516a98a18d4fd965a34db193fa1b5e5a6f389) or ![Image 205: {\displaystyle \#A.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/29203cc5fb3140ced8f48d52953319b94a4399d1)[[128]](https://en.wikipedia.org/wiki/Cardinality#cite_note-128)

For infinite sets, "cardinal number" is somewhat more difficult to define formally. Cardinal numbers are not usually thought of in terms of their formal definition, but immaterially in terms of their arithmetic/algebraic properties.[[129]](https://en.wikipedia.org/wiki/Cardinality#cite_note-129) The assumption that there is _some_[cardinal function](https://en.wikipedia.org/wiki/Cardinal_function "Cardinal function")![Image 206: {\displaystyle A\mapsto |A|}](https://wikimedia.org/api/rest_v1/media/math/render/svg/1a02041187e2bd1ba0ddb0c141fefa2955f5f3a0) which satisfies ![Image 207: {\displaystyle A\sim B\iff |A|=|B|}](https://wikimedia.org/api/rest_v1/media/math/render/svg/575300f168aa46dfe530c20d2c4add104b4c9c4b), sometimes called the _axiom of cardinality_[[130]](https://en.wikipedia.org/wiki/Cardinality#cite_note-130) or [_Hume's principle_](https://en.wikipedia.org/wiki/Hume%27s_principle "Hume's principle"),[[131]](https://en.wikipedia.org/wiki/Cardinality#cite_note-131) is sufficient for deriving most properties of cardinal numbers.[[132]](https://en.wikipedia.org/wiki/Cardinality#cite_note-132)

Commonly in mathematics, if a relation satisfies the properties of an [equivalence relation](https://en.wikipedia.org/wiki/Equivalence_relation "Equivalence relation"), the objects used to materialize this relation are [equivalence classes](https://en.wikipedia.org/wiki/Equivalence_classes "Equivalence classes"), which groups all the objects equivalent to one another. These called the _Frege–Russell_ cardinal numbers.[[133]](https://en.wikipedia.org/wiki/Cardinality#cite_note-133) However, this would mean that cardinal numbers are too large to form sets (apart from the cardinal number ![Image 208: {\displaystyle 0}](https://wikimedia.org/api/rest_v1/media/math/render/svg/2aae8864a3c1fec9585261791a809ddec1489950) whose only element is the [empty set](https://en.wikipedia.org/wiki/Empty_set "Empty set")), since, for example, the cardinal number ![Image 209: {\displaystyle 1}](https://wikimedia.org/api/rest_v1/media/math/render/svg/92d98b82a3778f043108d4e20960a9193df57cbf) would be the set of all sets with one element, and would therefore be a [proper class](https://en.wikipedia.org/wiki/Proper_class "Proper class"). Thus, due to [John von Neumann](https://en.wikipedia.org/wiki/John_von_Neumann "John von Neumann"), it is more common to assign [representatives](https://en.wikipedia.org/wiki/Representative_(mathematics) "Representative (mathematics)") of these classes.[[134]](https://en.wikipedia.org/wiki/Cardinality#cite_note-134)

[![Image 210](https://upload.wikimedia.org/wikipedia/commons/thumb/a/a5/Bijection.svg/224px-Bijection.svg.png)](https://en.wikipedia.org/wiki/File:Bijection.svg)

A [bijective function](https://en.wikipedia.org/wiki/Bijective_function "Bijective function"), ![Image 211: {\displaystyle f:X\to Y}](https://wikimedia.org/api/rest_v1/media/math/render/svg/abd1e080abef4bbdab67b43819c6431e7561361c) from the set X = {1,2,3,4} to the set Y demonstrates that Y has cardinality 4.

Given a basic sense of [natural numbers](https://en.wikipedia.org/wiki/Natural_numbers "Natural numbers"), a set is said to have cardinality ![Image 212: {\displaystyle n}](https://wikimedia.org/api/rest_v1/media/math/render/svg/a601995d55609f2d9f5e233e36fbe9ea26011b3b) if it can be put in one-to-one correspondence with the set ![Image 213: {\displaystyle \{1,\,2,\,\dots ,\,n\},}](https://wikimedia.org/api/rest_v1/media/math/render/svg/14502c3edd1bdf8a270fb0c6ab47a2b276ae0aee) analogous to [counting](https://en.wikipedia.org/wiki/Counting "Counting") its elements.[[135]](https://en.wikipedia.org/wiki/Cardinality#cite_note-:02-135)[[136]](https://en.wikipedia.org/wiki/Cardinality#cite_note-:13-136) For example, the set ![Image 214: {\displaystyle S=\{A,B,C,D\}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/e502f667705e33004a2c7717b219ebf598f8af87) has a natural correspondence with the set ![Image 215: {\displaystyle \{1,2,3,4\},}](https://wikimedia.org/api/rest_v1/media/math/render/svg/9eb89b8ef513b1960954e9de58b86ea8b1072754) and therefore is said to have cardinality 4. Other terminologies include "Its cardinality is 4" or "Its cardinal number is 4". In formal contexts, the natural numbers can be understood as some construction of objects satisfying the [Peano axioms](https://en.wikipedia.org/wiki/Peano_axioms "Peano axioms").[[137]](https://en.wikipedia.org/wiki/Cardinality#cite_note-137)

Showing that such a correspondence exists is not always trivial. [Combinatorics](https://en.wikipedia.org/wiki/Combinatorics "Combinatorics") is the area of mathematics primarily concerned with [counting](https://en.wikipedia.org/wiki/Counting "Counting"), both as a means and as an end to obtaining results, and certain properties of finite structures.[[138]](https://en.wikipedia.org/wiki/Cardinality#cite_note-138) The notion cardinality of finite sets is closely tied to many basic [combinatorial principles](https://en.wikipedia.org/wiki/Combinatorial_principles "Combinatorial principles"), and provides a set-theoretic foundation to prove them. It can be shown [by induction](https://en.wikipedia.org/wiki/By_induction "By induction") on the possible sizes of sets that finite cardinality corresponds uniquely with natural numbers (cf. _[Finite set §Uniqueness of cardinality](https://en.wikipedia.org/wiki/Finite\_set#Uniqueness\_of\_cardinality "Finite set")_).[[139]](https://en.wikipedia.org/wiki/Cardinality#cite_note-139) This is related—though not equivalent—to the [pigeonhole principle](https://en.wikipedia.org/wiki/Pigeonhole_principle "Pigeonhole principle").[[140]](https://en.wikipedia.org/wiki/Cardinality#cite_note-140)

The [addition principle](https://en.wikipedia.org/wiki/Addition_principle "Addition principle") asserts that given [disjoint](https://en.wikipedia.org/wiki/Disjoint_sets "Disjoint sets") sets ![Image 216: {\displaystyle A}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7daff47fa58cdfd29dc333def748ff5fa4c923e3) and ![Image 217: {\displaystyle B}](https://wikimedia.org/api/rest_v1/media/math/render/svg/47136aad860d145f75f3eed3022df827cee94d7a), ![Image 218: {\displaystyle |A\cup B|=|A|+|B|}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7f558cf8bc6b99ca442ef064c612c090e486e0a6), intuitively meaning that the sum of the parts is equal to the whole.[[141]](https://en.wikipedia.org/wiki/Cardinality#cite_note-141) The [multiplication principle](https://en.wikipedia.org/wiki/Multiplication_principle "Multiplication principle") asserts that given two sets ![Image 219: {\displaystyle A}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7daff47fa58cdfd29dc333def748ff5fa4c923e3) and ![Image 220: {\displaystyle B}](https://wikimedia.org/api/rest_v1/media/math/render/svg/47136aad860d145f75f3eed3022df827cee94d7a), ![Image 221: {\displaystyle |A\times B|=|A|\cdot |B|}](https://wikimedia.org/api/rest_v1/media/math/render/svg/4053f869f182e3272dc514d7b73282df8ee5155c), intuitively meaning that there are ![Image 222: {\displaystyle |A|\cdot |B|}](https://wikimedia.org/api/rest_v1/media/math/render/svg/ef914dea3f43f5dac1deed98cb75dd90066c9832) ways to pair objects from these sets.[[142]](https://en.wikipedia.org/wiki/Cardinality#cite_note-142) Both of these can be proven by a [bijective proof](https://en.wikipedia.org/wiki/Bijective_proof "Bijective proof"), together with induction.[[143]](https://en.wikipedia.org/wiki/Cardinality#cite_note-143) The more general result is the [inclusion–exclusion principle](https://en.wikipedia.org/wiki/Inclusion%E2%80%93exclusion_principle "Inclusion–exclusion principle"), which defines how to count the number of elements in overlapping sets.[[144]](https://en.wikipedia.org/wiki/Cardinality#cite_note-FOOTNOTEBrualdi2004160-144)

Naturally, a set is defined to be finite if it is [empty](https://en.wikipedia.org/wiki/Empty_set "Empty set") or can be put in correspondence with the set ![Image 223: {\displaystyle \{1,\,2,\,\dots ,\,n\},}](https://wikimedia.org/api/rest_v1/media/math/render/svg/14502c3edd1bdf8a270fb0c6ab47a2b276ae0aee) for some natural number ![Image 224: {\displaystyle n.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/e59df02a9f67a5da3c220f1244c99a46cc4eb1c6)[[135]](https://en.wikipedia.org/wiki/Cardinality#cite_note-:02-135)[[136]](https://en.wikipedia.org/wiki/Cardinality#cite_note-:13-136) However, there exist [other definitions of "finite"](https://en.wikipedia.org/wiki/Finite_set#Necessary_and_sufficient_conditions_for_finiteness "Finite set") which don't rely on a definition of "number." For example, a set is called [Dedekind-finite](https://en.wikipedia.org/wiki/Dedekind-finite "Dedekind-finite") if it cannot be put in one-to-one correspondece with a [proper subset](https://en.wikipedia.org/wiki/Proper_subset "Proper subset") of itself.[[145]](https://en.wikipedia.org/wiki/Cardinality#cite_note-145)

[![Image 225](https://upload.wikimedia.org/wikipedia/commons/thumb/e/e0/Aleph0.svg/250px-Aleph0.svg.png)](https://en.wikipedia.org/wiki/File:Aleph0.svg)

[Aleph-nought](https://en.wikipedia.org/wiki/Aleph-nought "Aleph-nought"), aleph-zero, or aleph-null: the smallest infinite cardinal number, and the cardinal number of the set of natural numbers. 

The [aleph numbers](https://en.wikipedia.org/wiki/Aleph_numbers "Aleph numbers") are a sequence of cardinal numbers that denote the size of [infinite sets](https://en.wikipedia.org/wiki/Infinite_sets "Infinite sets"), denoted with an [aleph](https://en.wikipedia.org/wiki/Aleph "Aleph")![Image 226: {\displaystyle \aleph ,}](https://wikimedia.org/api/rest_v1/media/math/render/svg/d65b13d13228d27d527ea28a6e34dd891c7a3035) the first letter of the [Hebrew alphabet](https://en.wikipedia.org/wiki/Hebrew_alphabet "Hebrew alphabet"). The first aleph number is ![Image 227: {\displaystyle \aleph _{0},}](https://wikimedia.org/api/rest_v1/media/math/render/svg/0dfce5b46ceb8fbe0b74194b9f94aa3a182e5599) called "aleph-nought", "aleph-zero", or "aleph-null", which represents the cardinality of the set of all [natural numbers](https://en.wikipedia.org/wiki/Natural_numbers "Natural numbers"): ![Image 228: {\displaystyle \aleph _{0}=|\mathbb {N} |=|\{0,1,2,3,\cdots \}|.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/00ccd1e85fe7ab80f685fde12eef2a7b9bfb4b93) Then, ![Image 229: {\displaystyle \aleph _{1}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/78c211ce8badf4ffbf9417ecceb0ef7ab0a8caed) represents the next largest cardinality. The most common way this is formalized in set theory is through [Von Neumann ordinals](https://en.wikipedia.org/wiki/Von_Neumann_ordinal "Von Neumann ordinal"), known as [Von Neumann cardinal assignment](https://en.wikipedia.org/wiki/Von_Neumann_cardinal_assignment "Von Neumann cardinal assignment").

[Ordinal numbers](https://en.wikipedia.org/wiki/Ordinal_number "Ordinal number") generalize the notion of _order_ to infinite sets. For example, 2 comes after 1, denoted ![Image 230: {\displaystyle 1<2,}](https://wikimedia.org/api/rest_v1/media/math/render/svg/2941776617a2c488cb312eacfb8456083af7cfe4) and 3 comes after both, denoted ![Image 231: {\displaystyle 1<2<3.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/09d5766f95267cc2083430944c8bc87a92269f1d) Then, one defines a new number, ![Image 232: {\displaystyle \omega ,}](https://wikimedia.org/api/rest_v1/media/math/render/svg/9b0d8eba2c8829fecf9414b15b1d02c24db3a553) which comes after every natural number, denoted ![Image 233: {\displaystyle 1<2<3<\cdots <\omega .}](https://wikimedia.org/api/rest_v1/media/math/render/svg/2fddb4b6e5871f0c6b6bc5472718b34a294ed1a8) Further ![Image 234: {\displaystyle \omega <\omega +1,}](https://wikimedia.org/api/rest_v1/media/math/render/svg/fa11d40033968978ee7e50649b189495dfc364c4) and so on. More formally, these ordinal numbers can be defined as follows:

![Image 235: {\displaystyle 0:=\{\},}](https://wikimedia.org/api/rest_v1/media/math/render/svg/8078737d801b5f581bb35c90e30037487ff394f2) the [empty set](https://en.wikipedia.org/wiki/Empty_set "Empty set"), ![Image 236: {\displaystyle 1:=\{0\},}](https://wikimedia.org/api/rest_v1/media/math/render/svg/739a1c42e54e67d562afd473e38d12ea1348aa6a)![Image 237: {\displaystyle 2:=\{0,1\},}](https://wikimedia.org/api/rest_v1/media/math/render/svg/782e14e2c288a15336dd0250d8b671d47dc85a53)![Image 238: {\displaystyle 3:=\{0,1,2\},}](https://wikimedia.org/api/rest_v1/media/math/render/svg/42e9face59baa1b27590fc05659a922d20331b3b) and so on. Then one can define ![Image 239: {\displaystyle m<n{\text{, if }}\,m\in n,}](https://wikimedia.org/api/rest_v1/media/math/render/svg/afc5cfac72da09d000d592cff69ef6cbfbbbf8b4) for example, ![Image 240: {\displaystyle 2\in \{0,1,2\}=3,}](https://wikimedia.org/api/rest_v1/media/math/render/svg/920b601fc96fbbef70d351730121a99826c493df) therefore ![Image 241: {\displaystyle 2<3.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/4643f272c93328896d6e94e61886cf05bb6613ac) Defining ![Image 242: {\displaystyle \omega :=\{0,1,2,3,\cdots \}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/dcef3060df6bcb4c5e6acbb051ea3960ba4af5e4) (a [limit ordinal](https://en.wikipedia.org/wiki/Limit_ordinal "Limit ordinal")) gives ![Image 243: {\displaystyle \omega }](https://wikimedia.org/api/rest_v1/media/math/render/svg/48eff443f9de7a985bb94ca3bde20813ea737be8) the desired property of being the smallest ordinal greater than all finite ordinal numbers. Further, ![Image 244: {\displaystyle \omega +1:=\{1,2,\cdots ,\omega \}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/efdb53d036060357714836b1c6a327fc039676f6), and so on.

Since ![Image 245: {\displaystyle \omega \sim \mathbb {N} }](https://wikimedia.org/api/rest_v1/media/math/render/svg/8ba0263de4cab1f814fa7736d2282ae898c27487) by the natural correspondence, one may define ![Image 246: {\displaystyle \aleph _{0}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/721cd7f8c15a2e72ad162bdfa5baea8eef98aab1) as the set of all finite ordinals. That is, ![Image 247: {\displaystyle \aleph _{0}:=\omega .}](https://wikimedia.org/api/rest_v1/media/math/render/svg/f4c7a9447c2f26cd7681b0907fc8b0fece0d1dfc) Then, ![Image 248: {\displaystyle \aleph _{1}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/78c211ce8badf4ffbf9417ecceb0ef7ab0a8caed) is the set of all countable ordinals (all ordinals ![Image 249: {\displaystyle \alpha }](https://wikimedia.org/api/rest_v1/media/math/render/svg/b79333175c8b3f0840bfb4ec41b8072c83ea88d3) with cardinality ![Image 250: {\displaystyle |\alpha |\leq \aleph _{0}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/58643dc20227bb06bac33eed70f84410351e7f81)), the [first uncountable ordinal](https://en.wikipedia.org/wiki/First_uncountable_ordinal "First uncountable ordinal"). Since a set cannot contain itself, ![Image 251: {\displaystyle \aleph _{1}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/78c211ce8badf4ffbf9417ecceb0ef7ab0a8caed) must have a strictly larger cardinality: ![Image 252: {\displaystyle \aleph _{0}<\aleph _{1}.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/1e8617e1aadafe263f67972f009199b226c81a14) Furthermore, ![Image 253: {\displaystyle \aleph _{2}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/765bb468708b2eec66bf2dc6505a4c92959d697d) is the set of all ordinals with cardinality less than or equal to ![Image 254: {\displaystyle \aleph _{1},}](https://wikimedia.org/api/rest_v1/media/math/render/svg/1af99741dafe89c100bb0426f65a633b7d2a2eb5) and in general the [successor cardinal](https://en.wikipedia.org/wiki/Successor_cardinal "Successor cardinal")![Image 255: {\displaystyle \kappa ^{+}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/5d7d17d0b32a3c5e666cc418d83ef1e0eae79b5c)is the set of all ordinals with cardinality up to ![Image 256: {\displaystyle \kappa }](https://wikimedia.org/api/rest_v1/media/math/render/svg/54ddec2e922c5caea4e47d04feef86e782dc8e6d). By the [well-ordering theorem](https://en.wikipedia.org/wiki/Well-ordering_theorem "Well-ordering theorem"), there cannot exist any set with cardinality between ![Image 257: {\displaystyle \aleph _{0}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/721cd7f8c15a2e72ad162bdfa5baea8eef98aab1) and ![Image 258: {\displaystyle \aleph _{1},}](https://wikimedia.org/api/rest_v1/media/math/render/svg/1af99741dafe89c100bb0426f65a633b7d2a2eb5) and every infinite set has some cardinality corresponding to some aleph ![Image 259: {\displaystyle \aleph _{\alpha },}](https://wikimedia.org/api/rest_v1/media/math/render/svg/b38697706c4d700455aa1cda5dd83dfc3da644cf) for some ordinal ![Image 260: {\displaystyle \alpha .}](https://wikimedia.org/api/rest_v1/media/math/render/svg/794850adc0db51d11a6d8cfa857538183424909c)

### Cardinal arithmetic

[[edit](https://en.wikipedia.org/w/index.php?title=Cardinality&action=edit&section=21 "Edit section: Cardinal arithmetic")]

[![Image 261](https://upload.wikimedia.org/wikipedia/commons/thumb/a/ab/AdditionShapes.svg/250px-AdditionShapes.svg.png)](https://en.wikipedia.org/wiki/File:AdditionShapes.svg)

One set has three distinct shapes while the other set has two. The consequence of the addition of the objects from the two sets is an instance of ![Image 262: {\displaystyle 3+2=5}](https://wikimedia.org/api/rest_v1/media/math/render/svg/12a0ae799cf5892cdf28c1b8c2502078ce6ada38)

Basic arithmetic can be done on cardinal numbers in a very natural way, by extending the theorems for finite combinatorial principles above. The intuitive principle that is ![Image 263: {\displaystyle A}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7daff47fa58cdfd29dc333def748ff5fa4c923e3) and ![Image 264: {\displaystyle B}](https://wikimedia.org/api/rest_v1/media/math/render/svg/47136aad860d145f75f3eed3022df827cee94d7a) are disjoint then addition of these sets is simply taking their [union](https://en.wikipedia.org/wiki/Union_(set_theory) "Union (set theory)"), written as ![Image 265: {\displaystyle |A\cup B|=|A|+|B|}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7f558cf8bc6b99ca442ef064c612c090e486e0a6). Thus if ![Image 266: {\displaystyle A}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7daff47fa58cdfd29dc333def748ff5fa4c923e3) and ![Image 267: {\displaystyle B}](https://wikimedia.org/api/rest_v1/media/math/render/svg/47136aad860d145f75f3eed3022df827cee94d7a) are infinite, cardinal addition is defined as ![Image 268: {\displaystyle |A|+|B|:=|A\sqcup B|}](https://wikimedia.org/api/rest_v1/media/math/render/svg/09bf17afac191ec9837b3395aa5c9fc18ba8c890) where ![Image 269: {\displaystyle \sqcup }](https://wikimedia.org/api/rest_v1/media/math/render/svg/1596aedf354da694149e44ce2bf53ede54eca8cb) denotes [disjoint union](https://en.wikipedia.org/wiki/Disjoint_union "Disjoint union"). Similarly, the multiplication of two sets is intuitively the number of ways to pair their elements (as in the [multiplication principle](https://en.wikipedia.org/wiki/Multiplication_principle "Multiplication principle")), therefore cardinal multiplication is defined as ![Image 270: {\displaystyle |A|\cdot |B|:=|A\times B|}](https://wikimedia.org/api/rest_v1/media/math/render/svg/e30bdbfdea3cbc55eb6220c19bdb09180a6bb027), where ![Image 271: {\displaystyle \times }](https://wikimedia.org/api/rest_v1/media/math/render/svg/0ffafff1ad26cbe49045f19a67ce532116a32703) denotes the [Cartesian product](https://en.wikipedia.org/wiki/Cartesian_product "Cartesian product"). These definitions can be shown to satisfy the basic properties of standard arithmetic:

Cardinal exponentiation ![Image 272: {\displaystyle |A|^{|B|}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/09fb0e31d2b339e94214d03ac8f414c9e00ad165) is defined via [set exponentiation](https://en.wikipedia.org/wiki/Set_exponentiation "Set exponentiation"), the set of all functions ![Image 273: {\displaystyle f:B\mapsto A}](https://wikimedia.org/api/rest_v1/media/math/render/svg/58ee6bcc96d4ec70d8b96bdac3009a0ad78b0335), that is, ![Image 274: {\displaystyle |A|^{|B|}:=|A^{B}|.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/0a125c6734ac8658b4990d831e16abb5a8a6d52f) For finite sets this can be shown to coincide with standard [natural number exponentiation](https://en.wikipedia.org/wiki/Exponentiation#Integer_exponents "Exponentiation"), but includes as a corollary that [zero to the power of zero](https://en.wikipedia.org/wiki/Zero_to_the_power_of_zero "Zero to the power of zero") is one ![Image 275: {\displaystyle (0^{0}=1)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/5b532c44d62283f7e2992e789350bae93fbc61c6) since there is exactly one function from the [empty set](https://en.wikipedia.org/wiki/Empty_set "Empty set") to itself: the [empty function](https://en.wikipedia.org/wiki/Empty_function "Empty function"). A combinatroial argument can be used to show ![Image 276: {\displaystyle 2^{|A|}=|{\mathcal {P}}(A)|}](https://wikimedia.org/api/rest_v1/media/math/render/svg/389b8ed476996cf7ef1c80497528b7ff58c0ee1e). In general, cardinal exponentiation is not as well-behaved as cardinal addition and multiplication. For example, even though it can be proven that the expression ![Image 277: {\displaystyle 2^{\aleph _{0}}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/779da5db4ed54fa334dd92089cdf1c284e45febb)does indeed correspond to some aleph, it is [unprovable](https://en.wikipedia.org/wiki/Unprovable "Unprovable") from standard set theories which aleph it corresponds to.

### Set of all cardinal numbers

[[edit](https://en.wikipedia.org/w/index.php?title=Cardinality&action=edit&section=22 "Edit section: Set of all cardinal numbers")]

The _set of all cardinal numbers_ refers to a hypothetical set which contains all cardinal numbers. Such a set cannot exist, which has been considered paradoxical, and related to the [Burali-Forti paradox](https://en.wikipedia.org/wiki/Burali-Forti_paradox "Burali-Forti paradox"). Using the definition of cardinal numbers as represetatives of their cardinalities. It begins by assuming there is some set ![Image 293: {\displaystyle S:=\{X\,|X{\text{ is a cardinal number}}\}.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/32784a887bf724f80721d55e8a359e42f1709d31) Then, if there is some largest cardinal ![Image 294: {\displaystyle \kappa \in S,}](https://wikimedia.org/api/rest_v1/media/math/render/svg/2483712470c97971a1a5fb3bc669df74ab8688b5) then the powerset ![Image 295: {\displaystyle 2^{\kappa }}](https://wikimedia.org/api/rest_v1/media/math/render/svg/77935d198ec4eba3fe76ff33e61be86b9525eb56) is strictly greater, and thus not in ![Image 296: {\displaystyle S.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/23bbb1f0f6ebdfa78b4fed06049640f7386bb44b) Conversly, if there is no largest element, then the [union](https://en.wikipedia.org/wiki/Union_(set_theory)#Arbitrary_union "Union (set theory)")![Image 297: {\displaystyle \bigcup S}](https://wikimedia.org/api/rest_v1/media/math/render/svg/bee5de6dfb89da11866e7f6d4a1af6fb403dfeac) contains the elements of all elements of ![Image 298: {\displaystyle S,}](https://wikimedia.org/api/rest_v1/media/math/render/svg/e0bcd8516b165aaacb234616d7d2d23478a35be7) and is therefore greater than or equal to each element. Since there is no largest element in ![Image 299: {\displaystyle S,}](https://wikimedia.org/api/rest_v1/media/math/render/svg/e0bcd8516b165aaacb234616d7d2d23478a35be7) for any element ![Image 300: {\displaystyle x\in S,}](https://wikimedia.org/api/rest_v1/media/math/render/svg/13ed21f8bd8f4c4c6f5fce9dd5ac8abf913cd338) there is another element ![Image 301: {\displaystyle y\in S}](https://wikimedia.org/api/rest_v1/media/math/render/svg/9d26e188ccec145558c09a23bb1d25149ea67973) such that ![Image 302: {\displaystyle |x|<|y|}](https://wikimedia.org/api/rest_v1/media/math/render/svg/9b5cb9fb4b302f5ab3e583a672604d9d8cfacb17) and ![Image 303: {\displaystyle |y|\leq {\Bigl |}\bigcup S{\Bigr |}.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/ed0e3d566e50927003101147baa9109c3e6970f6) Thus, for any ![Image 304: {\displaystyle x\in S,}](https://wikimedia.org/api/rest_v1/media/math/render/svg/13ed21f8bd8f4c4c6f5fce9dd5ac8abf913cd338)![Image 305: {\displaystyle |x|<{\Bigl |}\bigcup S{\Bigr |},}](https://wikimedia.org/api/rest_v1/media/math/render/svg/22ac4a33ee21bd369c9bd637cc0bc158e23462b7) and so ![Image 306: {\displaystyle {\Bigl |}\bigcup S{\Bigr |}\notin S.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/2d91bf8c04a07ee19c162d3e21aa6c070cbae89f) Thus, the collection of cardinal numbers is too large to form a set, and is a [proper class](https://en.wikipedia.org/wiki/Cardinality#Proper_classes).

Cardinality of the continuum
----------------------------

[[edit](https://en.wikipedia.org/w/index.php?title=Cardinality&action=edit&section=23 "Edit section: Cardinality of the continuum")]

[![Image 307](https://upload.wikimedia.org/wikipedia/commons/thumb/f/fd/Number_line.png/330px-Number_line.png)](https://en.wikipedia.org/wiki/File:Number_line.png)

The [number line](https://en.wikipedia.org/wiki/Number_line "Number line"), containing all points in its continuum.

The [number line](https://en.wikipedia.org/wiki/Number_line "Number line") is a geometric construct of the intuitive notions of "[space](https://en.wikipedia.org/wiki/Space "Space")" and "[distance](https://en.wikipedia.org/wiki/Distance "Distance")" wherein each point corresponds to a distinct quantity or position along a continuous path. The terms "continuum" and "continuous" refer to the totality of this line, having some space (other points) between any two points on the line ([dense](https://en.wikipedia.org/wiki/Dense_order "Dense order") and [archimedian](https://en.wikipedia.org/wiki/Archimedean_property "Archimedean property")) and the absence of any gaps ([completeness](https://en.wikipedia.org/wiki/Completeness_of_the_real_numbers "Completeness of the real numbers")), This intuitive construct is formalized by the set of [real numbers](https://en.wikipedia.org/wiki/Real_numbers "Real numbers")![Image 308: {\displaystyle (\mathbb {R} )}](https://wikimedia.org/api/rest_v1/media/math/render/svg/66964ab8a4912f2be91dc22cceb762be1d3d8d05) which model the continuum as a complete, densely ordered, uncountable set.

[![Image 309](https://upload.wikimedia.org/wikipedia/commons/thumb/9/99/Cantor_set_binary_tree.svg/330px-Cantor_set_binary_tree.svg.png)](https://en.wikipedia.org/wiki/File:Cantor_set_binary_tree.svg)

First five itterations approaching the Cantor set

The [cardinality of the continuum](https://en.wikipedia.org/wiki/Cardinality_of_the_continuum "Cardinality of the continuum"), denoted by "![Image 310: {\displaystyle {\mathfrak {c}}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/b21924b960341255be18e538e51404718f29cbc0)" (a lowercase [fraktur script](https://en.wikipedia.org/wiki/Fraktur_(script) "Fraktur (script)") "c"), remains invariant under various transformations and mappings, many considered surprising. For example, all intervals on the real line e.g. ![Image 311: {\displaystyle [0,1]}](https://wikimedia.org/api/rest_v1/media/math/render/svg/738f7d23bb2d9642bab520020873cccbef49768d), and ![Image 312: {\displaystyle [0,2]}](https://wikimedia.org/api/rest_v1/media/math/render/svg/120ef5837b0c64a40a2333f5aefd3c36fc458e91), have the same cardinality as the entire set ![Image 313: {\displaystyle \mathbb {R} }](https://wikimedia.org/api/rest_v1/media/math/render/svg/786849c765da7a84dbc3cce43e96aad58a5868dc). First, ![Image 314: {\displaystyle f(x)=2x}](https://wikimedia.org/api/rest_v1/media/math/render/svg/e3a8ebea86ba5d3a71121e0a4156f5ec07b25220) is a bijection from ![Image 315: {\displaystyle [0,1]}](https://wikimedia.org/api/rest_v1/media/math/render/svg/738f7d23bb2d9642bab520020873cccbef49768d) to ![Image 316: {\displaystyle [0,2]}](https://wikimedia.org/api/rest_v1/media/math/render/svg/120ef5837b0c64a40a2333f5aefd3c36fc458e91). Then, the [tangent function](https://en.wikipedia.org/wiki/Tangent_function "Tangent function") is a bijection from the interval ![Image 317: {\textstyle \left({\frac {-\pi }{2}}\,,{\frac {\pi }{2}}\right)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/a355dde5be6bc38336ae50f304bfde0d6548b287) to the whole real line. A more surprising example is the [Cantor set](https://en.wikipedia.org/wiki/Cantor_set "Cantor set"), which is defined as follows: take the interval ![Image 318: {\displaystyle [0,1]}](https://wikimedia.org/api/rest_v1/media/math/render/svg/738f7d23bb2d9642bab520020873cccbef49768d) and remove the middle third ![Image 319: {\textstyle \left({\frac {1}{3}},{\frac {2}{3}}\right)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/5f7998cd3d34b06f34d797f2e9c0e308c5342e6d), then remove the middle third of each of the two remaining segments, and continue removing middle thirds (see image). The Cantor set is the set of points that survive this process. This set that remains is all of the points whose decimal expansion can be written in [ternary](https://en.wikipedia.org/wiki/Ternary_numeral_system "Ternary numeral system") without a 1. Reinterpreting these decimal expansions as [binary](https://en.wikipedia.org/wiki/Binary_number "Binary number") (e.g. by replacing the 2s with 1s) gives a bijection between the Cantor set and the interval ![Image 320: {\displaystyle [0,1]}](https://wikimedia.org/api/rest_v1/media/math/render/svg/738f7d23bb2d9642bab520020873cccbef49768d).

[![Image 321](https://upload.wikimedia.org/wikipedia/commons/thumb/6/64/Peanocurve.svg/500px-Peanocurve.svg.png)](https://en.wikipedia.org/wiki/File:Peanocurve.svg)

Three iterations of a [Peano curve](https://en.wikipedia.org/wiki/Peano_curve "Peano curve") construction, whose [limit](https://en.wikipedia.org/wiki/Limit_of_a_sequence "Limit of a sequence") is a [space-filling curve](https://en.wikipedia.org/wiki/Space-filling_curve "Space-filling curve").

[Space-filling curves](https://en.wikipedia.org/wiki/Space-filling_curves "Space-filling curves") are continuous surjective maps from the [unit interval](https://en.wikipedia.org/wiki/Unit_interval "Unit interval")![Image 322: {\displaystyle [0,1]}](https://wikimedia.org/api/rest_v1/media/math/render/svg/738f7d23bb2d9642bab520020873cccbef49768d) onto the [unit square](https://en.wikipedia.org/wiki/Unit_square "Unit square") on ![Image 323: {\displaystyle \mathbb {R} ^{2}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/e150115ab9f63023215109595b76686a1ff890fd), with classical examples such as the [Peano curve](https://en.wikipedia.org/wiki/Peano_curve "Peano curve") and [Hilbert curve](https://en.wikipedia.org/wiki/Hilbert_curve "Hilbert curve"). Although such maps are not injective, they are indeed surjective, and thus suffice to demonstrate cardinal equivalence. They can be reused at each dimension to show that ![Image 324: {\displaystyle |\mathbb {R} |=|\mathbb {R} ^{n}|={\mathfrak {c}}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/269f834604b73c8506abf7d234a9909f13867538) for any dimension ![Image 325: {\displaystyle n\geq 1.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/27f9176dbc447117f4bf4eb2cbe6749f669e60ab) The infinite [cartesian product](https://en.wikipedia.org/wiki/Cartesian_product "Cartesian product")![Image 326: {\displaystyle \mathbb {R} ^{\infty }}](https://wikimedia.org/api/rest_v1/media/math/render/svg/da5e5160fa2811da2c516b0fa543236c5cf707fc), can also be shown to have cardinality ![Image 327: {\displaystyle {\mathfrak {c}}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/b21924b960341255be18e538e51404718f29cbc0). This can be established by cardinal exponentiation: ![Image 328: {\displaystyle |\mathbb {R} ^{\infty }|={\mathfrak {c}}^{\aleph _{0}}=\left(2^{\aleph _{0}}\right)^{\aleph _{0}}=2^{(\aleph _{0}\cdot \aleph _{0})}=2^{\aleph _{0}}={\mathfrak {c}}=|\mathbb {R} |}](https://wikimedia.org/api/rest_v1/media/math/render/svg/31ac24fd2aa10fffcb236f141e7fc217316c03e2). Thus, the real numbers, all finite-dimensional real spaces, and the countable cartesian product share the same cardinality.

As shown in [§Uncountable sets](https://en.wikipedia.org/wiki/Cardinality#Uncountable_sets), the set of real numbers is strictly larger than the set of natural numbers. Specifically, ![Image 329: {\displaystyle |\mathbb {R} |=|{\mathcal {P}}(\mathbb {N} )|}](https://wikimedia.org/api/rest_v1/media/math/render/svg/808bdb4437769e551067b58ad09f4f422bde076b). The [Continuum Hypothesis](https://en.wikipedia.org/wiki/Continuum_Hypothesis "Continuum Hypothesis") (CH) asserts that the real numbers have the next largest cardinality after the natural numbers, that is ![Image 330: {\displaystyle |\mathbb {R} |=\aleph _{1}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/948a68092d411cf2e7491f37f24fbd4c2348f458). As shown by [Gödel](https://en.wikipedia.org/wiki/Kurt_G%C3%B6del "Kurt Gödel") and [Cohen](https://en.wikipedia.org/wiki/Paul_Cohen "Paul Cohen"), the continuum hypothesis is [independent](https://en.wikipedia.org/wiki/Independence_(mathematical_logic) "Independence (mathematical logic)") of [ZFC](https://en.wikipedia.org/wiki/Zermelo%E2%80%93Fraenkel_set_theory_with_the_axiom_of_choice "Zermelo–Fraenkel set theory with the axiom of choice"), a standard axiomatization of set theory; that is, it is impossible to prove the continuum hypothesis or its negation from ZFC—provided that ZFC is [consistent](https://en.wikipedia.org/wiki/Consistency "Consistency").[[146]](https://en.wikipedia.org/wiki/Cardinality#cite_note-146)[[147]](https://en.wikipedia.org/wiki/Cardinality#cite_note-147)[[148]](https://en.wikipedia.org/wiki/Cardinality#cite_note-148) The [Generalized Continuum Hypothesis](https://en.wikipedia.org/wiki/Generalized_Continuum_Hypothesis "Generalized Continuum Hypothesis") (GCH) extends this to all infinite cardinals, stating that ![Image 331: {\displaystyle 2^{\aleph _{\alpha }}=\aleph _{\alpha +1}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/0b3e77a787cb7dc28f91ebdf7de8c2403135cca8) for every ordinal ![Image 332: {\displaystyle \alpha }](https://wikimedia.org/api/rest_v1/media/math/render/svg/b79333175c8b3f0840bfb4ec41b8072c83ea88d3). Without GHC, the cardinality of ![Image 333: {\displaystyle \mathbb {R} }](https://wikimedia.org/api/rest_v1/media/math/render/svg/786849c765da7a84dbc3cce43e96aad58a5868dc) cannot be written in terms of specific alephs. The [Beth numbers](https://en.wikipedia.org/wiki/Beth_numbers "Beth numbers") provide a concise notation for powersets of the real numbers starting from ![Image 334: {\displaystyle \beth _{0}=|\mathbb {N} |}](https://wikimedia.org/api/rest_v1/media/math/render/svg/979e99857f1cbb77b289710a579884749e8a461a), then ![Image 335: {\displaystyle \beth _{1}=2^{\beth _{0}}=|\mathbb {R} |}](https://wikimedia.org/api/rest_v1/media/math/render/svg/9b80e8d1dd6f081dbb009f713f312f70416a65ad), and ![Image 336: {\displaystyle \beth _{2}=|{\mathcal {P}}(\mathbb {R} )|=2^{\beth _{1}}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/d64245e159d0586354ba916da451cfdadb909ecd), and in general ![Image 337: {\displaystyle \beth _{n+1}=2^{\beth _{n}}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/f5d6c2cc4582ca63778a9d35ac75d8fa63eccde5) and ![Image 338: {\displaystyle \beth _{\lambda }=\bigcup _{\alpha <\lambda }\beth _{\alpha }}](https://wikimedia.org/api/rest_v1/media/math/render/svg/5343f518b7071d9698cc4e6905054f63b3609c22) if ![Image 339: {\displaystyle \lambda }](https://wikimedia.org/api/rest_v1/media/math/render/svg/b43d0ea3c9c025af1be9128e62a18fa74bedda2a) is a [limit ordinal](https://en.wikipedia.org/wiki/Limit_ordinal "Limit ordinal").

[![Image 340](https://upload.wikimedia.org/wikipedia/commons/thumb/6/66/Lowenheim-skolem.svg/330px-Lowenheim-skolem.svg.png)](https://en.wikipedia.org/wiki/File:Lowenheim-skolem.svg)

Illustration of the [Löwenheim–Skolem theorem](https://en.wikipedia.org/wiki/L%C3%B6wenheim%E2%80%93Skolem_theorem "Löwenheim–Skolem theorem"), where ![Image 341: {\displaystyle {\mathcal {M}}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/2cc2abebd45ec020509a0ec548b67c9a2cb7cecd) and ![Image 342: {\displaystyle {\mathcal {N}}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/b7551c7bed2cd2ee83e10536d157c94a5f8f72fd) are models of set theory, and ![Image 343: {\displaystyle \kappa }](https://wikimedia.org/api/rest_v1/media/math/render/svg/54ddec2e922c5caea4e47d04feef86e782dc8e6d) is an arbitrary infinite cardinal number.

In [model theory](https://en.wikipedia.org/wiki/Model_theory "Model theory"), a [model](https://en.wikipedia.org/wiki/Model_(mathematical_logic) "Model (mathematical logic)") corresponds to a specific interpretation of a [formal language](https://en.wikipedia.org/wiki/Formal_language "Formal language") or [theory](https://en.wikipedia.org/wiki/Theory_(mathematical_logic) "Theory (mathematical logic)"). It consists of a [domain](https://en.wikipedia.org/wiki/Domain_of_discourse "Domain of discourse") (a set of objects) and an [interpretation](https://en.wikipedia.org/wiki/Interpretation_(logic) "Interpretation (logic)") of the symbols in the language, such that the axioms of the theory are satisfied within this structure. In [first-order logic](https://en.wikipedia.org/wiki/First-order_logic "First-order logic"), the [Löwenheim–Skolem theorem](https://en.wikipedia.org/wiki/L%C3%B6wenheim%E2%80%93Skolem_theorem "Löwenheim–Skolem theorem") states that if a theory has an infinite model, then it also has models of every other infinite cardinality. Applied to set theory, it asserts that [Zermelo–Fraenkel set theory](https://en.wikipedia.org/wiki/Zermelo%E2%80%93Fraenkel_set_theory "Zermelo–Fraenkel set theory"), which proves the existence of uncountable sets such as ![Image 344: {\displaystyle {\mathcal {P}}(\mathbb {N} )}](https://wikimedia.org/api/rest_v1/media/math/render/svg/37b896f76c785d477a86be5965fbdf3814947022), nevertheless has a countable model. Thus, [Skolem's paradox](https://en.wikipedia.org/wiki/Skolem%27s_paradox "Skolem's paradox") was posed as follows: how can it be that there exists a domain of set theory which only contains countably many objects, but is capable of [satisfying](https://en.wikipedia.org/wiki/Satisfiability "Satisfiability") the statement "there exists a set with uncountably many elements"?[[149]](https://en.wikipedia.org/wiki/Cardinality#cite_note-:62-149)

Skolem's paradox does not only apply to ZFC, but _any_ first-order set theory, if it is [consistent](https://en.wikipedia.org/wiki/Consistent "Consistent"), has a model which is countable. A mathematical explanation of the paradox, showing that it is not a true contradiction in mathematics, was first given in 1922 by [Thoralf Skolem](https://en.wikipedia.org/wiki/Thoralf_Skolem "Thoralf Skolem"). He explained that the countability or uncountability of a set is not [absolute](https://en.wikipedia.org/wiki/Absoluteness_(logic) "Absoluteness (logic)"), but relative to the model in which the cardinality is measured. This is because, for example, if the set ![Image 345: {\displaystyle X}](https://wikimedia.org/api/rest_v1/media/math/render/svg/68baa052181f707c662844a465bfeeb135e82bab) is countable in a model of set theory then there is a bijection ![Image 346: {\displaystyle f:\mathbb {N} \mapsto X.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/3fa06f857e3b84288924b710e1cc6e5a2b0cdf20) But a submodel containing ![Image 347: {\displaystyle X}](https://wikimedia.org/api/rest_v1/media/math/render/svg/68baa052181f707c662844a465bfeeb135e82bab) which excludes all such functions would thus contain no bijection between ![Image 348: {\displaystyle X}](https://wikimedia.org/api/rest_v1/media/math/render/svg/68baa052181f707c662844a465bfeeb135e82bab) and ![Image 349: {\displaystyle \mathbb {N} }](https://wikimedia.org/api/rest_v1/media/math/render/svg/fdf9a96b565ea202d0f4322e9195613fb26a9bed), and therefore ![Image 350: {\displaystyle X}](https://wikimedia.org/api/rest_v1/media/math/render/svg/68baa052181f707c662844a465bfeeb135e82bab) would be uncountable. In [second-order](https://en.wikipedia.org/wiki/Second-order_logic "Second-order logic") and [higher-order logics](https://en.wikipedia.org/wiki/Higher-order_logics "Higher-order logics"), the Löwenheim–Skolem theorem does not hold. This is due to the fact that second-order logic quantifies over all subsets of the domain. Skolem's work was harshly received by [Ernst Zermelo](https://en.wikipedia.org/wiki/Ernst_Zermelo "Ernst Zermelo"), who argued against the limitations of first-order logic and Skolem's notion of "relativity", but the result quickly came to be accepted by the mathematical community.[[150]](https://en.wikipedia.org/wiki/Cardinality#cite_note-150)[[149]](https://en.wikipedia.org/wiki/Cardinality#cite_note-:62-149)

[[edit](https://en.wikipedia.org/w/index.php?title=Cardinality&action=edit&section=25 "Edit section: Alternative and additional axioms")]

### Without the axiom of choice

[[edit](https://en.wikipedia.org/w/index.php?title=Cardinality&action=edit&section=26 "Edit section: Without the axiom of choice")]

[![Image 351](https://upload.wikimedia.org/wikipedia/commons/thumb/c/cd/Axiome_du_choix.png/330px-Axiome_du_choix.png)](https://en.wikipedia.org/wiki/File:Axiome_du_choix.png)

Illustration of the axiom of choice, with each set represented as a jar and its elements represented as marbles.

The [Axiom of Choice](https://en.wikipedia.org/wiki/Axiom_of_Choice "Axiom of Choice") (AC) is a controversial principle in the foundations of mathematics. Roughly, it states that given any collection of non-empty sets, it is possible to construct a new set by choosing one element from each set, even if the collection is infinite. In many cases, a set created by choosing elements can be made without invoking the axiom of choice. But it is not possible to do this in general, in which case, the axiom of choice must be invoked.

If the Axiom of Choice is assumed to be false, it has several implications:

*   There may exist sets which are [Dedekind-finite](https://en.wikipedia.org/wiki/Dedekind-finite "Dedekind-finite"), meaning they cannot be put in bijection with a proper subset of itself, but whose elements cannot be counted (put in bijection with ![Image 352: {\displaystyle \{1,2,\cdots ,n\}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7ce690fff69c1add46052877e69d2a6cd1af684e) for any n).
*   Cardinal inequality ![Image 353: {\displaystyle (\preceq )}](https://wikimedia.org/api/rest_v1/media/math/render/svg/31e7fa28d3a55d818bf10cecfdf83d2e68aec2d7) cannot be a total order on all sets. That is, given any two sets ![Image 354: {\displaystyle A,B,}](https://wikimedia.org/api/rest_v1/media/math/render/svg/c6adce00c0dbd0268c0b63529ef77c10d371c5a6) it may be that neither ![Image 355: {\displaystyle A\preceq B}](https://wikimedia.org/api/rest_v1/media/math/render/svg/bd084bf1b0042f6422a02013f8051a9d2eaee18b) nor ![Image 356: {\displaystyle B\preceq A}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7630f81ffd3df39067ccda5c1c69dba497043a8e) hold. Further, since Alephs can naturally be compared, there exist sets which do not correspond to any Aleph number.
*   In ZF alone, it is impossible to prove there exists a [cardinal function](https://en.wikipedia.org/wiki/Cardinal_function "Cardinal function")![Image 357: {\displaystyle S\mapsto |S|}](https://wikimedia.org/api/rest_v1/media/math/render/svg/5668d9e1709014bdcb2970dfa9fc4ad91a4afb7c) which satisfies the [cardinal number axiom](https://en.wikipedia.org/wiki/Cardinality#Cardinal_numbers), and satisfies ![Image 358: {\displaystyle |S|\sim S}](https://wikimedia.org/api/rest_v1/media/math/render/svg/9341c721a58ed04f08794c9153621b26ca1bf803) for all ![Image 359: {\displaystyle S.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/23bbb1f0f6ebdfa78b4fed06049640f7386bb44b)[[151]](https://en.wikipedia.org/wiki/Cardinality#cite_note-151) Thus, some authors reserve ![Image 360: {\displaystyle \operatorname {card} (S)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/35c3b3cb3523c0f543ebab3b9210d2c3decafec6) for the cardinality of well-orderable sets. Still, the function can be defined using [Scott's trick](https://en.wikipedia.org/wiki/Scott%27s_trick "Scott's trick"), which assigns each set ![Image 361: {\displaystyle S}](https://wikimedia.org/api/rest_v1/media/math/render/svg/4611d85173cd3b508e67077d4a1252c9c05abca2) to the least rank ![Image 362: {\displaystyle V_{\alpha }}](https://wikimedia.org/api/rest_v1/media/math/render/svg/582d40a9ff663187250948b07bb66456162c2042) of the [Von Neumann universe](https://en.wikipedia.org/wiki/Von_Neumann_universe "Von Neumann universe") which contains a set equinumerous to ![Image 363: {\displaystyle S.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/23bbb1f0f6ebdfa78b4fed06049640f7386bb44b)
*   One may define the "infinite product" of cardinal numbers as the cardinality of the set of all [sequences](https://en.wikipedia.org/wiki/Sequences "Sequences") of their elements. Asserting that this set is always nonempty is equivalent to the axiom of choice. This is why Bertrand Russell called AC the _Multiplicative axiom_.
*   Although the [Generalized Continuum Hypothesis](https://en.wikipedia.org/wiki/Generalized_Continuum_Hypothesis "Generalized Continuum Hypothesis") (GCH) is independent of ZFC, it can be shown that ZF+GCH is sufficient to prove AC. Thus, if AC is false, it follows that the GCH does not hold.

[![Image 364](https://upload.wikimedia.org/wikipedia/commons/thumb/1/14/Frank-R%C3%BChl_sample-Tav.svg/187px-Frank-R%C3%BChl_sample-Tav.svg.png)](https://en.wikipedia.org/wiki/File:Frank-R%C3%BChl_sample-Tav.svg)

The [Hebrew](https://en.wikipedia.org/wiki/Hebrew_alphabet "Hebrew alphabet") letter [Tav](https://en.wikipedia.org/wiki/Taw "Taw"), denoting the size of the "[Absolute infinite](https://en.wikipedia.org/wiki/Absolute_infinite "Absolute infinite")"

Some set theories allow for [proper classes](https://en.wikipedia.org/wiki/Proper_classes "Proper classes"), which are, roughly, collections "too large" to form sets. For example, the [Universe of all sets](https://en.wikipedia.org/wiki/Universe_(mathematics) "Universe (mathematics)"), the [class of all cardinal numbers](https://en.wikipedia.org/wiki/Cardinality#Set_of_all_cardinal_numbers), or the [class of all ordinal numbers](https://en.wikipedia.org/wiki/Burali-Forti_paradox "Burali-Forti paradox"). Such set theories include [Von Neumann–Bernays–Gödel set theory](https://en.wikipedia.org/wiki/Von_Neumann%E2%80%93Bernays%E2%80%93G%C3%B6del_set_theory "Von Neumann–Bernays–Gödel set theory"), and [Morse–Kelley set theory](https://en.wikipedia.org/wiki/Morse%E2%80%93Kelley_set_theory "Morse–Kelley set theory"). In such set theories, some authors find this definition more elegant than assigning representatives, as it more accurately describes the concept by "definition by abstraction".

Proper classes can, in a sense, be assigned cardinalities. Cantor originally called the cardinality of a proper class "[Absolute infinite](https://en.wikipedia.org/wiki/Absolute_infinite "Absolute infinite")", denoted with [tav](https://en.wikipedia.org/wiki/Taw "Taw")**ת‎** (the last letter of the [Hebrew alphabet](https://en.wikipedia.org/wiki/Hebrew_alphabet "Hebrew alphabet")), and associated the concept with [God](https://en.wikipedia.org/wiki/God "God"). The first to distinguish between sets and classes was [John von Neumann](https://en.wikipedia.org/wiki/John_von_Neumann "John von Neumann"), who formalized the notion of "too large to form sets". More accurately, he defined a class to be a proper class if and only if it was equinumerous with the whole Universe of sets (cf. [_Axiom of limitation of size_](https://en.wikipedia.org/wiki/Axiom_of_limitation_of_size "Axiom of limitation of size")). Thus, all proper classes have the same "size". The axiom has several implications, mostly relating to the [limitation of size](https://en.wikipedia.org/wiki/Limitation_of_size "Limitation of size") principles of early set theory. It implies the [axiom of specification](https://en.wikipedia.org/wiki/Axiom_of_specification "Axiom of specification"), the [axiom of replacement](https://en.wikipedia.org/wiki/Axiom_of_replacement "Axiom of replacement"), the [axiom of union](https://en.wikipedia.org/wiki/Axiom_of_union "Axiom of union"), and the [axiom of global choice](https://en.wikipedia.org/wiki/Axiom_of_global_choice "Axiom of global choice").

### Axiom of constructability

[[edit](https://en.wikipedia.org/w/index.php?title=Cardinality&action=edit&section=28 "Edit section: Axiom of constructability")]

The [axiom of constructibility](https://en.wikipedia.org/wiki/Axiom_of_constructibility "Axiom of constructibility") was the axiom introduced by [Kurt Gödel](https://en.wikipedia.org/wiki/Kurt_G%C3%B6del "Kurt Gödel") to prove the relative consistency of the axiom of choice and generalized continuum hypothesis. It states that all sets are "constructable" (cf. _[Constructible universe §What L is](https://en.wikipedia.org/wiki/Constructible\_universe#What\_L\_is "Constructible universe")_). It has several consequences, naturally, including AC and GCH, but resolves [several other important questions in set theory](https://en.wikipedia.org/wiki/Axiom_of_constructibility#Statements_true_in_L "Axiom of constructibility"). The axiom, however, is not well-accepted by set-theorists, most believing it to be "too restrictive". In part it is because the axiom is contradicted by sufficiently strong [large cardinal axioms](https://en.wikipedia.org/wiki/Large_cardinal_axiom "Large cardinal axiom").
