[Analytic geometry](https://en.wikipedia.org/wiki/Analytic_geometry "Analytic geometry") associates to each point in the [Euclidean plane](https://en.wikipedia.org/wiki/Euclidean_plane "Euclidean plane") an ordered pair. The red [ellipse](https://en.wikipedia.org/wiki/Ellipse "Ellipse") is associated with the set of all pairs (_x_,_y_) such that ⁠_x_ 2/4⁠ + _y_ 2 = 1.

In [mathematics](https://en.wikipedia.org/wiki/Mathematics "Mathematics"), an **ordered pair**, denoted (_a_, _b_), is a pair of objects in which their order is significant. The ordered pair (_a_, _b_) is different from the ordered pair (_b_, _a_), unless _a_ = _b_. In contrast, the _[unordered pair](https://en.wikipedia.org/wiki/Unordered\_pair "Unordered pair")_, denoted {_a_, _b_}, always equals the unordered pair {_b_, _a_}.

Ordered pairs are also called [2-tuples](https://en.wikipedia.org/wiki/Tuple "Tuple"), or [sequences](https://en.wikipedia.org/wiki/Sequence "Sequence") (sometimes, lists in a computer science context) of length 2. Ordered pairs of [scalars](https://en.wikipedia.org/wiki/Scalar_(mathematics) "Scalar (mathematics)") are sometimes called 2-dimensional [vectors](https://en.wikipedia.org/wiki/Vector_(mathematics_and_physics) "Vector (mathematics and physics)"). (Technically, this is an abuse of [terminology](https://en.wikipedia.org/wiki/Terminology "Terminology") since an ordered pair need not be an element of a [vector space](https://en.wikipedia.org/wiki/Vector_space "Vector space").) The entries of an ordered pair can be other ordered pairs, enabling the [recursive](https://en.wikipedia.org/wiki/Recursive_definition "Recursive definition") definition of ordered [_n_-tuples](https://en.wikipedia.org/wiki/N-tuple "N-tuple") (ordered lists of _n_ objects). For example, the ordered triple (_a_,_b_,_c_) can be defined as (_a_, (_b_,_c_)), i.e., as one pair nested in another.

In the ordered pair (_a_, _b_), the object _a_ is called the _first entry_, and the object _b_ the _second entry_ of the pair. Alternatively, the objects are called the first and second _components_, the first and second _coordinates_, or the left and right _projections_ of the ordered pair.

[Cartesian products](https://en.wikipedia.org/wiki/Cartesian_product "Cartesian product") and [binary relations](https://en.wikipedia.org/wiki/Binary_relation "Binary relation") (and hence [functions](https://en.wikipedia.org/wiki/Function_(mathematics) "Function (mathematics)")) are defined in terms of ordered pairs, cf. picture.

Let ![Image 2: {\displaystyle (a_{1},b_{1})}](https://wikimedia.org/api/rest_v1/media/math/render/svg/4169cf3c7bae0f4515e37aa3095f85d47636d56b) and ![Image 3: {\displaystyle (a_{2},b_{2})}](https://wikimedia.org/api/rest_v1/media/math/render/svg/8eebf25545e61dd0b2385710f8ea0373a1904be3) be ordered pairs. Then the _characteristic_ (or _defining_) _property_ of the ordered pair is: ![Image 4: {\displaystyle (a_{1},b_{1})=(a_{2},b_{2}){\text{  if and only if  }}a_{1}=a_{2}{\text{ and }}b_{1}=b_{2}.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/09247d864749eb0d9b7c15009f0c0c38cbc041ad)

The [set](https://en.wikipedia.org/wiki/Set_(mathematics) "Set (mathematics)") of all ordered pairs whose first entry is in some set _A_ and whose second entry is in some set _B_ is called the [Cartesian product](https://en.wikipedia.org/wiki/Cartesian_product "Cartesian product") of _A_ and _B_, and written _A_ × _B_. A [binary relation](https://en.wikipedia.org/wiki/Binary_relation "Binary relation") between sets _A_ and _B_ is a [subset](https://en.wikipedia.org/wiki/Subset "Subset") of _A_ × _B_.

The (_a_, _b_) notation may be used for other purposes, most notably as denoting [open intervals](https://en.wikipedia.org/wiki/Open_interval "Open interval") on the [real number line](https://en.wikipedia.org/wiki/Real_number_line "Real number line"). In such situations, the context will usually make it clear which meaning is intended.[[1]](https://en.wikipedia.org/wiki/Ordered_pair#cite_note-1)[[2]](https://en.wikipedia.org/wiki/Ordered_pair#cite_note-2) For additional clarification, the ordered pair may be denoted by the variant notation ![Image 5: {\textstyle \langle a,b\rangle }](https://wikimedia.org/api/rest_v1/media/math/render/svg/2d6f4228961f67899af5cbf5a660587b0ca30aaa), but this notation also has other uses.

The left and right projection of a pair _p_ is usually denoted by π 1(_p_) and π 2(_p_), or by π _ℓ_(_p_) and π _r_(_p_), respectively. In contexts where arbitrary _n_-tuples are considered, π _n_

_i_(_t_) is a common notation for the _i_-th component of an _n_-tuple _t_.

Informal and formal definitions
-------------------------------

[[edit](https://en.wikipedia.org/w/index.php?title=Ordered_pair&action=edit&section=2 "Edit section: Informal and formal definitions")]

In some introductory mathematics textbooks an informal (or intuitive) definition of ordered pair is given, such as

> For any two objects a and b, the ordered pair (_a_, _b_) is a notation specifying the two objects a and b, in that order.[[3]](https://en.wikipedia.org/wiki/Ordered_pair#cite_note-Wolf-3)

This is usually followed by a comparison to a set of two elements; pointing out that in a set a and b must be different, but in an ordered pair they may be equal and that while the order of listing the elements of a set doesn't matter, in an ordered pair changing the order of distinct entries changes the ordered pair.

This "definition" is unsatisfactory because it is only descriptive and is based on an intuitive understanding of _order_. However, as is sometimes pointed out, no harm will come from relying on this description and almost everyone thinks of ordered pairs in this manner.[[4]](https://en.wikipedia.org/wiki/Ordered_pair#cite_note-4)

A more satisfactory approach is to observe that the characteristic property of ordered pairs given above is all that is required to understand the role of ordered pairs in mathematics. Hence the ordered pair can be taken as a [primitive notion](https://en.wikipedia.org/wiki/Primitive_notion "Primitive notion"), whose associated axiom is the characteristic property. This was the approach taken by the [N. Bourbaki](https://en.wikipedia.org/wiki/Nicolas_Bourbaki "Nicolas Bourbaki") group in its _Theory of Sets_, published in 1954. However, this approach also has its drawbacks as both the existence of ordered pairs and their characteristic property must be axiomatically assumed.[[3]](https://en.wikipedia.org/wiki/Ordered_pair#cite_note-Wolf-3)

Another way to rigorously deal with ordered pairs is to define them formally in the context of set theory. This can be done in several ways and has the advantage that existence and the characteristic property can be proven from the axioms that define the set theory. One of the most cited versions of this definition is due to Kuratowski (see below) and his definition was used in the second edition of Bourbaki's _Theory of Sets_, published in 1970. Even those mathematical textbooks that give an informal definition of ordered pairs will often mention the formal definition of Kuratowski in an exercise.

Defining the ordered pair using set theory
------------------------------------------

[[edit](https://en.wikipedia.org/w/index.php?title=Ordered_pair&action=edit&section=3 "Edit section: Defining the ordered pair using set theory")]

If one agrees that [set theory](https://en.wikipedia.org/wiki/Set_theory "Set theory") is an appealing [foundation of mathematics](https://en.wikipedia.org/wiki/Foundations_of_mathematics "Foundations of mathematics"), then all mathematical objects must be defined as [sets](https://en.wikipedia.org/wiki/Set_(mathematics) "Set (mathematics)") of some sort. Hence if the ordered pair is not taken as primitive, it must be defined as a set.[[5]](https://en.wikipedia.org/wiki/Ordered_pair#cite_note-5) Several set-theoretic definitions of the ordered pair are given below (see also Diepert).[[6]](https://en.wikipedia.org/wiki/Ordered_pair#cite_note-6)

### Wiener's definition

[[edit](https://en.wikipedia.org/w/index.php?title=Ordered_pair&action=edit&section=4 "Edit section: Wiener's definition")]

[Norbert Wiener](https://en.wikipedia.org/wiki/Norbert_Wiener "Norbert Wiener") proposed the first set theoretical definition of the ordered pair in 1914:[[7]](https://en.wikipedia.org/wiki/Ordered_pair#cite_note-7)![Image 6: {\displaystyle \left(a,b\right):=\left\{\left\{\left\{a\right\},\,\emptyset \right\},\,\left\{\left\{b\right\}\right\}\right\}.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/26302a77c94ef1c326e85755ee5b557373f723e3) He observed that this definition made it possible to define the [types](https://en.wikipedia.org/wiki/Type_theory "Type theory") of _[Principia Mathematica](https://en.wikipedia.org/wiki/Principia\_Mathematica "Principia Mathematica")_ as sets. _Principia Mathematica_ had taken types, and hence [relations](https://en.wikipedia.org/wiki/Relation_(mathematics) "Relation (mathematics)") of all arities, as [primitive](https://en.wikipedia.org/wiki/Primitive_notion "Primitive notion").

Wiener used {{_b_}} instead of {_b_} to make the definition compatible with [type theory](https://en.wikipedia.org/wiki/Type_theory "Type theory") where all elements in a class must be of the same "type". With _b_ nested within an additional set, its type is equal to ![Image 7: {\displaystyle \{\{a\},\emptyset \}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/ed082e4f1c398481f586a7a4444101490a66c396)'s.

### Hausdorff's definition

[[edit](https://en.wikipedia.org/w/index.php?title=Ordered_pair&action=edit&section=5 "Edit section: Hausdorff's definition")]

About the same time as Wiener (1914), [Felix Hausdorff](https://en.wikipedia.org/wiki/Felix_Hausdorff "Felix Hausdorff") proposed his definition: ![Image 8: {\displaystyle (a,b):=\left\{\{a,1\},\{b,2\}\right\}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/e177babea88568f35c702ed8fd7ab6b6ef3d7329) "where 1 and 2 are two distinct objects different from a and b."[[8]](https://en.wikipedia.org/wiki/Ordered_pair#cite_note-8)

### Kuratowski's definition

[[edit](https://en.wikipedia.org/w/index.php?title=Ordered_pair&action=edit&section=6 "Edit section: Kuratowski's definition")]

In 1921 [Kazimierz Kuratowski](https://en.wikipedia.org/wiki/Kazimierz_Kuratowski "Kazimierz Kuratowski") offered the now-accepted definition[[9]](https://en.wikipedia.org/wiki/Ordered_pair#cite_note-9)[[10]](https://en.wikipedia.org/wiki/Ordered_pair#cite_note-10) of the ordered pair (_a_, _b_): ![Image 9: {\displaystyle (a,\ b)_{K}\;:=\ \{\{a\},\ \{a,\ b\}\}.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/847d0988f7af401d7425c4fb6000e09089a8cb91) When the first and the second coordinates are identical, the definition obtains: ![Image 10: {\displaystyle (x,\ x)_{K}=\{\{x\},\{x,\ x\}\}=\{\{x\},\ \{x\}\}=\{\{x\}\}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/87d8190e7646244dc62bc59d43115a2628295f88)

Given some ordered pair _p_, the property "_x_ is the first coordinate of _p_" can be formulated as: ![Image 11: {\displaystyle \forall Y\in p:x\in Y.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/d206628609aade10d6ed53d7ede5da652149e5c7) The property "_x_ is the second coordinate of _p_" can be formulated as: ![Image 12: {\displaystyle (\exists Y\in p:x\in Y)\land (\forall Y_{1},Y_{2}\in p:(x\in Y_{1}\land x\in Y_{2})\rightarrow Y_{1}=Y_{2}).}](https://wikimedia.org/api/rest_v1/media/math/render/svg/42dc0e342e804dcab3c1adee3ca2daeee9b04b6b) In the case that the left and right coordinates are identical, the right [conjunct](https://en.wikipedia.org/wiki/Logical_conjunction "Logical conjunction")![Image 13: {\displaystyle (\forall Y_{1},Y_{2}\in p:(x\in Y_{1}\land x\in Y_{2})\rightarrow Y_{1}=Y_{2})}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7363aa38d2ed662ae53cadcf06d30e02d3f8f186) is trivially true, since ![Image 14: {\displaystyle Y_{1}=Y_{2}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/a2e55044bce4728ffac52b858c844e163e0bc368) is the case.

If ![Image 15: {\displaystyle p=(x,y)=\{\{x\},\{x,y\}\}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/f66c79152d9b6d6e98fc6a0e5134cb11d679d0d3) then:

![Image 16: {\displaystyle \bigcap p=\bigcap {\bigg \{}\{x\},\{x,y\}{\bigg \}}=\{x\}\cap \{x,y\}=\{x\},}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7e803b5164960fe1867225a7eda5af416115338f)![Image 17: {\displaystyle \bigcup p=\bigcup {\bigg \{}\{x\},\{x,y\}{\bigg \}}=\{x\}\cup \{x,y\}=\{x,y\}.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/184b262d2ac4da5350de3d65dda1e3dc9349b2a1)
This is how we can extract the first coordinate of a pair (using the [iterated-operation notation](https://en.wikipedia.org/wiki/Iterated_binary_operation#Notation "Iterated binary operation") for [arbitrary intersection](https://en.wikipedia.org/wiki/Intersection_(set_theory)#Arbitrary_intersections "Intersection (set theory)") and [arbitrary union](https://en.wikipedia.org/wiki/Union_(set_theory)#Arbitrary_unions "Union (set theory)")): ![Image 18: {\displaystyle \pi _{1}(p)=\bigcup \bigcap p=\bigcup \{x\}=x.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/313009c2d340e0e787ff7968d13934a06d487fa7)

This is how the second coordinate can be extracted: ![Image 19: {\displaystyle \pi _{2}(p)=\bigcup \left\{\left.a\in \bigcup p\,\right|\,\bigcup p\neq \bigcap p\rightarrow a\notin \bigcap p\right\}=\bigcup \left\{\left.a\in \{x,y\}\,\right|\,\{x,y\}\neq \{x\}\rightarrow a\notin \{x\}\right\}=\bigcup \{y\}=y.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/4da20ee45e75b2b4d2dc43213409bdd3995a7022)

(if ![Image 20: {\displaystyle x\neq y}](https://wikimedia.org/api/rest_v1/media/math/render/svg/f51b711ca7f932963cdb268b0817dc72d6258733), then the set ![Image 21: {\displaystyle \{y\}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/144dbb8cb5d9e202a8d1ead595f39c167238858c) could be obtained more simply: ![Image 22: {\displaystyle \{y\}=\{\left.a\in \{x,y\}\,\right|\,a\notin \{x\}\}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/bb8d88b7e462d7635d33fa169a9cace2169bef8c), but the previous formula also takes into account the case when ![Image 23: {\displaystyle x=y}](https://wikimedia.org/api/rest_v1/media/math/render/svg/409a91214d63eabe46ec10ff3cbba689ab687366).)

Note that ![Image 24: {\displaystyle \pi _{1}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/542cbd3dacd0a061d666ed7fc4ed7ad15b47444b) and ![Image 25: {\displaystyle \pi _{2}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/5b4d39c450ad33d7c407aec6fff9f225463ac1f0) are [generalized functions](https://en.wikipedia.org/wiki/Function_(mathematics)#In_the_foundations_of_mathematics "Function (mathematics)"), in the sense that their domains and codomains are [proper classes](https://en.wikipedia.org/wiki/Proper_classes "Proper classes").

The above Kuratowski definition of the ordered pair is "adequate" in that it satisfies the characteristic property that an ordered pair must satisfy, namely that ![Image 26: {\displaystyle (a,b)=(x,y)\leftrightarrow (a=x)\land (b=y)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/2bd7146dacbf7ec4a37cc414ffaa2dc0bd636035). In particular, it adequately expresses 'order', in that ![Image 27: {\displaystyle (a,b)=(b,a)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/d9bec7ed289fc839d085ca6aacab9071ac7c9938) is false unless ![Image 28: {\displaystyle b=a}](https://wikimedia.org/api/rest_v1/media/math/render/svg/0b2e4888d97a754d4bfa4da297b226788a73c6b8). There are other definitions, of similar or lesser complexity, that are equally adequate:

The **reverse** definition is merely a trivial variant of the Kuratowski definition, and as such is of no independent interest. The definition **short** is so-called because it requires two rather than three pairs of [braces](https://en.wikipedia.org/wiki/Braces_(punctuation) "Braces (punctuation)"). Proving that **short** satisfies the characteristic property requires the [Zermelo–Fraenkel set theory](https://en.wikipedia.org/wiki/Zermelo%E2%80%93Fraenkel_set_theory "Zermelo–Fraenkel set theory")[axiom of regularity](https://en.wikipedia.org/wiki/Axiom_of_regularity "Axiom of regularity").[[12]](https://en.wikipedia.org/wiki/Ordered_pair#cite_note-12) Moreover, if one uses [von Neumann's set-theoretic construction of the natural numbers](https://en.wikipedia.org/wiki/Set-theoretic_definition_of_natural_numbers#Definition_as_von_Neumann_ordinals "Set-theoretic definition of natural numbers"), then 2 is defined as the set {0, 1} = {0, {0}}, which is indistinguishable from the pair (0, 0)short. Yet another disadvantage of the **short** pair is the fact that, even if _a_ and _b_ are of the same type, the elements of the **short** pair are not. (However, if _a_=_b_ then the **short** version keeps having cardinality 2, which is something one might expect of any "pair", including any "ordered pair".)

#### Proving that definitions satisfy the characteristic property

[[edit](https://en.wikipedia.org/w/index.php?title=Ordered_pair&action=edit&section=8 "Edit section: Proving that definitions satisfy the characteristic property")]

Prove: (_a_, _b_) = (_c_, _d_) [if and only if](https://en.wikipedia.org/wiki/If_and_only_if "If and only if")_a_ = _c_ and _b_ = _d_.

**Kuratowski**:

_If_. If _a_ = _c_ and _b_ = _d_, then {{_a_}, {_a_, _b_}} = {{_c_}, {_c_, _d_}}. Thus (_a, b_)K = (_c_, _d_)K.

_Only if_. Two cases: _a_ = _b_, and _a_ ≠ _b_.

If _a_ = _b_:

(_a, b_)K = {{_a_}, {_a_, _b_}} = {{_a_}, {_a_, _a_}} = {{_a_}}.{{_c_}, {_c_, _d_}} = (_c_, _d_)K = (_a_, _b_)K = {{_a_}}.Thus {_c_} = {_c_, _d_} = {_a_}, which implies _a_ = _c_ and _a_ = _d_. By hypothesis, _a_ = _b_. Hence _b_ = _d_.
If _a_ ≠ _b_, then (_a_, _b_)K = (_c_, _d_)K implies {{_a_}, {_a_, _b_}} = {{_c_}, {_c_, _d_}}.

Suppose {_c_, _d_} = {_a_}. Then _c_ = _d_ = _a_, and so {{_c_}, {_c_, _d_}} = {{_a_}, {_a_, _a_}} = {{_a_}, {_a_}} = {{_a_}}. But then {{_a_}, {_a, b_}} would also equal {{_a_}}, so that _b_ = _a_ which contradicts _a_ ≠ _b_.Suppose {_c_} = {_a_, _b_}. Then _a_ = _b_ = _c_, which also contradicts _a_ ≠ _b_.Therefore {_c_} = {_a_}, so that _c = a_ and {_c_, _d_} = {_a_, _b_}.If _d_ = _a_ were true, then {_c_, _d_} = {_a_, _a_} = {_a_} ≠ {_a_, _b_}, a contradiction. Thus _d_ = _b_ is the case, so that _a_ = _c_ and _b_ = _d_.
**Reverse**:

 (_a, b_)reverse = {{_b_}, {_a, b_}} = {{_b_}, {_b, a_}} = (_b, a_)K.

_If_. If (_a, b_)reverse = (_c, d_)reverse, (_b, a_)K = (_d, c_)K. Therefore, _b = d_ and _a = c_.

_Only if_. If _a = c_ and _b = d_, then {{_b_}, {_a, b_}} = {{_d_}, {_c, d_}}. Thus (_a, b_)reverse = (_c, d_)reverse.

**Short:**[[13]](https://en.wikipedia.org/wiki/Ordered_pair#cite_note-13)

_If_: If _a = c_ and _b = d_, then {_a_, {_a, b_}} = {_c_, {_c, d_}}. Thus (_a, b_)short = (_c, d_)short.

_Only if_: Suppose {_a_, {_a, b_}} = {_c_, {_c, d_}}. Then _a_ is in the left hand side, and thus in the right hand side. Because equal sets have equal elements, one of _a = c_ or _a_ = {_c, d_} must be the case.

If _a_ = {_c, d_}, then by similar reasoning as above, {_a, b_} is in the right hand side, so {_a, b_} = _c_ or {_a, b_} = {_c, d_}. If {_a, b_} = _c_ then _c_ is in {_c, d_} = _a_ and _a_ is in _c_, and this combination contradicts the axiom of regularity, as {_a, c_} has no minimal element under the relation "element of."If {_a, b_} = {_c, d_}, then _a_ is an element of _a_, from _a_ = {_c, d_} = {_a, b_}, again contradicting regularity.Hence _a = c_ must hold.
Again, we see that {_a, b_} = _c_ or {_a, b_} = {_c, d_}.

The option {_a, b_} = _c_ and _a = c_ implies that _c_ is an element of _c_, contradicting regularity.So we have _a = c_ and {_a, b_} = {_c, d_}, and so: {_b_} = {_a, b_} \ {_a_} = {_c, d_} \ {_c_} = {_d_}, so _b_ = _d_.

### Quine–Rosser definition

[[edit](https://en.wikipedia.org/w/index.php?title=Ordered_pair&action=edit&section=9 "Edit section: Quine–Rosser definition")]

[Rosser](https://en.wikipedia.org/wiki/J._Barkley_Rosser "J. Barkley Rosser") (1953)[[14]](https://en.wikipedia.org/wiki/Ordered_pair#cite_note-14) employed a definition of the ordered pair due to [Quine](https://en.wikipedia.org/wiki/Willard_van_Orman_Quine "Willard van Orman Quine") which requires a prior definition of the [natural numbers](https://en.wikipedia.org/wiki/Natural_number "Natural number"). As ![Image 33: {\displaystyle x\setminus \mathbb {N} }](https://wikimedia.org/api/rest_v1/media/math/render/svg/4601987d3caf60fe2f00e9c4ca2f1d77a3b335e6) is the set of the elements of ![Image 34: {\displaystyle x}](https://wikimedia.org/api/rest_v1/media/math/render/svg/87f9e315fd7e2ba406057a97300593c4802b53e4) not in ![Image 35: {\displaystyle \mathbb {N} }](https://wikimedia.org/api/rest_v1/media/math/render/svg/fdf9a96b565ea202d0f4322e9195613fb26a9bed) go on with ![Image 36: {\displaystyle \varphi (x):=\sigma [x]=\{\sigma (\alpha )\mid \alpha \in x\}=(x\setminus \mathbb {N} )\cup \{n+1:n\in (x\cap \mathbb {N} )\}.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/d53912e00528be154feef3e3a83fe8b6a08339ba) This is the [set image](https://en.wikipedia.org/wiki/Image_(mathematics)#Image_of_a_subset "Image (mathematics)") of a set ![Image 37: {\displaystyle x}](https://wikimedia.org/api/rest_v1/media/math/render/svg/87f9e315fd7e2ba406057a97300593c4802b53e4) under ![Image 38: {\displaystyle \sigma }](https://wikimedia.org/api/rest_v1/media/math/render/svg/59f59b7c3e6fdb1d0365a494b81fb9a696138c36), [sometimes denoted](https://en.wikipedia.org/wiki/Image_(mathematics)#Other_terminology "Image (mathematics)") by ![Image 39: {\displaystyle \sigma ''x}](https://wikimedia.org/api/rest_v1/media/math/render/svg/2eef8479b565147e0e6dc2f312da049f27c65658) as well. Applying function ![Image 40: {\displaystyle \varphi }](https://wikimedia.org/api/rest_v1/media/math/render/svg/33ee699558d09cf9d653f6351f9fda0b2f4aaa3e) to a set _x_ simply increments every natural number in it. In particular, ![Image 41: {\displaystyle \varphi (x)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/4c4046f1f2de7df04bde418ba2bc4d3898ac2385) never contains contain the number 0, so that for any sets _x_ and _y_, ![Image 42: {\displaystyle \varphi (x)\neq \{0\}\cup \varphi (y).}](https://wikimedia.org/api/rest_v1/media/math/render/svg/d70340bf8b070f4818bd0b5b93bf6c12613bf543) Further, define ![Image 43: {\displaystyle \psi (x):=\sigma [x]\cup \{0\}=\varphi (x)\cup \{0\}.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/4ff087bba5936537ad424e8179fde6cf74e54059) By this, ![Image 44: {\displaystyle \psi (x)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/a596a1fb4130a47f6b88c66150497338bd6cbccc) does always contain the number 0.

Finally, define the ordered pair (_A_, _B_) as the disjoint union ![Image 45: {\displaystyle (A,B):=\varphi [A]\cup \psi [B]=\{\varphi (a):a\in A\}\cup \{\varphi (b)\cup \{0\}:b\in B\}.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/eb6ca4663acb45c04927b9af820b2b60125ae381) (which is ![Image 46: {\displaystyle \varphi ''A\cup \psi ''B}](https://wikimedia.org/api/rest_v1/media/math/render/svg/6a0cb4d5deba4b30a1118c829e9627fad3b7964e) in alternate notation).

Extracting all the elements of the pair that do not contain 0 and undoing ![Image 47: {\displaystyle \varphi }](https://wikimedia.org/api/rest_v1/media/math/render/svg/33ee699558d09cf9d653f6351f9fda0b2f4aaa3e) yields _A_. Likewise, _B_ can be recovered from the elements of the pair that do contain 0.[[15]](https://en.wikipedia.org/wiki/Ordered_pair#cite_note-15)

For example, the pair ![Image 48: {\displaystyle (\{\{a,0\},\{b,c,1\}\},\{\{d,2\},\{e,f,3\}\})}](https://wikimedia.org/api/rest_v1/media/math/render/svg/16407dad99c633854e68928439305839335a2fba) is encoded as ![Image 49: {\displaystyle \{\{a,1\},\{b,c,2\},\{d,3,0\},\{e,f,4,0\}\}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/454da43f45ad8cf3b790718f110bb1dfd9d6a987) provided ![Image 50: {\displaystyle a,b,c,d,e,f\notin \mathbb {N} }](https://wikimedia.org/api/rest_v1/media/math/render/svg/0b2f82f7e856bf851117fd6a7fcca6f10fad512e).

In [type theory](https://en.wikipedia.org/wiki/Type_theory "Type theory") and in outgrowths thereof such as the axiomatic set theory [NF](https://en.wikipedia.org/wiki/New_Foundations "New Foundations"), the Quine–Rosser pair has the same type as its projections and hence is termed a "type-level" ordered pair. Hence this definition has the advantage of enabling a [function](https://en.wikipedia.org/wiki/Function_(mathematics) "Function (mathematics)"), defined as a set of ordered pairs, to have a type only 1 higher than the type of its arguments. This definition works only if the set of natural numbers is infinite. This is the case in [NF](https://en.wikipedia.org/wiki/New_Foundations "New Foundations"), but not in [type theory](https://en.wikipedia.org/wiki/Type_theory "Type theory") or in [NFU](https://en.wikipedia.org/wiki/New_Foundations "New Foundations"). [J. Barkley Rosser](https://en.wikipedia.org/wiki/J._Barkley_Rosser "J. Barkley Rosser") showed that the existence of such a type-level ordered pair (or even a "type-raising by 1" ordered pair) implies the [axiom of infinity](https://en.wikipedia.org/wiki/Axiom_of_infinity "Axiom of infinity"). For an extensive discussion of the ordered pair in the context of Quinian set theories, see Holmes (1998).[[16]](https://en.wikipedia.org/wiki/Ordered_pair#cite_note-16)

### Cantor–Frege definition

[[edit](https://en.wikipedia.org/w/index.php?title=Ordered_pair&action=edit&section=10 "Edit section: Cantor–Frege definition")]

Early in the development of the set theory, before paradoxes were discovered, Cantor followed Frege by defining the ordered pair of two sets as the class of all relations that hold between these sets, assuming that the notion of relation is primitive:[[17]](https://en.wikipedia.org/wiki/Ordered_pair#cite_note-17)![Image 51: {\displaystyle (x,y)=\{R:xRy\}.}](https://wikimedia.org/api/rest_v1/media/math/render/svg/0b491bf78a32c2176ad81597505c98fced80f1cf)

This definition is inadmissible in most modern formalized set theories and is methodologically similar to defining the [cardinal](https://en.wikipedia.org/wiki/Cardinality "Cardinality") of a set as the class of all sets equipotent with the given set.[[18]](https://en.wikipedia.org/wiki/Ordered_pair#cite_note-18)

[Morse–Kelley set theory](https://en.wikipedia.org/wiki/Morse%E2%80%93Kelley_set_theory "Morse–Kelley set theory") makes free use of [proper classes](https://en.wikipedia.org/wiki/Proper_class "Proper class").[[19]](https://en.wikipedia.org/wiki/Ordered_pair#cite_note-19)[Morse](https://en.wikipedia.org/wiki/Anthony_Morse "Anthony Morse") defined the ordered pair so that its projections could be proper classes as well as sets. (The Kuratowski definition does not allow this.) He first defined ordered pairs whose projections are sets in Kuratowski's manner. He then _redefined_ the pair ![Image 52: {\displaystyle (x,y)=(\{0\}\times s(x))\cup (\{1\}\times s(y))}](https://wikimedia.org/api/rest_v1/media/math/render/svg/bdeca536dd727abd6433464efdb16ac7849ececb) where the component Cartesian products are Kuratowski pairs of sets and where ![Image 53: {\displaystyle s(x)=\{\emptyset \}\cup \{\{t\}\mid t\in x\}}](https://wikimedia.org/api/rest_v1/media/math/render/svg/f551ddec5cc774f540b103d6a4415db6fc3838ec)

This renders possible pairs whose projections are proper classes. The Quine–Rosser definition above also admits [proper classes](https://en.wikipedia.org/wiki/Proper_class "Proper class") as projections. Similarly the triple is defined as a 3-tuple as follows: ![Image 54: {\displaystyle (x,y,z)=(\{0\}\times s(x))\cup (\{1\}\times s(y))\cup (\{2\}\times s(z))}](https://wikimedia.org/api/rest_v1/media/math/render/svg/7bfeb6fe9334a59fba4687a8e230b819e3979e4d)

The use of the singleton set ![Image 55: {\displaystyle s(x)}](https://wikimedia.org/api/rest_v1/media/math/render/svg/10f51eec88706f1a26ac7430dc71d92c15e71a6e) which has an inserted empty set allows tuples to have the uniqueness property that if _a_ is an _n_-tuple and b is an _m_-tuple and _a_ = _b_ then _n_ = _m_. Ordered triples which are defined as ordered pairs do not have this property with respect to ordered pairs.

[![Image 56](https://upload.wikimedia.org/wikipedia/commons/thumb/8/89/CategoricalProduct-03.svg/250px-CategoricalProduct-03.svg.png)](https://en.wikipedia.org/wiki/File:CategoricalProduct-03.svg)

[Commutative diagram](https://en.wikipedia.org/wiki/Commutative_diagram "Commutative diagram") for the set product _X_ 1×_X_ 2.

A category-theoretic [product](https://en.wikipedia.org/wiki/Product_(category_theory) "Product (category theory)")_A_ × _B_ in a [category of sets](https://en.wikipedia.org/wiki/Category_of_sets "Category of sets") represents the set of ordered pairs, with the first element coming from _A_ and the second coming from _B_. In this context the characteristic property above is a consequence of the [universal property](https://en.wikipedia.org/wiki/Universal_property "Universal property") of the product and the fact that elements of a set _X_ can be identified with morphisms from 1 (a one element set) to _X_. While different objects may have the universal property, they are all [naturally isomorphic](https://en.wikipedia.org/wiki/Naturally_isomorphic "Naturally isomorphic").
